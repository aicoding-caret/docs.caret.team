<Note>
これらのツールはCaret専用機能です。
</Note>

Caretは、AIが画像やドキュメントを直接分析できる強力なツールを提供します。これらのツールのおかげで、ユーザーがファイルを手動で添付しなくても、AIはパスだけでファイルを読み取り分析できます。

## 画像分析ツール (analyze_image)

`analyze_image`ツールは画像ファイルを分析して、内容を説明したり特定の質問に回答します。

### ユースケース

- **UI/UXレビュー**: スクリーンショットでレイアウトの問題、配置エラー、視覚的なバグを確認
- **テキスト抽出**: スクリーンショット、エラーダイアログ、ログからテキストを抽出
- **コードレビュー**: コードのスクリーンショットを分析しパターンを識別
- **デザイン比較**: UI実装とデザインモックアップを比較
- **エラー分析**: エラーダイアログやスタックトレースを解釈

### サポートされる形式

- PNG, JPG, JPEG, WebP, GIF, AVIF, BMP, TIFF

### 要件

- **Caretアカウントログインが必要**: 画像分析はCaret APIを通じて処理されます
- **自動画像最適化**: 大きな画像は送信前に自動的に最適化されます

## ドキュメント読み取りツール (read_document)

`read_document`ツールはさまざまなドキュメント形式を読み取り、テキストを抽出します。

### Clineとの比較

| 機能 | Cline | Careti |
|------|-------|-------|
| PDF読み取り | ユーザー添付のみ | **AIがパスで直接読み取り** |
| DOCX/XLSX読み取り | ユーザー添付のみ | **AIがパスで直接読み取り** |
| HWPX（韓国語ハングル） | ❌ 未サポート | ✅ **サポート** |
| HWP 5.0（レガシーハングル） | ❌ 未サポート | ✅ **サポート** |
| PPTX | ❌ 未サポート | ✅ **サポート** |

### サポートされる形式

| 形式 | 拡張子 | 説明 |
|------|--------|------|
| PDF | `.pdf` | Adobe PDFドキュメント |
| Word | `.docx` | Microsoft Word 2007+ |
| Excel | `.xlsx` | Microsoft Excel 2007+ |
| PowerPoint | `.pptx` | Microsoft PowerPoint 2007+ |
| ハングル（新） | `.hwpx` | Hancom Office Hangul 2014+ |
| ハングル（旧） | `.hwp` | Hancom Office Hangul 5.0 |
| Jupyter | `.ipynb` | Jupyter Notebook |

### セキュリティ

- **パス検証**: パストラバーサル攻撃を防ぐためのパス正規化
- **ファイルサイズ制限**: 最大50MB
- **形式検証**: サポートされる拡張子のみ許可
