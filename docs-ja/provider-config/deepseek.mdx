<BrandName /> は `deepseek-chat` や `deepseek-reasoner` を含む DeepSeek API を通じたモデルアクセスをサポートしています。

**ウェブサイト:** [https://platform.deepseek.com/](https://platform.deepseek.com/)

### API キーの取得

1.  **サインアップ/サインイン:** [DeepSeek Platform](https://platform.deepseek.com/) にアクセスします。アカウントを作成するか、サインインしてください。
2.  **API キーに移動:** プラットフォームの [API keys](https://platform.deepseek.com/api_keys) セクションで API キーを確認します。
3.  **キーを作成:** "Create new API key" をクリックします。キーに分かりやすい名前を付けてください（例："<BrandName />"）。
4.  **キーをコピー:** **重要:** API キーを _すぐに_ コピーしてください。再表示することはできません。安全に保管してください。

### サポートされているモデル

<BrandName /> は以下の DeepSeek モデルをサポートしています：

-   `deepseek-v3-0324`（コーディングタスクに推奨）
-   `deepseek-r1`（推論タスクに推奨）

### <BrandName /> での設定

1.  **<BrandName /> 設定を開く:** <BrandName /> パネルの ⚙️ アイコンをクリックします。
2.  **プロバイダーを選択:** "API Provider" ドロップダウンから "DeepSeek" を選択します。
3.  **API キーを入力:** "DeepSeek API Key" フィールドに DeepSeek API キーを貼り付けます。
4.  **モデルを選択:** "Model" ドロップダウンから希望するモデルを選択します。

### ヒントと注意事項

-   **価格:** モデルコストの詳細については、[DeepSeek Pricing](https://api-docs.deepseek.com/quick_start/pricing/) ページを参照してください。
