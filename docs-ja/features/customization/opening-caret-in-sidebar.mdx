初期状態では Careti は VS Code 左側のサイドバーに表示されます。より快適な作業のために右サイドバーへ移動することを推奨します。左にファイル、右に Careti を配置でき、コードベースを見ながら対話できます。

## VS Code

右サイドバーで開く手順:

<Steps>
  <Step title="拡張ビューの整列">
    拡張ビューが左側に縦表示されていることを確認します。
  </Step>
  <Step title="右側ビューを開く">
    右側パネルを開くボタン(通常 GitHub Copilot チャット用)をクリックします。必要に応じて `Option + CMD/Ctrl + B` も使えます。
  </Step>
  <Step title="Careti アイコンをドラッグ">
    Careti アイコンを右側パネル上部のナビ領域にドラッグします。
  </Step>
</Steps>

## Cursor

Cursor で右サイドバーに表示する手順:

<Steps>
  <Step title="アクティビティバーの向き">
    Cursor はデフォルトで横向きアクティビティバーです([詳細](https://cursor.com/docs/configuration/migrations/vscode#activity-bar-orientation))。縦向きに変更するには:
    
    1. コマンドパレットを開く (`CMD/Ctrl + Shift + P`)
    2. "Preferences: Open Settings (UI)" を検索
    3. `workbench.activityBar.orientation` を検索
    4. 値を `vertical` に設定
    5. Cursor を再起動
  </Step>
  <Step title="AI パネルを開く">
    Cursor のキューブアイコン(AI Pane)をクリックして右側パネルを開きます。
  </Step>
  <Step title="Careti をドラッグ">
    Careti アイコンを AI Pane のサイドバーへドラッグします。
  </Step>
</Steps>

	<img
		src="https://storage.googleapis.com/cline_public_images/Cursor-sidebar.gif"

設定後は右側で Careti が読み込まれ、通常通り使用できます。
