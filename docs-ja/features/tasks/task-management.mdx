<BrandName /> はタスク履歴を管理するための機能を提供します。タスクが増えるほどこれらの機能が重要になります。

## タスク履歴へのアクセス

タスク履歴の開き方は複数あります:

1. <BrandName /> サイドバーの **"History"** ボタン
2. コマンドパレットで "<BrandName />: Show Task History" を検索
3. キーボードショートカット (VS Code 設定による)

## タスク履歴の画面

タスク履歴ビューは、すべてのタスクを表示・管理する中心画面です。検索、並び替え、フィルタリングで効率化できます。

### 検索とフィルタ

#### 検索バー
- タスク内容のファジー検索
- プロンプト/応答/コードを対象
- 入力と同時に絞り込み
- 一致箇所をハイライト

#### 並び替え
- **新しい順** (デフォルト)
- **古い順**
- **費用が高い順**
- **トークン使用量順**
- **関連度順**

#### お気に入りフィルタ
- 星付きタスクのみ表示
- 重要タスクに素早くアクセス
- 検索と併用可能

## タスク操作

各タスクで可能な操作:

### 主要操作

- **Open**: タスクを開き直す
- **Resume**: 中断タスクを継続
- **Export**: 会話を Markdown で保存

### 管理操作

- **お気に入り** ⭐: 星アイコンで重要タスクを固定
- **削除** 🗑️: 個別タスク削除(お気に入りは保護)
- **複製**: 既存タスクから新規作成

## ⭐ お気に入り

重要なタスクを保護し、すぐに参照できます。

### 使い方

**お気に入り登録**
- 星アイコンをクリック
- 再クリックで解除

**保護機能**
- お気に入りは誤削除から保護
- 一括削除では除外
- 明示的に解除可能

**活用例**
- 参考実装
- 成功パターン
- 再利用コード
- マイルストーン
- 学習用例
