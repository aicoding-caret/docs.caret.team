## <BrandName /> CLI とは？

<BrandName /> CLI はターミナル上で AI コーディングエージェントを直接実行します。CI/CD で git diff をパイプして自動コードレビューを行ったり、複数インスタンスを並行実行して開発を分担したり、既存のシェルワークフローに統合できます。

CLI はシステム内のインスタンスを追跡し、人が読みやすい出力／スクリプト向け JSON／プレーンテキスト／リッチ出力を提供します。

<Tip>
今すぐ始めるなら [インストールガイド](/ja/caret-cli/installation) を確認してください。
</Tip>

## できること

**自動コードメンテナンス**
- 日次実行で lint 問題を検出・修正
- セキュリティ脆弱性のスキャンと自動パッチ
- 依存更新とテスト実行

**マルチインスタンス開発**
- フロント/バックエンドを同時に別インスタンスで実行
- ブランチごとにインスタンスを分離
- 複数 PR の並行レビュー

**カスタムワークフロー**
- git hooks と連携したシェルスクリプト
- 複雑なデータ構造を <BrandName /> にパイプ
- jq/grep/awk など既存ツールと統合

**CI/CD 統合**
- GitHub Actions で PR 自動レビュー
- GitLab パイプラインでスキーマ変更分析
- Jenkins でテスト失敗の解析と修正提案
