## 前提条件

Careti CLI は Node.js 20 以上が必要です。最良の体験のため Node.js 22 を推奨します。

バージョン確認:

```bash
node --version
```

## インストール

```bash
npm install -g caret
```

インストール後、Careti アカウントで認証します:

```bash
caret auth
```

認証ウィザードが起動し、ログインと好みのモデルプロバイダー設定を行います。

<Note>
Careti CLI はインストールだけでは認証されません。一般的な CLI と違い、`caret auth` の実行が必須です。
`caret config set ...` はプロバイダー設定用で、アカウント認証の代わりにはなりません。
</Note>

## クイックスタート

数秒で Careti を開始できます:

```bash
caret
```

これだけで OK です。`caret` を実行したディレクトリで対話セッションが開始されます。タスクを入力し、計画を確認してから `/act` で実行します。

対話なしで素早く実行する場合:

```bash
caret "Add unit tests to utils.js"
```

単一コマンドで実行できるため、軽い作業や自動化に最適です。

<Tip>
初めてなら対話モード(`caret`)から始めましょう。慣れたら [3つの主要フロー](/ja/caret-cli/three-core-flows) を確認してください。
</Tip>
