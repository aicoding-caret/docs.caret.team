Caret の動作を体験するチュートリアルです。会話だけで要件を伝え、ファイル生成と修正を確認できます。

## 前提条件

- **Caret インストール済み** ([インストールガイド](/ja/getting-started/installing-caret))
- **AI モデル選択済み** ([モデル設定](/ja/getting-started/selecting-your-model))
- **フォルダを開いている** (空でも可)

## Step 1: Caret を開く

サイドバーの Caret アイコンをクリックするとチャットが開きます。

<Info>
**Tip:** `Cmd+Shift+P` / `Ctrl+Shift+P` で "Caret: Open In New Tab" を検索して開けます。
</Info>

## Step 2: タスクを依頼

次のプロンプトを貼り付けます:

```
Create a simple website in a single HTML file. It should have:
- A welcome message saying "Hello from Caret!"
- A colorful gradient background
- A button that cycles through different color themes when clicked
- Modern, clean design
- All CSS and JavaScript should be included in the same HTML file
```

<Frame>
  <img src="https://storage.googleapis.com/cline_public_images/chat-prompt.png" alt="Caret Chat Prompt"/>
</Frame>

## Step 3: 何が起きる？

Caret は以下を実行します:

1. **単一ファイル作成**
   - `index.html`

2. **承認を要求** (自動承認が無効な場合)
   - "Approve" をクリックして実行

3. **数秒で完了**

## Step 4: サイトを確認

1. **`index.html` を探す**
2. 右クリックして "Reveal in Finder/Explorer" → ブラウザで開く
3. ボタンをクリックしてテーマ変更を確認

## 変更を試す

```
Add a counter that shows how many times the button has been clicked
```

または

```
Make the welcome message fade in when the page loads
```

Caret は前のコンテキストを理解して更新します。

<Tip>
**できるようになること:**
- 明確なプロンプトで依頼
- 承認して実行
- すばやいプロジェクト生成
- 反復改善
</Tip>
