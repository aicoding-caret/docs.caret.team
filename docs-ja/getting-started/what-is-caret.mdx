# Caretとは何ですか？

**Caret = Cline v3.38.2 + デュアルモード + ペルソナ + フルローカライズ**

Caretは最新Cline(v3.38.2)と100%互換しつつ、個人化されたAI体験と拡張機能を加えたAIコーディングエージェントです。オートコンプリートではなく、コードベースを理解し、計画・実行まで行う本物のコーディングパートナーです。

## ✨ Caretを選ぶべき理由

1. **🔄 デュアルモード** — Caret(拡張) ↔ Cline(クラシック) をワンクリック切替
2. **🎭 ペルソナシステム** — 自分だけのAIコンパニオンと一緒にコーディング
3. **🧠 最適化プロンプト** — コストと性能のバランスを取った応答
4. **🌍 フルローカライズ** — UI/ドキュメント4言語 (en/ko/ja/zh)
5. **⌨️ 強化UX** — 入力履歴・ショートカット・自動モデル検索
6. **🤖 幅広いモデル** — 20プロバイダー、260のユニークモデル (Cline v3.38.2互換)

👉 **すべての独占機能の詳細を見る**: [Caret独占機能の概要](/ja/caret-exclusive/overview)

### Clineモード (クラシック方式)
- Plan/Actアプローチ - 既存のClineユーザーに馴染みのある方式
- すべての設定互換 - 既存の設定をそのまま使用
- 安定性保証 - Clineの検証済みコアエンジン

### Caretモード (拡張機能)
- ペルソナシステム - 自分だけの個人化されたAIコンパニオン
- 強化されたシステムプロンプト - よりスマートで効率的なAI
- 4言語サポート - 完璧な多言語環境
- Chatbot/Agent - より直感的な会話スタイル

👉 **詳細を見る**: [デュアルモードシステム](/ja/caret-exclusive/dual-prompt-modes)

## Clineベースの検証済みアーキテクチャ

Caretは検証済みのClineアーキテクチャをベースに、韓国の開発者とグローバルユーザーのための追加機能を提供します。

### 完全な透明性
Caretがファイルを読み、アプローチを検討し、変更を提案するプロセスをリアルタイムで観察しましょう。すべての決定が見え、すべての編集は実行前にレビューできます。

### あなたのモデル、あなたのコントロール
複雑な推論にはClaudeを、大規模なコンテキストにはGeminiを、効率性にはQwen3 Coderを使用しましょう。新しいモデルがリリースされたら即座に切り替えることができます。あなたのAPIキー、あなたの選択。

### 実際のエンジニアリングのために構築
Caretは次のことができます:
- コードベース全体で**ファイルの読み書き**
- **ターミナルコマンドの実行**とエラーのデバッグ
- コードを書く前に**複雑な機能を計画**
- MCPサーバーを通じて**外部システムに接続**
- インテリジェントなコンテキスト管理で**大規模なコードベースを理解**

## Clineとの関係

**Caretは[Cline](https://github.com/cline/cline)オープンソースプロジェクトのフォークです。** 元のClineプロジェクトと開発者に深く感謝します。

Caretは **Cline v3.38.2** をベースに、**4つのコア拡張**を提供します:
1. 🎭 個人化AIペルソナシステム
2. 🌍 英語・韓国語・日本語・中国語のフルローカライズ
3. 🧠 最適化されたプロンプトシステム
4. 🔄 Caret/Clineデュアルモード切替

**その他にもプロンプト履歴、強化されたプロバイダー設定、そしてAIの信頼性を高める[知識同期ワークフロー](/ja/caret-exclusive/knowledge-parity-system)など[追加の独占機能](/ja/caret-exclusive/overview)を提供します。**

**すべてのClineの基本機能はCaretで完全に保存され機能します。**
