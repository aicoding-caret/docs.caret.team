# Caretとは何ですか？

**Caret = Cline + デュアルモード + ペルソナ + 7言語 + 画像/ドキュメントツール**

CaretはClineと100%互換しつつ、個人化されたAI体験と拡張機能を加えたAIコーディングエージェントです。既存のCline設定をそのまま使用し、266モデル対応、Caretアカウントで毎月無料クレジットも提供。

1. **🔄 デュアルモード** — Caret(拡張) ↔ Cline(クラシック) をワンクリック切替
2. **🎭 ペルソナシステム** — 自分だけのAIコンパニオンと一緒にコーディング
3. **🧠 最適化プロンプト** — コストと性能のバランスを取った応答
4. **🌍 7言語サポート** — UI/ドキュメント完全ローカライズ (EN/KO/JA/ZH/FR/DE/RU)
5. **⌨️ 強化UX** — 入力履歴・ショートカット・自動モデル検索
6. **🤖 266モデル** — 31プロバイダー、Cline互換
7. **🛠️ 画像/ドキュメントツール** — テキストモデルでも画像生成・分析、ドキュメント読み込み可能

## 📁 デュアルディレクトリ & /init

**トークン最適化AIコンテキスト + ユーザー言語ドキュメントを1:1ミラーリングで管理**

```
.agents/                       # AI用 (英語、トークン最適化)
├── context/agents-rules.json  # ルールSoT (Single Source of Truth)
├── workflows/                 # タスクワークフロー
├── skills/                    # AIスキル
└── hooks/                     # 自動化フック

.users/                        # ユーザー用 (ユーザー言語)
├── context/agents-rules.md    # 詳細説明
├── workflows/                 # (.agents/ ミラー)
├── skills/                    # (.agents/ ミラー)
└── hooks/                     # (.agents/ ミラー)
```

**主要機能**:
- **`/init`**: プロジェクト分析後にコンテキストファイルを自動生成
- **1:1ミラーリング**: `.agents/` ↔ `.users/` の変更を同期維持
- **AGENTS.md / CLAUDE.md**: コンテキスト接続の標準エントリーポイント
- **レガシーフォールバック**: 既存のルールパスを自動検出してマイグレーション案内

👉 **詳細を見る**: [AI組織知識とルール標準化](/ja/caret-exclusive/knowledge-parity-system)

## Clineベースの検証済みアーキテクチャ

Caretは検証済みのClineアーキテクチャをベースに、韓国の開発者とグローバルユーザーのための追加機能を提供します。

### 完全な透明性
Caretがファイルを読み、アプローチを検討し、変更を提案するプロセスをリアルタイムで観察しましょう。すべての決定が見え、すべての編集は実行前にレビューできます。

### あなたのモデル、あなたのコントロール
複雑な推論にはClaudeを、大規模なコンテキストにはGeminiを、効率性にはQwen3 Coderを使用しましょう。新しいモデルがリリースされたら即座に切り替えることができます。あなたのAPIキー、あなたの選択。

### 実際のエンジニアリングのために構築
Caretは次のことができます:
- コードベース全体で**ファイルの読み書き**
- **ターミナルコマンドの実行**とエラーのデバッグ
- コードを書く前に**複雑な機能を計画**
- MCPサーバーを通じて**外部システムに接続**
- インテリジェントなコンテキスト管理で**大規模なコードベースを理解**

## Clineとの関係

**Caretは[Cline](https://github.com/cline/cline)オープンソースプロジェクトのフォークです。** 元のClineプロジェクトと開発者に深く感謝します。

Caretは Cline をベースに、以下の**コア拡張**を提供します:

1. 🎭 個人化AIペルソナシステム
2. 🌍 7言語フルローカライズ (EN/KO/JA/ZH/FR/DE/RU)
3. 🧠 最適化されたプロンプトシステム
4. 🔄 Caret/Clineデュアルモード切替
5. 🛠️ 画像/ドキュメントツール (Caretアカウント)
6. 📁 デュアルディレクトリアーキテクチャ & /init
7. ☁️ 追加AIプロバイダー (Z.AI, Naver, Upstage など)

**すべてのClineの基本機能はCaretで完全に保存され機能します。**
