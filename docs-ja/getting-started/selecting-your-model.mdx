Careti はリクエストを理解しコードを書くために AI モデルが必要です。どの専門家と働くかを選ぶように、モデルごとに強みとコストが異なります。

## クイックスタート: プロバイダーを選ぶ

最も簡単なのは **Careti** プロバイダーから始めることです:

1. **Careti 設定を開く**: Careti チャット右上の歯車(⚙️)をクリック
2. API Provider のドロップダウンで **"Careti"** を選択
3. モデルを選択 - **Claude Sonnet 4.5** または **DeepSeek V3** から始めるのがおすすめ

<Frame>
  <img src="https://storage.googleapis.com/cline_public_images/step2-provider.png" alt="Select Careti Provider" />
</Frame>

**これで完了!** API キーを管理する必要なく、複数のモデルをすぐに使えます。

<Tip>
**無料モデルの提供**: Careti はパートナープロバイダー経由で無料推論を提供することがあります。提供時はモデルのドロップダウンに表示されます。
</Tip>

## 別のプロバイダーを使う

自分の API キーを使いたい場合は、次のようなプロバイダーを選べます:

- **OpenRouter** - コスパがよく、モデルが豊富
- **Anthropic** - Claude モデルを直接利用
- **OpenAI** - GPT モデル
- **Google Gemini** - Google の AI モデル
- **Ollama** - ローカルで実行

プロバイダーを選んだ後は:
1. プロバイダーのサイトで API キーを取得
2. Careti 設定の API Key 欄に貼り付け
3. モデルを選択

<Note>
多くのプロバイダーは API キー発行前に支払い情報の登録が必要です。
</Note>

## どのモデルを選ぶべき?

初めてなら次をおすすめします:

| 優先度 | おすすめモデル | 理由 |
|---------------|-------------------|-----|
| **安定性** | Claude Sonnet 4.5 | コーディング作業で最も安定 |
| **コスパ** | DeepSeek V3 | 低コストで優れた性能 |
| **速度** | Qwen3 Coder | 高速応答 |
| **プライバシー** | Ollama モデル | 自分のPCで実行 |

会話履歴を失うことなく、いつでもモデルを切り替えられます。
