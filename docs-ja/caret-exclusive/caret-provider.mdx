# Caretプロバイダー

CaretプロバイダーはAPIキーなしで即座にAIコーディングアシスタントを使用できるようにする統合認証システムです。現在開発中で、まもなくリリース予定です。

## これは何ですか？

Caretを初めて使用する多くの開発者が直面する最初の障壁は、まさにAPIキーの設定です。OpenAIやAnthropicのようなAIプロバイダーのAPIキーを発行し、支払い情報を登録し、使用量を管理するプロセスは煩わしいことがあります。

Caretプロバイダーはこのような初期参入障壁を取り除きます。Caretアカウント1つあれば、別途のAPIキー設定なしに即座にAIアシスタントを使用できます。

## Google Gemini優先サポート

Caretプロバイダーの最初のサポートモデルは**Google Gemini**になる予定です。Geminiは強力なコーディング能力を持ちながら、相対的に手頃な価格で提供されるため、初心者と個人開発者に適しています。

## 無料クレジット

Caretプロバイダーを初めて使用するすべてのユーザーに**無料クレジット**を提供する予定です。これにより、実際のコスト負担なしにCaretの機能を十分にテストできます。

## 今後の計画

Geminiサポート後、他のAIモデルもCaretプロバイダーを通じて順次使用できるように追加する計画です。

## 既存のAPIキーとの関係

Caretプロバイダーがリリースされても、既存の直接APIキー入力方式は引き続きサポートされます。次のような選択肢を持つことができます：

**Caretプロバイダー使用**: 簡単な開始、統合管理、無料クレジット活用

**直接APIキー使用**: 既存アカウント活用、より多くのモデル選択、コスト直接管理

両方の方法を一緒に使用することも可能です。
