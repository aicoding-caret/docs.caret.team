# ルール優先順位システム

Caretはルールの標準パスを`.agents/context/`に統一しました。レガシールールパスはサポートしません。

## 標準ルールソース

- ワークスペース規칙: `.agents/context/`
- ディレクトリ単位の指示: `AGENTS.md`（再帰）
- グローバルルール: Documents/Caret/Rules

## 変更理由

- 重複ロードの防止
- 競合の排除
- 決定的な読み込み

## 移行

既存のレガシールールは`.agents/context/`へ移動してください。
