# 強化されたプロバイダー設定

## 🚀 Caretのプロバイダー競争力

**CaretはClineよりも多くのプロバイダーをサポートし、より特化したプロバイダー機能を提供します。**

### なぜCaretのプロバイダーがより強力なのか？

1. **自動モデル検索**: LiteLLM API連携で100以上のプロバイダーを自動表示
2. **リアルタイム更新**: 新モデル追加もアップデート不要で即反映
3. **BizRouter・国内プロバイダー**: BizRouter/LiteLLM改善、国内プロバイダー（Naver、Upstageなど）を順次追加予定
4. **簡単設定**: 手動入力不要、ドロップダウンで選ぶだけ

## 📋 概要

Caretは**LiteLLM API連携**を通じてプロバイダー設定プロセスを完全に自動化しました。サポートされているモデルを手動で検索したり入力したりする必要はもうありません。

## 🎯 コア機能

### 自動モデル検索

プロバイダーを選択すると、そのプロバイダーがサポートする**すべてのAIモデルが自動的に表示**されます。

**動作方法**:
1. Caret設定でAIプロバイダーを選択（例：OpenAI、Anthropic）
2. LiteLLM APIが自動的にサポートモデルリストを検索
3. ドロップダウンから希望のモデルを選択

### リアルタイム更新

新しいAIモデルがリリースされると、**Caretを更新する必要なく**自動的にモデルリストに追加されます。

- LiteLLM API経由で最新のモデル情報をリアルタイム取得
- 常に最新のモデルラインアップを利用可能

### 広範なプロバイダーサポート

**100以上のAIプロバイダー**をサポート：

- **主要グローバル**: OpenAI、Anthropic、Google Gemini、Cohere、AI21 Labs、Hugging Face
- **追加**: Mistral、Fireworks、Groq、Cerebras、Deepseek など

## 🔄 Clineとの違い

| 項目 | Cline | Careti |
|------|-------|-------|
| モデル入力方式 | 手動入力 | 自動検索 |
| モデルリスト確認 | ドキュメント参照必要 | ドロップダウンに自動表示 |
| 新モデルサポート | 更新必要 | 自動反映 |
| サポートプロバイダー | 限定的 | 100以上 |

## 💡 ユースケース

### ユースケース1: プロバイダー比較

複数のAIプロバイダーを比較したいとき:

1. 各プロバイダーを選択するとサポートモデルが自動表示
2. モデル名、性能、価格を比較
3. 最適なモデルを選択

### ユースケース2: 最新モデルの即時利用

新しいAIモデルが公開されたとき:

1. Caretを再起動
2. プロバイダー設定を開く
3. 新モデルが自動でリストに表示

### ユースケース3: 国内AIモデルの利用

韓国など東アジアのモデルを使いたいとき:

1. 国内プロバイダーを選択（例: Naver, Upstage など順次追加予定）
2. 対応モデルが自動表示
3. 母語に強いモデルを選択

## 🌏 BizRouter・国内プロバイダー連携

- **BizRouter**: LiteLLMと連携しモデル自動検索とヘルスチェックを改善中
- **国内プロバイダー**: Naver, Upstage などを順次追加予定（韓国語モデル優先）
- すべての連携で「自動モデル検索・リアルタイム更新」をそのまま活用

## 🔧 技術実装

### LiteLLM API 連携

Caretは [LiteLLM](https://docs.litellm.ai/) API を利用しています:

```
GET https://api.litellm.ai/v1/models
→ サポートされるすべてのAIモデル一覧を返却
```

### バックエンドアーキテクチャ

- gRPCサービス経由でモデル検索
- 効率的なAPI呼び出しとキャッシング
- エラーハンドリングとフォールバック

## 💼 プロバイダーパートナーシップ

### 新しいプロバイダーを追加したいですか？

**Caretチームとのビジネス提携**を通じて、御社のAIプロバイダーをCaretに公式統合できます。

### 提携の利点

1. **自動モデル検索**: LiteLLM API連携により御社のすべてのモデルが自動表示
2. **マーケティング効果**: Caretユーザーに御社のAIサービスを露出
3. **技術サポート**: Caretチームが統合プロセスを専任サポート
4. **継続的な更新**: 新モデルリリース時に自動的にCaretに反映

### 連携プロセス

1. **お問い合わせ**: support@caretive.ai までご連絡
2. **検討**: Caretチームが技術的実現性を確認
3. **統合**: API連携とテストを実施
4. **公開**: Caret公式プロバイダーとして登録
