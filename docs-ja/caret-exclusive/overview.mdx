# Caretの独占機能

Caretは単なるClineのリブランディングではありません。Clineの堅牢な基盤の上に実質的な拡張機能を追加し、世界中の開発者により良いコーディング体験を提供します。

## 一目で比較

| 機能 | Cline | Caret |
|------|-------|-------|
| ペルソナシステム | ❌ | ✅ |
| デュアルモード (Agent/Chatbot) | ❌ | ✅ |
| プロンプト履歴 (↑↓キー) | ❌ | ✅ |
| ルール優先順位システム | ❌ | ✅ |
| 完全多言語UI | ❌ | ✅ (4言語) |
| 自動プロバイダー検索 | ❌ | ✅ |

## Caretの独占機能を探索

### 🎭 ペルソナシステム

AIの名前とアバターをカスタマイズして、本当に自分だけのコーディングパートナーを作成できます。

- 5つの組み込みペルソナ（Caret、Oh Sarang、Madobe Ichika、Cyan Macin、Tando Ubuntu）
- カスタム画像アップロード（通常状態と思考中状態）
- persona.mdファイルで性格を定義

[詳細を見る →](/ja/caret-exclusive/persona-system)

### 🔄 デュアルモードシステム

異なる作業タイプに最適化された2つのモードを切り替えます：

- **Agentモード**: 柔軟な会話フローを持つ自律的なAIパートナー
- **Chatbotモード**: 制御されたAI応答を持つ開発者主導のアドバイザー

*注：ClineはPlan/Actアプローチのみサポート*

[詳細を見る →](/ja/caret-exclusive/dual-prompt-modes)

### ⌨️ プロンプト履歴システム

ターミナルと同じように↑↓矢印キーで以前の入力を素早く再利用できます。

- 永続的なストレージ（VS Codeの再起動後も保持）
- 最大1000件の履歴エントリ
- 自動重複削除

[詳細を見る →](/ja/features/input-history)

### 📋 ルール優先順位システム

明確な優先順位で複数のルールファイルをインテリジェントにマージします：

```
優先順位: .agents/context
```

- 重複ルールを自動的にブロック（トークン節約）
- 最優先ルールのみをプロンプトに含める

[詳細を見る →](/ja/caret-exclusive/knowledge-parity-system)

### 🤝 AI-開発者の知識同期

CaretのAIは、どのようにして開発者と同じ知識を共有し、高い信頼性を確保しているのでしょうか？その背景にあるオープンソースの開発ワークフローを紹介します。このシステムは、Caret AIの透明性と信頼性の基盤となります。

[詳細を見る →](/ja/caret-exclusive/knowledge-parity-system)

### 🌍 完全な多言語サポート

UIとドキュメントの両方が完全に翻訳された真の多言語サポート：

- 韓国語、英語、日本語、中国語
- すべての設定とメッセージが翻訳済み
- 言語別のドキュメントサイト

[詳細を見る →](/ja/caret-exclusive/multilingual-ui)

### 🎨 OEMブランド切り替え（エンタープライズ）

CaretとCodeCenterのブランディングをリアルタイムで切り替えることができるシステムです。

- プライベートカスタマイズプロジェクト
- ホワイトラベルソリューション

**お問い合わせ**: support@caret.team

[詳細を見る →](/ja/caret-exclusive/brand-switching)

### 🚀 プロバイダー設定の自動化（関連機能）

AIモデルを選択する際、利用可能なモデルを手動で検索するのは面倒です。CaretはLiteLLM APIとの統合によってこのプロセスを自動化しました。

**自動モデル検索**: プロバイダーを選択すると、そのプロバイダーがサポートするすべてのAIモデルが自動的に表示されます。OpenAIを選択すると、GPT-4、GPT-3.5-Turboなどの利用可能なモデルが即座に表示されます。Anthropicを選択すると、Claudeシリーズが表示されます。

**リアルタイム更新**: LiteLLM APIから最新のモデル情報をリアルタイムで取得するため、新しいAIモデルがリリースされると、Caretの更新なしに自動的にリストに表示されます。

**広範なサポート**: OpenAI、Anthropic、Google Geminiを含む**100以上のAIプロバイダー**をサポートしています。Cohere、AI21 Labs、Hugging Faceなど、多様なプロバイダーのモデルを簡単に使用できます。

**韓国AI企業との提携**: Caretは韓国のAI企業と協力して、より多くのプロバイダーをサポートしています。韓国語に特化したAIモデルを活用したい場合、近いうちにCaretで簡単に設定できるようになります。

この自動化のおかげで、開発者は複雑な設定プロセスなしに、希望するAIモデルを素早く選択してコーディングを開始できます。

[詳細を見る →](/ja/caret-exclusive/enhanced-provider-setup)

## なぜCaretを選ぶべきか？

### すぐに開始可能

無料のGeminiクレジットでAPIキー設定なしにすぐにコーディングを開始できます（近日公開）。

### 一貫したターミナルUX

プロンプト履歴システムにより、bash/zshと同じ↑↓キー体験を提供し、開発者にとって自然な感覚です。

### あなただけのAI

ペルソナシステムでAIの名前とアバターを好みに合わせてカスタマイズできます。

### 柔軟なワークフロー

Agentモードは柔軟な会話に最適化され、Chatbotモードは開発者主導の作業に最適です。

### 母国語サポート

韓国語を含む4言語の完全なサポートで、より快適に使用できます。
