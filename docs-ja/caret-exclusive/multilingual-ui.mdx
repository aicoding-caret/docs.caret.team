# 完全な多言語UI

母国語で快適にコーディングしてください。<BrandName />はUIと文書をネイティブ言語で提供し、東アジアから順次拡大中です。

## サポート言語

<BrandName />は4つの言語を完璧にサポートします：

### 🇰🇷 韓国語 (Korean)
- **UI**: すべての設定、メニュー、メッセージがハングル化
- **文書**: docs.careti.ai/ko
- **AI応答**: 韓国語で自動応答

### 🇺🇸 英語 (English)
- **UI**: デフォルト言語
- **文書**: docs.careti.ai/en
- **AI応答**: 英語で自動応答

### 🇯🇵 日本語 (Japanese)
- **UI**: 完全な日本語翻訳
- **文書**: docs.careti.ai/ja
- **AI応答**: 日本語で自動応答

### 🇨🇳 中国語 (Chinese)
- **UI**: 簡体字中国語サポート
- **文書**: docs.careti.ai/zh
- **AI応答**: 中国語で自動応答

## 言語設定方法

### 1. 自動検出 (推奨)

<BrandName />はVS Codeの言語設定を自動的に検出します：

```json
// VS Code settings.json
{
  "locale": "ja"  // 日本語自動適用
}
```

### 2. 手動設定

<BrandName />設定で直接選択：

1. 設定を開く (⚙️)
2. **一般** タブ → **言語** セクション
3. 希望する言語を選択
4. VS Codeを再起動

### 3. コマンドパレット

```
Cmd+Shift+P (Mac) / Ctrl+Shift+P (Windows/Linux)
> Careti: Change Language
```

## AI応答言語

### 自動言語マッチング

設定したUI言語でAIが自動応答します：

**韓国語設定時**:
```
ユーザー: "React 컴포넌트 만들어줘"
AI: "Reactコンポーネントを生成します..."
```

**英語設定時**:
```
User: "Create a React component"
AI: "I'll create a React component..."
```

### 手動言語指定

特定のリクエストのみ別の言語を使用：

```
"Explain this code in English"
→ 日本語UIでも英語で応答

"このコードを日本語で説明してください"
→ 英語UIでも日本語で応答
```

## よくある質問

**Q: 言語を変更すると既存のチャット履歴は？**
A: そのまま維持されます。新しいメッセージから選択した言語で表示されます。

**Q: AIモデルがすべての言語をサポートしていますか？**
A: はい、Claude、GPT、Geminiなど主要モデルはすべて多言語をよくサポートします。

**Q: 翻訳がおかしい部分があります。**
A: GitHubにイシューを投稿してください。迅速に修正します。

**Q: 他の言語追加予定は？**
A: コミュニティリクエストに応じてスペイン語、ドイツ語などを検討中です。
