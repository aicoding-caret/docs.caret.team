**よく効く順にまとめています:**

- **bash に切り替える**

  1. Caret の右上 **Settings** をクリック
  2. 左メニューの **"Terminal Settings"** をクリック
  3. **"Default Terminal Profile"** で **"bash"** を選択

- **タイムアウトを増やす**

  1. Caret の右上 **Settings** をクリック
  2. 左メニューの **"Terminal Settings"** をクリック
  3. "Shell integration timeout (seconds)" を **10** に設定

- **ターミナル再利用を無効化**
  1. Caret の右上 **Settings** をクリック
  2. 左メニューの **"Terminal Settings"** をクリック
  3. **"Enable aggressive terminal reuse"** のチェックを外す

## プラットフォーム別

### macOS + Oh-My-Zsh

```bash
# VSCode 用の最小設定
echo 'export TERM=xterm-256color' > ~/.zshrc-vscode
echo 'export PAGER=cat' >> ~/.zshrc-vscode
```

### Windows PowerShell

```powershell
# 管理者として実行
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

### WSL

- WSL でフォルダを開く: `code .`
- **"Terminal Settings"** で **"WSL Bash"** を選択
- "Shell integration timeout (seconds)" を **15** に設定
