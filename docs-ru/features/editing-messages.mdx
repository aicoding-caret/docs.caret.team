Careti позволяет вам редактировать сообщения чата в задаче после отправки. Эта функция позволяет вам уточнить ваши запросы без запуска новой задачи, помогая вам получить лучшие результаты с минимальными нарушениями вашего рабочего процесса.

## Когда редактировать сообщения

Вы можете захотеть отредактировать сообщение, когда:

-   Вы не получили желаемые результаты
-   Вы подумали о лучшем способе сформулировать ваш запрос
-   Вам нужно добавить больше информации или контекста
-   Вы допустили опечатку или ошибку в исходном сообщении

## Как редактировать сообщения

1. Нажмите на любое сообщение в разговоре (кроме исходного сообщения задачи)
2. Отредактируйте текст по мере необходимости
3. Используйте параметры восстановления для переотправки вашего запроса

## Параметры восстановления

Когда вы редактируете сообщение, у вас есть два варианта восстановления:

### Restore Chat

Опция "Restore Chat":

-   Восстанавливает только состояние задачи
-   Переотправляет запрос API с вашим отредактированным сообщением
-   Сохраняет все изменения файла, сделанные до этого момента
-   Полезна, когда вы хотите сохранить текущее состояние вашей рабочей области

### Restore All

Опция "Restore All":

-   Восстанавливает как состояние задачи, так и состояние рабочей области
-   Переотправляет запрос API с вашим отредактированным сообщением
-   Возвращает вашу рабочую область к тому, как она была на этот момент разговора
-   Использует [контрольные точки](/ru/features/checkpoints) под капотом для восстановления вашей рабочей области
-   Полезна, когда вы хотите попробовать совершенно другой подход

## Клавиатурные сокращения

При редактировании сообщения вы можете использовать эти клавиатурные сокращения:

-   **Escape**: Выйти из режима редактирования без изменений
-   **Enter**: Восстановить только задачу (эквивалентно "Restore Chat")
-   **Cmd/Ctrl + Enter**: Восстановить задачу и рабочую область (эквивалентно "Restore All")
-   **Shift + Enter**: Вставить новую строку / разрыв строки в вашем сообщении

## Лучшие практики

-   Используйте редактирование сообщений для небольших корректировок ваших запросов
-   Для основных изменений направления рассмотрите запуск новой задачи
-   При использовании "Restore All" имейте в виду, что любые изменения файла, сделанные после этого сообщения, будут отменены
-   Редактируйте сообщения ближе к началу разговора, чтобы избежать потери значительного прогресса
