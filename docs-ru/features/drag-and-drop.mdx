Перетаскивание файлов в Caret — это быстрый способ добавить изображения, код и другие файлы в ваши разговоры.

<Frame>
	<img
		src="https://storage.googleapis.com/cline_public_images/docs/assets/dran-n-drop.gif"
		alt="Перетаскивание файлов в чат Caret"
	/>
</Frame>

<Note>Из-за особенностей VS Code, чтобы перетащить файлы в ввод чата Caret, вам нужно держать `Shift` во время перетаскивания.</Note>

Перетаскивание файлов рабочей области в Caret автоматически создаст [упоминание файла](/ru/features/at-mentions/file-mentions). Это позволяет вам ссылаться на файл в вашем разговоре без необходимости печатания пути.

### Перетаскивание из Finder/File Explorer

Вы можете перетащить файлы прямо из файлового менеджера вашей системы в Caret:

<Frame>
	<img
		src="https://storage.googleapis.com/cline_public_images/docs/assets/drag-n-drop-finder.gif"
		alt="Перетаскивание файлов из Finder в Caret"
	/>
</Frame>

### Поддерживаемые типы файлов

Caret поддерживает перетаскивание внешних изображений, PDF-файлов, CSV, Excel и других текстовых файлов из вашей файловой системы, а также файлов из вашей рабочей области.
