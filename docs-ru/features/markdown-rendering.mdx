# Рендеринг Markdown

<BrandName /> полностью поддерживает GitHub Flavored Markdown (GFM) в ответах ИИ. Таблицы, зачёркнутый текст, блоки кода и многое другое красиво отображаются, делая информацию более понятной.

## Поддерживаемые Функции

### Таблицы GFM

Когда ИИ отвечает сравнительными таблицами или списками опций, они отображаются в структурированном формате.

```markdown
| Функция | Cline | Careti |
|---------|-------|--------|
| Таблицы GFM | ❌ | ✅ |
| Зачёркивание | ❌ | ✅ |
| Mermaid | ✅ | ✅ |
```

**Результат рендеринга:**

| Функция | Cline | Careti |
|---------|-------|--------|
| Таблицы GFM | ❌ | ✅ |
| Зачёркивание | ❌ | ✅ |
| Mermaid | ✅ | ✅ |

### Выравнивание Таблиц

Вы можете указать выравнивание столбцов в таблицах:

- `:---` - По левому краю
- `:---:` - По центру
- `---:` - По правому краю

### Зачёркивание

Используйте формат `~~текст~~` для отображения зачёркнутого текста.

```markdown
~~Этот текст удалён~~
```

**Результат рендеринга:** ~~Этот текст удалён~~

### Диаграммы Mermaid

Блок-схемы, диаграммы последовательности и многое другое визуально отображаются.

### Подсветка Кода

Поддерживается подсветка синтаксиса для различных языков программирования.

## Отличия от Cline

<BrandName /> добавляет поддержку расширений GFM, которая недоступна в оригинальном Cline:

| Функция | Cline | <BrandName /> |
|---------|-------|---------------|
| Базовый Markdown | ✅ | ✅ |
| Таблицы GFM | ❌ сырой текст | ✅ рендеринг таблиц |
| Выравнивание таблиц | ❌ | ✅ |
| Зачёркивание | ❌ сырой текст | ✅ рендеринг зачёркивания |
| Mermaid | ✅ | ✅ |
| Подсветка кода | ✅ | ✅ |

## Техническая Информация

Эта функция реализована с использованием плагина `remark-gfm`. Версия `remark-gfm@1.0.0` используется для совместимости с `react-remark`.
