Команды для кода Caret переносят помощь AI непосредственно в ваш редактор, позволяя вам взаимодействовать с вашим кодом, не покидая рабочий процесс. С помощью простого щелчка правой кнопкой мыши вы можете добавить код в Caret, а с помощью меню лампочки вы можете исправлять ошибки, получать объяснения или улучшать свой код.

## Доступные команды для кода

Когда вы взаимодействуете с кодом в вашем редакторе, вы можете получить доступ к командам Caret двумя способами:

### Контекстное меню по щелчку правой кнопкой мыши

Когда вы щелкаете правой кнопкой мыши по выделенному коду, вы увидите:

	<img src="https://storage.googleapis.com/cline_public_images/docs/assets/code-commands.png" alt="Right Click Menu" />
</Frame>

#### Add to Caret

Команда "Add to Caret" отправляет выбранный вами код на панель чата Caret. Это идеально подходит для:

-   Задания вопросов о конкретных фрагментах кода
-   Запроса улучшений или оптимизаций
-   Получения объяснений сложной логики

Когда вы используете эту команду, Caret автоматически включает:

-   Путь к файлу (в виде упоминания файла)
-   Выбранный код с правильным форматированием
-   Язык программирования для точной подсветки синтаксиса

### Меню лампочки (Code Actions)

Когда вы видите значок лампочки в вашем редакторе, щелкните его, чтобы получить доступ к этим командам Caret:

<Frame>
	<img src="https://storage.googleapis.com/cline_public_images/docs/assets/lightbulb-actions.png" alt="Lightbulb Menu" />
</Frame>

#### Fix with Caret

Команда "Fix with Caret" появляется в меню лампочки, когда в вашем коде есть ошибки или предупреждения. Эта команда:

1. Захватывает выбранный код
2. Определяет ошибки или предупреждения из diagnostics VSCode
3. Отправляет и то, и другое в Caret с запросом на исправление проблем
4. Предоставляет решение, которое решает конкретные проблемы

Это невероятно полезно для быстрого устранения синтаксических ошибок, предупреждений линтера или проблем с типами без необходимости вручную описывать проблему.

#### Explain with Caret

Команда "Explain with Caret" помогает вам понять сложный код. Когда вы выбираете код и используете эту команду из меню лампочки, Caret:

1. Анализирует выбранный код
2. Предоставляет четкое объяснение того, что делает код
3. Разбивает сложную логику на понятные части
4. Выделяет важные используемые шаблоны или методы

#### Improve with Caret

Команда "Improve with Caret" помогает вам улучшить ваш код. Когда вы выбираете код и используете эту команду из меню лампочки, Caret:

1. Анализирует выбранный код на предмет потенциальных улучшений
2. Предлагает оптимизации, рефакторинги или лучшие практики
3. Объясняет причины предложенных изменений
4. Предоставляет улучшенный код, который поддерживает исходную функциональность

## Как использовать команды для кода

Использовать команды для кода Caret очень просто:

### Для команд по щелчку правой кнопкой мыши:

1. Выберите код, с которым вы хотите работать
2. Щелкните правой кнопкой мыши, чтобы открыть контекстное меню
3. Выберите "Add to Caret"
4. Просмотрите результат на панели чата Caret

### Для команд меню лампочки:

1. Выберите код, с которым вы хотите работать
2. Найдите значок лампочки, который появляется на поле редактора
3. Щелкните лампочку, чтобы увидеть доступные действия
4. Выберите соответствующую команду Caret (Fix, Explain или Improve)
5. Просмотрите результат на панели чата Caret

После использования любой команды вы можете:

-   Задать дополнительные вопросы
-   Запросить изменения в решении
-   Применить изменения обратно к вашему коду

## Как это работает под капотом

Когда вы используете команду для кода, вот что происходит за кулисами:

1. **Выбор кода**: Расширение захватывает выбранный вами код и его контекст
2. **Сбор метаданных**: Caret собирает важные метаданные:

    - Путь и имя файла
    - Язык программирования
    - Любые связанные diagnostics (ошибки/предупреждения)
    - Окружающий контекст кода, когда это уместно

3. **Обработка команд**:

    - Для "Add to Caret" код форматируется и отправляется на панель чата
    - Для "Fix with Caret" код и diagnostics анализируются и генерируется исправление
    - Для "Explain with Caret" код анализируется для предоставления четкого объяснения
    - Для "Improve with Caret" код анализируется на предмет потенциальных оптимизаций и улучшений

4. **Интеграция с чатом**: Результаты появляются на панели чата Caret, где вы можете:
    - Увидеть ответ ИИ
    - Задать дополнительные вопросы
    - Применить предложенные изменения

Эта плавная интеграция между вашим редактором и возможностями AI Caret позволяет легко получить помощь, не нарушая ваш процесс кодирования.

## Советы по эффективному использованию

-   **Выбирайте полные логические единицы**: Когда это возможно, выбирайте целые функции, классы или модули, чтобы предоставить Caret полный контекст
-   **Включайте импорты**: Для языковой помощи включайте соответствующие импорты, чтобы Caret понимал зависимости
-   **Комбинируйте с @ mentions**: Для сложных проблем используйте команды для кода вместе с упоминаниями файлов или проблем для получения дополнительного контекста
-   **Используйте сочетания клавиш**: Ускорьте свой рабочий процесс, [назначив сочетания клавиш](/russian/features/commands-and-shortcuts/keyboard-shortcuts) для общих команд для кода

В следующий раз, когда вы столкнетесь с проблемой с фрагментом кода, попробуйте использовать команды для кода Caret вместо переключения на отдельный интерфейс чата. Вы будете поражены тем, насколько более эффективным станет ваш рабочий процесс, когда помощь AI будет интегрирована непосредственно в ваш редактор.
