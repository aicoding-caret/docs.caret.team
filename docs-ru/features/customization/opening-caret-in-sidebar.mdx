По умолчанию, при первой установке Caret появляется в левой боковой панели VS Code вместе с проводником файлов и другими расширениями. Однако для улучшения опыта программирования мы рекомендуем переместить Caret в правую боковую панель. Это позволит вам сохранить видимость файлов проекта в левой боковой панели, одновременно общаясь с Caret справа, предоставляя вам полный обзор вашей кодовой базы, пока Caret работает над вашим проектом.

## VS Code

Чтобы открыть Caret в правой боковой панели:

<Steps>
  <Step title="Выровняйте представление расширения">
    Убедитесь, что представление вашего расширения выровнено по вертикали слева
  </Step>
  <Step title="Откройте правую боковую панель">
    Нажмите кнопку, которая открывает правую боковую панель в VS Code (обычно используется для открытия чата GitHub Copilot). При желании используйте сочетание клавиш `Option + CMD/Ctrl + B`.
  </Step>
  <Step title="Перетащите иконку Caret">
    Перетащите иконку Caret на панель навигации в верхней части этой правой панели
  </Step>
</Steps>

## Cursor

Чтобы открыть Caret в правой боковой панели:

<Steps>
  <Step title="Выровняйте расширения">
    Cursor по умолчанию использует горизонтальную панель действий, чтобы оптимизировать пространство для интерфейса чата AI ([подробности здесь](https://cursor.com/docs/configuration/migrations/vscode#activity-bar-orientation)). Чтобы переключиться на вертикальную:
    
    1. Откройте Command Palette (`CMD/Ctrl + Shift + P`)
    2. Найдите "Preferences: Open Settings (UI)"
    3. Найдите `workbench.activityBar.orientation`
    4. Установите значение `vertical`
    5. Перезапустите Cursor, чтобы изменения вступили в силу
  </Step>
  <Step title="Откройте AI Pane">
    Нажмите кнопку с иконкой куба Cursor (AI Pane), которая открывает агента Cursor (правую боковую панель)
  </Step>
  <Step title="Перетащите Caret в боковую панель AI Pane">
    Перетащите иконку Caret прямо в боковую панель AI Pane.
  </Step>
</Steps>
