# Управление задачами

По мере использования Caret вы будете накапливать много задач со временем. Система управления задачами помогает вам организовывать, фильтровать, искать и очищать историю задач, чтобы ваше рабочее пространство оставалось эффективным.

## Доступ к истории задач

Вы можете открыть историю задач двумя способами:

1.  Нажмите кнопку **History** на боковой панели Caret
2.  Используйте Command Palette и найдите **"Caret: Show Task History"**

## Функции истории задач

Представление истории задач включает несколько мощных инструментов.

### Поиск и фильтрация

-   **Строка поиска**: Используйте нечёткий поиск в верхней части для быстрого поиска задач по содержимому.
-   **Параметры сортировки**: Сортируйте задачи по:
    -   Самые последние (по умолчанию)
    -   Самые старые
    -   Самые дорогие (наивысшая стоимость API)
    -   Наибольшее количество токенов (наивысшее использование токенов)
    -   Наиболее релевантные (при поиске)
-   **Фильтр избранного**: Переключитесь для показа только избранных задач.

### Действия с задачами

Каждая задача в представлении истории поддерживает эти действия:

-   **Открыть**: Щелкните по задаче, чтобы открыть её снова в чате Caret.
-   **Избранное**: Щелкните значок звёздочки, чтобы отметить задачу как избранную.
-   **Удалить**: Удалите отдельную задачу (избранные задачи защищены).
-   **Экспортировать**: Экспортируйте разговор в виде Markdown.

## Добавление задач в избранное

Избранное помогает вам держать важные задачи в легком доступе и сохранять их.

### Как работает избранное

-   **Переключить избранное**: Щелкните значок звёздочки рядом с любой задачей.
-   **Защита**: Избранные задачи защищены от отдельного и массового удаления (можно переопределить).
-   **Фильтрация**: Используйте фильтр избранного для быстрого доступа к важным задачам.

## Массовые действия

Представление истории задач поддерживает несколько действий группировки.

-   **Множественный выбор**: Используйте флажки для выбора нескольких задач.
-   **Выбрать всё / Деселектировать всё**: Быстро выберите или очистите выделение.
-   **Удалить выбранные**: Удалите все выбранные задачи.
-   **Удалить всё**: Удалите все задачи из истории (избранное сохраняется, если явно не включено).

## Лучшие практики

1.  **Отмечайте важные задачи**: Отмечайте задачи, которые служат справочником или часто используются в разговорах.
2.  **Регулярно очищайте**: Удаляйте старые или неиспользуемые задачи для улучшения производительности.
3.  **Используйте поиск**: Используйте нечёткий поиск для быстрого поиска конкретных разговоров.
4.  **Экспортируйте ценные задачи**: Сохраняйте важные задачи в виде Markdown для внешней ссылки.

Управление задачами помогает вам сохранять организованный рабочий процесс в Caret — быстро находить прошлые разговоры, сохранять важные задачи и держать вашу историю чистой и эффективной.
