Готовы увидеть Careti в действии? Это практическое руководство проведёт вас через создание веб-сайта за минуту. Вы убедитесь, как Careti понимает ваши требования, создаёт файлы и итерирует вашу обратную связь — всё через естественный разговор.

К концу этого руководства вы создадите рабочий веб-сайт и узнаете основы работы с Careti.

## Предварительные требования

- **Careti установлен** в вашем редакторе ([Руководство по установке](/ru/getting-started/installing-caret))
- **AI модель выбрана** ([Настройка модели](/ru/getting-started/selecting-your-model))
- **Любая папка открыта** в вашем редакторе (или создайте новую пустую папку)

## Шаг 1: Откройте Careti

Нажмите на значок Careti на боковой панели вашего редактора (левая сторона). Панель чата откроется.

<Info>
**Быстрый совет:** Вы также можете использовать `Cmd+Shift+P` (Mac) или `Ctrl+Shift+P` (Windows/Linux) и выполнить поиск "Careti: Open In New Tab"
</Info>

## Шаг 2: Дайте Careti задачу

Скопируйте и вставьте это приглашение в чат Careti:

```
Create a simple website in a single HTML file. It should have:
- A welcome message saying "Hello from Careti!"
- A colorful gradient background
- A button that cycles through different color themes when clicked
- Modern, clean design
- All CSS and JavaScript should be included in the same HTML file
```

<Frame>
  <img src="https://storage.googleapis.com/cline_public_images/chat-prompt.png" alt="Careti Chat Prompt"/>

Нажмите Enter и смотрите как работает Careti!

## Шаг 3: Что произойдёт дальше

Careti будет:

1. **Создать один файл:**
   - `index.html` - Полная веб-страница со встроенным CSS и JavaScript

2. **Попросить одобрение** (если вы не включили автоматическое одобрение)
   - Нажмите "Approve" чтобы позволить Careti создать файл
   - Вы можете сначала проверить, что он планирует сделать

3. **Завершить задачу** в течение нескольких секунд

## Шаг 4: Просмотрите ваш веб-сайт

После завершения работы Careti:

1. **Найдите `index.html`** в обозревателе файлов вашего редактора
2. **Щелкните правой кнопкой мыши** и выберите:
   - "Reveal in Finder/Explorer" затем дважды щелкните для открытия в вашем браузере
3. **Нажмите кнопку** чтобы увидеть как меняются цветовые темы!

## Попробуйте сделать изменения

В том же чате попробуйте попросить:

```
Add a counter that shows how many times the button has been clicked
```

или

```
Make the welcome message fade in when the page loads
```

Careti понимает контекст из вашего предыдущего разговора и обновит файл соответственно.
