Careti нуждается в AI модели, чтобы понять ваши запросы и написать код. Думайте об этом как о выборе, с каким экспертом работать - разные модели имеют разные сильные стороны и стоимость.

## Быстрый старт: выберите свой провайдер

Самый простой способ начать работу - использовать **Careti** как ваш провайдер:

1. **Откройте настройки Careti**: Нажмите значок шестеренки (⚙️) в верхнем правом углу чата Careti
2. **Выберите "Careti"** из выпадающего списка поставщиков API
3. **Выберите модель** из выпадающего списка - мы рекомендуем начать с **Claude Sonnet 4.5** или **DeepSeek V3**

<Frame>
  <img src="https://storage.googleapis.com/cline_public_images/step2-provider.png" alt="Выберите провайдера Careti" />
</Frame>

**Вот и все!** Никаких ключей API для управления, и вы получаете доступ к множеству моделей.

<Tip>
**Доступны бесплатные модели**: Careti иногда предлагает бесплатное выведение через партнерских провайдеров. Когда они доступны, вы увидите эти опции в своем выпадающем списке моделей.
</Tip>

## Альтернатива: используйте другого провайдера

Если вы предпочитаете использовать свои собственные ключи API, вы можете выбрать из провайдеров, таких как:

- **OpenRouter** - отличное соотношение цены и качества, множество моделей
- **Anthropic** - прямой доступ к моделям Claude
- **OpenAI** - доступ к моделям GPT
- **Google Gemini** - AI модели Google
- **Ollama** - запускайте модели локально на вашем компьютере

После выбора провайдера вам потребуется:
1. Получить ключ API с их веб-сайта
2. Вставить его в поле API Key в настройках Careti
3. Выбрать свою модель

<Note>
Большинство провайдеров требуют информацию об оплате перед генерацией ключей API.
</Note>

## Какую модель мне выбрать?

Если вы только начинаете, мы рекомендуем:

| Ваш приоритет | Выберите эту модель | Почему |
|-------------|-------------------|-----|
| **Надежность** | Claude Sonnet 4.5 | Наиболее надежна для задач кодирования |
| **Стоимость** | DeepSeek V3 | Отличная производительность при низкой стоимости |
| **Скорость** | Qwen3 Coder | Быстрые ответы |
| **Конфиденциальность** | Любая модель Ollama | Запускается на вашем компьютере |

Вы можете переключаться между моделями в любое время без потери вашего диалога.
