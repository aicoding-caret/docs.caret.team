> **Совет:** Если кодирование для вас совершенно ново, проделайте каждый шаг медленно. Не нужно спешить — Caret будет вас направлять.

### Начало работы

Перед началом кодирования подготовьте следующие необходимые вещи.

#### 1. **VS Code**

Популярный, бесплатный и мощный редактор кода.

-   [<u>Загрузите VS Code</u>](https://code.visualstudio.com/)

**Рекомендуемое видео на YouTube:** [<u>Как установить VS Code</u>](https://www.youtube.com/watch?v=MlIzFUI1QGA)

> **Совет от профессионала:** Установите VS Code в папку Applications (macOS) или Program Files (Windows), чтобы он был легко доступен из Dock или меню "Пуск".

#### 2. **Организация проектов**

Создайте папку `Caret` в папке Documents для всех ваших проектов по кодированию.

-   **macOS:** `/Users/[Ваше имя]/Documents/Caret`
-   **Windows:** `C:\Users\[Ваше имя]\Documents\Caret`

Организуйте проекты четко внутри папки `Caret`.

-   `Documents/Caret/workout-app` _(например, приложение трекер фитнеса)_
-   `Documents/Caret/portfolio-website` _(например, веб-сайт портфолио)_

> **Совет:** Организованность с самого начала сэкономит вам время и путаницу позже.

#### 3. **Установите расширение Caret для VS Code**

Установите расширение Caret прямо в VS Code, чтобы улучшить свой рабочий процесс.

-   Учебное пособие по началу работы расширения Caret

**Рекомендуемое видео на YouTube:** [<u>Как установить расширения VS Code</u>](https://www.youtube.com/watch?v=E7trgwZa-mk)

> **Совет от профессионала:** После установки перезагрузите VS Code, чтобы убедиться, что расширение полностью активировано.

#### 4. **Необходимые инструменты разработки**

Основное программное обеспечение, которое вам понадобится для эффективного кодирования:

-   Homebrew (macOS)
-   Node.js
-   Git

[<u>Следуйте пошаговому руководству Cline по установке необходимых инструментов разработки.</u>](https://docs.caret.bot/ru/getting-started/installing-dev-essentials#installing-dev-essentials)

    -   [<u>Установите Homebrew на Mac</u>](https://www.youtube.com/watch?v=hwGNgVbqasc)

> **Примечание:** Если у вас возникнут проблемы с разрешениями при установке, попробуйте запустить Terminal или Command Prompt с правами администратора.

Вы готовы! Начните кодировать умнее и быстрее с помощью **Caret**.
