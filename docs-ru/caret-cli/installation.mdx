## Предварительные требования

Careti CLI требует Node.js версии 20 или выше. Мы рекомендуем использовать Node.js 22 для наилучшего опыта.

Чтобы проверить вашу версию Node.js:

```bash
node --version
```

## Установка

```bash
npm install -g caret
```

После установки выполните аутентификацию в своей учетной записи Careti:

```bash
caret auth
```

Это запустит мастер аутентификации для входа в систему и настройки предпочтительного поставщика моделей AI.

<Note>
Careti CLI не выполняет автоматическую аутентификацию при установке. В отличие от типичных CLI, вы должны явно запустить `caret auth`.
`caret config set ...` предназначен для настроек провайдера и не заменяет вход в учетную запись.
</Note>

## Быстрый старт

Начните работу с Careti за считанные секунды:

```bash
caret
```

Вот и все! Запуск `caret` в любом каталоге начинает интерактивный сеанс, в котором вы можете общаться с AI agent. Введите свою задачу, просмотрите план и введите `/act`, когда будете готовы к выполнению.

Для еще более быстрого выполнения без взаимодействия:

```bash
caret "Add unit tests to utils.js"
```

Это запускает Careti с одной командой, что идеально подходит для быстрых задач или автоматизации.

<Tip>
Новичок в Careti CLI? Начните с интерактивного режима (`caret`), чтобы увидеть, как это работает. Когда освоитесь, изучите [три основных рабочих процесса](/russian/caret-cli/three-core-flows) для расширенных вариантов использования.
</Tip>
