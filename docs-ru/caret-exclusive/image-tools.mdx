# ^ Image Tools

ИИ Careti может **генерировать** и **анализировать** изображения.

## Отличия от Cline

| Функция | Cline | Careti |
|------|-------|-------|
| Генерация изображений | Не поддерживается | **Поддерживается через инструмент generate_image** |
| Анализ изображений (модели без vision) | Не поддерживается | **Поддерживается через инструмент analyze_image** |
| Настройки соотношения сторон/размера | Не поддерживается | **Настраивается в UI** |
| Генерация на основе эталонного изображения | Не поддерживается | **Поддержка Image-to-Image** |

## Генерация изображений

### Пример использования

```
User: Create a cute cat image
AI: [Uses generate_image tool] → Generates image → Saves to assets/
```

### Опции конфигурации

| Опция | Значения | Описание |
|------|-----|------|
| **Ratio** | `16:9`, `9:16`, `4:3`, `3:4`, `1:1` | Соотношение сторон изображения |
| **Size** | `1K`, `2K`, `3K`, `4K` | Разрешение изображения |

Расположение настроек: Settings > Image Generation

### Место сохранения

Сгенерированные изображения сохраняются в папку проекта `assets/`:

```
assets/
├── img_abc123.png        # Generated image
└── img_abc123.md         # Metadata (prompt, settings, etc.)
```

## Анализ изображений

### Vision-модели против текстовых моделей

| Тип модели | Метод анализа изображений |
|----------|----------------|
| **Vision-модели** (GPT-4o, Claude 3.5 и т. д.) | Прямой анализ через `read_file` |
| **Текстовые модели** (o1, GLM-4 и т. д.) | Использует инструмент `analyze_image` |

### Пример использования

```
# Non-vision model (GLM-4.7, etc.)
User: [Attaches image] What's in this image?
AI: [Uses analyze_image tool] → Analyzes with Gemini Flash → Returns results
```

### Выбор модели для анализа

При использовании текстовых моделей вы можете выбрать модель для анализа изображений:

- `gemini-2.5-flash` (по умолчанию)
- `gemini-3.0-flash-preview`

Расположение настроек: Settings > Model Info > Image Analysis Model

## Требования к аутентификации

| Инструмент | Требования |
|------|----------|
| `generate_image` | Требуется вход в Careti |
| `analyze_image` | Требуется вход в Careti (только для текстовых моделей) |

## Конфигурация

### Настройки авто-одобрения (Auto-approve)

| Настройка | По умолчанию | Описание |
|------|--------|------|
| `generateImages` | `true` | Включить инструмент генерации изображений |
| `analyzeImages` | `true` | Включить инструмент анализа изображений |

Расположение настроек: Settings > Auto-approve

## Безопасность

### Одобрение анализа изображений

| Расположение файла | Поведение |
|-----------|------|
| Внутри рабочего пространства | ✅ Одобрено автоматически |
| Вне рабочего пространства | ⚠️ Требуется одобрение пользователя |

### Форматы изображений

Поддерживаемые расширения изображений:
- `.png`, `.jpg`, `.jpeg`, `.webp`
- `.gif`, `.avif`, `.bmp`, `.tiff`

## Известные ограничения

1. **Требуется аутентификация**: Для работы инструментов с изображениями необходим вход в Careti
2. **Одиночное выполнение**: Инструменты для работы с изображениями могут запускаться только по одному за раз
3. **Лимит размера**: Максимальный размер изображения составляет 7500px
