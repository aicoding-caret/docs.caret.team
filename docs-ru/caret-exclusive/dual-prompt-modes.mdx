# Система двойного режима

Caret сохраняет workflow Plan/Act от Cline, добавляя при этом свой собственный опыт **Chatbot/Agent**. Выберите режим, который соответствует вашему желанию в автономии и контроле.

## Как это работает

- **Режим Agent (Caret)**: Полная автономия. Читает/записывает файлы и выполняет команды, но вы можете прервать его и задать вопросы в любое время. Представьте себе «парного программиста, который может продолжать кодировать после того, как вы что-то уточнили».
- **Режим Chatbot (Caret)**: Только руководство. Читает/ищет/отвечает на вопросы; запись и выполнение команд заблокированы по умолчанию, чтобы вы оставались за рулем.
- **Режим Plan/Act (Cline)**: Исходный двухэтапный процесс. Plan запрашивает одобрение, Act выполняет, пока вы его не остановите.
- **Замена терминологии**: В режиме Caret вы видите метки Chatbot/Agent; переключение в режим Cline показывает привычные термины и поведение Plan/Act.

## Система подсказок и защитные барьеры

- **JSON-управляемые подсказки**: Caret собирает системные подсказки для каждого режима, поэтому тон и руководство соответствуют Chatbot и Agent.
- **Ограничения инструментов по режиму**: Agent может использовать все инструменты. Chatbot предназначен только для чтения/поиска/вопросов; запись/выполнение заблокированы для предотвращения непреднамеренных изменений.
- **Переключайтесь в любое время**: Изменяйте режимы в настройках; поведение и метки обновляются немедленно.
