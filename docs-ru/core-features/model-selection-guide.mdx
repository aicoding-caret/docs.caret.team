## Что такое AI Model?

Представьте AI Model как "мозг", который питает Careti. Когда вы просите Careti написать код, исправить ошибки или рефакторить ваш проект, именно модель понимает ваш запрос и генерирует ответ.

**Основные моменты:**
- **Models - это обученные AI системы**, которые понимают естественный язык и код
- **Разные models имеют разные сильные стороны**: некоторые превосходны в сложных рассуждениях, другие уделяют приоритетное внимание скорости или стоимости
- **Вы выбираете, какую model использует Careti**, как выбор между разными экспертами для разных задач
- **Доступ к Models осуществляется через API providers** - такие компании, как Anthropic, OpenAI и OpenRouter, размещают эти models

**Почему это важно:** Model, которую вы выберете, напрямую влияет на возможности Careti, качество ответа, скорость и стоимость. Premium model может прекрасно справляться со сложным рефакторингом, но стоить дороже, в то время как бюджетная model отлично подходит для рутинных задач за небольшую часть цены.

## Как выбрать Model в Careti

Выполните следующие 5 простых шагов, чтобы запустить Careti с предпочитаемой вами AI model:

### Шаг 1: Откройте настройки Careti

Сначала вам нужно получить доступ к панели конфигурации Careti.

**Два способа открыть настройки:**
- **Быстрый метод**: Нажмите **значок шестеренки (⚙️)** в правом верхнем углу интерфейса чата Careti.
- **Палитра команд**: Нажмите **Cmd/Ctrl + Shift + P** → введите "Careti: Open Settings"

<Frame>
  <img src="https://storage.googleapis.com/cline_public_images/step1-config.png" alt="Careti Settings Panel" />
</Frame>

Откроется панель настроек, показывающая параметры конфигурации с "API Provider" вверху.

<Note>
Панель настроек запоминает вашу последнюю конфигурацию, поэтому вам нужно будет настроить это только один раз.
</Note>

### Шаг 2: Выберите API Provider

Выберите предпочитаемого AI provider из выпадающего меню.

<Frame>
  <img src="https://storage.googleapis.com/cline_public_images/step2-provider.png" alt="Careti Settings Panel" />
</Frame>

**Популярные providers вкратце:**

| Provider | Лучше всего подходит для | Примечания |
|----------|----------|-------|
| **Careti** | Самая простая настройка | Не требуются API keys, доступ к нескольким models, включая stealth models |
| **OpenRouter** | Искатели выгоды | Несколько models, конкурентоспособные цены |
| **Anthropic** | Надежность | Claude models, самое надежное использование инструментов |
| **OpenAI** | Новейшие технологии | GPT models |
| **Google Gemini** | Большой контекст | Google's AI models |
| **AWS Bedrock** | Для предприятий | Расширенные функции |
| **Ollama** | Конфиденциальность | Запуск models локально |

См. [полный список providers](/russian/provider-config/openrouter) для получения дополнительных параметров, включая Cerebras, Vertex AI, Azure и другие.

<Info>
**Рекомендуется для начинающих:** Начните с **Careti** в качестве своего provider - не требуется управление API key, мгновенный доступ к нескольким models и случайный бесплатный вывод через партнерских providers.
</Info>

### Шаг 3: Добавьте свой API Key (или войдите в систему)

Следующий шаг зависит от того, какого provider вы выбрали.

#### Если вы выбрали **Careti** в качестве своего provider:

- **API key не требуется!** Просто войдите в свою учетную запись Careti
- Нажмите кнопку **Sign In**, когда будет предложено
- Вы будете перенаправлены на [app.caret.bot](https://app.caret.bot) для аутентификации
- После входа в систему вернитесь в свою IDE

#### Если вы выбрали любого другого provider:

Вам нужно будет получить API key от выбранного вами provider:

1. **Посетите веб-сайт своего provider, чтобы получить API key:**
   - **Anthropic**: [console.anthropic.com](https://console.anthropic.com/)
   - **OpenRouter**: [openrouter.ai/keys](https://openrouter.ai/keys)
   - **OpenAI**: [platform.openai.com/api-keys](https://platform.openai.com/api-keys)
   - **Google**: [aistudio.google.com/apikey](https://aistudio.google.com/apikey)
   - **Другие**: См. [Руководство по настройке Provider](/russian/provider-config/openrouter)

2. **Сгенерируйте новый API key** на веб-сайте provider

3. **Скопируйте API key** в буфер обмена

4. **Вставьте свой key** в поле **"API Key"** в настройках Careti

5. **Автоматическое сохранение** - Ваш key надежно хранится в хранилище секретов вашего редактора

<Frame>
  <img src="https://storage.googleapis.com/cline_public_images/step3-API.png" alt="Careti API Selection" />
</Frame>

<Warning>
**Для большинства providers требуется оплата**: Большинству providers требуется платежная информация перед генерацией keys. Вы платите только за то, что используете (обычно 0,01-0,10 доллара США за задачу кодирования).
</Warning>

### Шаг 4: Выберите свою Model

После того, как ваш API key добавлен (или вы вошли в систему), становится доступным раскрывающееся меню **"Model"**.

<Frame>
  <img src="https://storage.googleapis.com/cline_public_images/step4-model.png" alt="Careti Model Selection" />
</Frame>

**Краткое руководство по выбору model:**

| Ваш приоритет | Выберите эту Model | Почему |
|---------------|-------------------|-----|
| **Максимальная надежность** | Claude Sonnet 4.5 | Самое надежное использование инструментов, отлично справляется со сложными задачами |
| **Лучшее соотношение цены и качества** | DeepSeek V3 или Qwen3 Coder | Отличная производительность по бюджетным ценам |
| **Самая высокая скорость** | Qwen3 Coder на Cerebras | Молниеносные ответы |
| **Запуск локально** | Любая Ollama model | Полная конфиденциальность, не требуется интернет |
| **Новейшие функции** | GPT-5 | Новейшие возможности OpenAI |

Не знаете, что выбрать? Начните с **Claude Sonnet 4.5** для надежности или **DeepSeek V3** для соотношения цены и качества.

<Tip>
Вы можете переключать models в любое время, не теряя разговор. Попробуйте разные models, чтобы найти то, что лучше всего подходит для ваших конкретных задач.
</Tip>

## Выбор правильной Model

Выбор правильной model включает в себя балансировку нескольких факторов. Используйте эту структуру, чтобы найти свой идеальный вариант:

<Note>
**Профессиональные советы**: Настройте отдельные models для Plan Mode и Act Mode. Максимально используйте сильные стороны каждой model. Например, используйте бюджетную model для обсуждений планирования и premium model для реализации.
</Note>

### Ключевые факторы выбора

| Фактор | Что следует учитывать | Рекомендация |
|--------|------------------|----------------|
| **Сложность задачи** | Простые исправления по сравнению со сложным рефакторингом | Бюджетные models для рутинных задач; Premium models для сложной работы |
| **Бюджет** | Ежемесячный лимит расходов | 10-30 долларов США: Бюджетные, 30-100 долларов США: Средний уровень, 100 долларов США и более: Premium |
| **Context Window** | Размер проекта и количество файлов | Небольшой: 32K-128K, Средний: 128K-200K, Большой: 400K+ |
| **Скорость** | Требования ко времени ответа | Интерактивный: быстрые models, Фоновый: Reasoning models OK |
| **Надежность инструментов** | Сложные операции | Claude превосходен в использовании инструментов; Протестируйте другие в своем рабочем процессе |
| **Provider** | Потребности в доступе и ценах | OpenRouter: Много вариантов, Прямой: Быстрее/надежнее, Локальный: Конфиденциальность |

## Open Source vs Closed Source

### Преимущества Open Source
- **Несколько providers** конкурируют за их размещение
- **Более низкие цены** из-за конкуренции
- **Выбор Provider** - переключитесь, если один выйдет из строя
- **Более быстрые циклы инноваций**

### Доступные Models Open Source
- **Qwen3 Coder** (Apache 2.0)
- **Z AI GLM 4.5** (MIT)
- **Kimi K2** (Open source)
- **Серия DeepSeek** (Различные лицензии)
