Careti поддерживает доступ к моделям через DeepSeek API, включая `deepseek-chat` и `deepseek-reasoner`.

**Веб-сайт:** [https://platform.deepseek.com/](https://platform.deepseek.com/)

### Получение API Key

1.  **Регистрация/Вход:** Перейдите на [DeepSeek Platform](https://platform.deepseek.com/). Создайте аккаунт или войдите в систему.
2.  **Перейдите к API Keys:** Найдите свои API Keys в разделе [API keys](https://platform.deepseek.com/api_keys) платформы.
3.  **Создайте ключ:** Нажмите «Create new API key». Дайте вашему ключу понятное имя (например, «Careti»).
4.  **Скопируйте ключ:** **Важно:** Скопируйте API Key _немедленно_. Вы не сможете увидеть его снова. Храните его в безопасном месте.

### Поддерживаемые модели

Careti поддерживает следующие модели DeepSeek:

-   `deepseek-v3-0324` (рекомендуется для задач программирования)
-   `deepseek-r1` (рекомендуется для задач, требующих рассуждений)

### Настройка в Careti

1.  **Откройте Careti Settings:** Нажмите на иконку ⚙️ на панели Careti.
2.  **Выберите Provider:** Выберите «DeepSeek» в выпадающем списке «API Provider».
3.  **Введите API Key:** Вставьте ваш DeepSeek API Key в поле «DeepSeek API Key».
4.  **Выберите Model:** Выберите нужную модель в выпадающем списке «Model».

### Советы и примечания

-   **Цены:** Информацию о стоимости использования моделей можно найти на странице [DeepSeek Pricing](https://api-docs.deepseek.com/quick_start/pricing/).
