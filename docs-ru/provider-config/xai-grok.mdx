xAI — это компания, создавшая Grok, большую языковую модель, известную своими разговорными способностями и большим контекстным окном (context window). Модели Grok разработаны для предоставления полезных, информативных и контекстуально релевантных ответов.

**Веб-сайт:** [https://x.ai/](https://x.ai/)

### Получение API Key

1.  **Регистрация/Вход:** Перейдите в [xAI Console](https://console.x.ai/). Создайте аккаунт или войдите в систему.
2.  **Перейдите к API Keys:** Откройте раздел API keys в вашей панели управления.
3.  **Создайте ключ:** Нажмите, чтобы создать новый API key. Дайте вашему ключу описательное имя (например, «Careti»).
4.  **Скопируйте ключ:** **Важно:** Скопируйте API key *немедленно*. Вы не сможете увидеть его снова. Храните его в безопасном месте.

### Поддерживаемые модели

Careti поддерживает следующие модели xAI Grok:

#### Модели Grok-3

-   `grok-3-beta` (По умолчанию) — бета-модель Grok-3 от xAI с контекстным окном 131K
-   `grok-3-fast-beta` — быстрая бета-модель Grok-3 от xAI с контекстным окном 131K
-   `grok-3-mini-beta` — мини бета-модель Grok-3 от xAI с контекстным окном 131K
-   `grok-3-mini-fast-beta` — быстрая мини бета-модель Grok-3 от xAI с контекстным окном 131K

#### Модели Grok-2

-   `grok-2-latest` — модель Grok-2 от xAI — последняя версия с контекстным окном 131K
-   `grok-2` — модель Grok-2 от xAI с контекстным окном 131K
-   `grok-2-1212` — модель Grok-2 от xAI (версия 1212) с контекстным окном 131K

#### Модели Grok Vision

-   `grok-2-vision-latest` — модель Grok-2 Vision от xAI — последняя версия с поддержкой изображений и контекстным окном 32K
-   `grok-2-vision` — модель Grok-2 Vision от xAI с поддержкой изображений и контекстным окном 32K
-   `grok-2-vision-1212` — модель Grok-2 Vision от xAI (версия 1212) с поддержкой изображений и контекстным окном 32K
-   `grok-vision-beta` — бета-модель Grok Vision от xAI с поддержкой изображений и контекстным окном 8K

#### Устаревшие модели

-   `grok-beta` — бета-модель Grok от xAI (устаревшая) с контекстным окном 131K

### Настройка в Careti

1.  **Откройте настройки Careti:** Нажмите на иконку настроек (⚙️) в панели Careti.
2.  **Выберите провайдера:** Выберите «xAI» в выпадающем списке «API Provider».
3.  **Введите API Key:** Вставьте ваш xAI API key в поле «xAI API Key».
4.  **Выберите модель:** Выберите нужную модель Grok в выпадающем списке «Model».

### Возможности рассуждения (Reasoning)

Модели Grok 3 Mini обладают специализированными возможностями рассуждения, что позволяет им «думать перед ответом» — это особенно полезно для решения сложных задач.

#### Модели с поддержкой рассуждения

Рассуждение поддерживается только в:

-   `grok-3-mini-beta`
-   `grok-3-mini-fast-beta`

Модели Grok 3 `grok-3-beta` и `grok-3-fast-beta` не поддерживают рассуждение.

#### Управление интенсивностью рассуждения (Reasoning Effort)

При использовании моделей с поддержкой рассуждения вы можете контролировать, насколько усердно модель думает, с помощью параметра `reasoning_effort`:

-   `low`: Минимальное время на раздумья, использование меньшего количества токенов для быстрых ответов.
-   `high`: Максимальное время на раздумья, использование большего количества токенов для решения сложных задач.

Выбирайте `low` для простых запросов, которые должны выполняться быстро, и `high` для более трудных задач, где задержка ответа менее важна.

#### Ключевые особенности

-   **Пошаговое решение задач**: Модель методично обдумывает проблему перед тем, как дать ответ.
-   **Сильные стороны в математике и вычислениях**: Отлично справляется с числовыми задачами и логическими головоломками.
-   **Доступ к цепочке рассуждений**: Процесс мышления модели доступен через поле `reasoning_content` в объекте ответа API.

### Советы и примечания

-   **Контекстное окно:** Большинство моделей Grok обладают большими контекстными окнами (до 131K токенов), что позволяет включать значительные объемы кода и контекста в ваши промпты.
-   **Возможности Vision:** Выбирайте модели с поддержкой vision (`grok-2-vision-latest`, `grok-2-vision` и т. д.), когда вам нужно обработать или проанализировать изображения.
-   **Цены:** Стоимость варьируется в зависимости от модели: затраты на входные токены составляют от $0.3 до $5.0 за миллион токенов, а на выходные — от $0.5 до $25.0 за миллион токенов. Актуальную информацию о ценах смотрите в документации xAI.
-   **Баланс производительности:** Варианты «Fast» обычно обеспечивают более быстрое время ответа, но могут иметь более высокую стоимость, в то время как варианты «mini» более экономичны, но могут иметь несколько меньшие возможности.
