Caret поддерживает доступ к моделям через платформу Fireworks AI, которая предлагает быстрый и экономичный доступ к широкому спектру современных языковых моделей с открытым исходным кодом. Разработанная для обеспечения скорости и надежности, Fireworks AI предоставляет варианты serverless развертывания с OpenAI-совместимыми API и контекстным окном до 256 000 токенов.

**Сайт:** [https://fireworks.ai/](https://fireworks.ai/)

### Получение API Key

1. **Регистрация/Вход:** Перейдите на [Fireworks AI](https://fireworks.ai/) и создайте аккаунт или войдите в систему.
2. **Перейдите к API Keys:** После входа в систему перейдите на [страницу API Keys](https://app.fireworks.ai/settings/users/api-keys) в настройках аккаунта.
3. **Создайте ключ:** Нажмите «Create API key» и дайте вашему ключу описательное название (например, «Caret»).
4. **Скопируйте ключ:** **Важно:** Скопируйте API key _немедленно_. Вы не сможете увидеть его снова. Храните его в безопасном месте.

### Поддерживаемые модели

Caret поддерживает следующие модели Fireworks AI:

-   `accounts/fireworks/models/kimi-k2-instruct` (По умолчанию)
-   `accounts/fireworks/models/qwen3-235b-a22b-instruct-2507`
-   `accounts/fireworks/models/qwen3-coder-480b-a35b-instruct`
-   `accounts/fireworks/models/deepseek-r1-0528`
-   `accounts/fireworks/models/deepseek-v3`

### Конфигурация в Caret

1. **Откройте настройки Caret:** Нажмите на иконку шестеренки (⚙️) на панели Caret.
2. **Выберите провайдера:** Выберите «Fireworks AI» в выпадающем списке «API Provider».
3. **Введите API Key:** Вставьте ваш Fireworks AI API key в поле «Fireworks AI API Key».
4. **Выберите модель:** Выберите нужную модель в выпадающем списке «Model». Модель по умолчанию — Kimi K2.

### Советы и примечания

-   **Экономичность:** Fireworks AI предлагает значительно более низкие цены, чем проприетарные модели, сохраняя при этом конкурентоспособную производительность.
-   **Большие контекстные окна:** Большинство моделей поддерживают от 128K до 256K токенов, что подходит для обработки больших документов и ведения длительных диалогов.
-   **Совместимость с OpenAI:** Провайдер использует OpenAI-совместимый формат API с поддержкой streaming и отслеживанием использования.
-   **Rate Limits:** У Fireworks AI есть ограничения частоты запросов, основанные на использовании. Отслеживайте потребление в панели управления и при необходимости рассмотрите возможность повышения тарифного плана.
-   **API Keys:** Хранятся локально на вашем устройстве в целях безопасности.
-   **Цены:** Актуальные тарифы можно найти на [странице цен Fireworks AI](https://fireworks.ai/pricing). Указанные цены приведены за миллион токенов.
