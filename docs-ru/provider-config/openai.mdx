Careti поддерживает прямой доступ к моделям через официальный OpenAI API.

**Веб-сайт:** [https://openai.com/](https://openai.com/)

### Получение API Key

1.  **Регистрация/Вход:** Посетите [OpenAI Platform](https://platform.openai.com/). Вам потребуется создать аккаунт или войти в систему, если он у вас уже есть.
2.  **Перейдите к API Keys:** После входа в систему перейдите в [раздел API keys](https://platform.openai.com/api-keys) вашего аккаунта.
3.  **Создайте ключ:** Нажмите «Create new secret key». Рекомендуется дать ключу понятное название (например, «Careti API Key»).
4.  **Скопируйте ключ:** **Важно:** Немедленно скопируйте сгенерированный API Key. Из соображений безопасности OpenAI не покажет его повторно. Храните этот ключ в надежном и безопасном месте.

### Поддерживаемые модели

Careti совместим с различными моделями OpenAI, включая, но не ограничиваясь:

-   'o3'
-   `o3-mini` (medium reasoning effort)
-   'o4-mini'
-   `o3-mini-high` (high reasoning effort)
-   `o3-mini-low` (low reasoning effort)
-   `o1`
-   `o1-preview`
-   `o1-mini`
-   `gpt-4o`
-   `gpt-4o-mini`
-   'gpt-4.1'
-   'gpt-4.1-mini'

Актуальный список доступных моделей и их возможностей можно найти в официальной [документации OpenAI Models](https://platform.openai.com/docs/models).

### Настройка в Careti

1.  **Откройте настройки Careti:** Нажмите на иконку шестеренки (⚙️) на панели Careti.
2.  **Выберите провайдера:** Выберите «OpenAI» в выпадающем меню «API Provider».
3.  **Введите API Key:** Вставьте ваш OpenAI API Key в поле «OpenAI API Key».
4.  **Выберите модель:** Выберите нужную модель из выпадающего списка «Model».
5.  **(Опционально) Base URL:** Если вам нужно использовать прокси или кастомный Base URL для OpenAI API, вы можете ввести его здесь. Большинству пользователей не потребуется менять значение по умолчанию.

### Советы и примечания

-   **Цены:** Обязательно ознакомьтесь со страницей [OpenAI Pricing](https://openai.com/pricing) для получения подробной информации о стоимости использования различных моделей.
-   **Azure OpenAI Service:** Если вы планируете использовать сервис Azure OpenAI, обратите внимание, что отдельная документация по Azure OpenAI для Careti может находиться отдельно, или вам может потребоваться настроить его как OpenAI-совместимый эндпоинт, если такая функциональность поддерживается в Careti для пользовательских конфигураций.
