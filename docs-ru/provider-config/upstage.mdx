Upstage — это ведущая корейская AI-компания, разрабатывающая серию больших языковых моделей Solar. Модели Solar известны своей отличной производительностью для корейского языка и эффективной inference speed.

**Веб-сайт:** [https://console.upstage.ai](https://console.upstage.ai)

## Получение API Key

1. **Создайте учетную запись**: Зарегистрируйтесь в [Upstage Console](https://console.upstage.ai).
2. **Сгенерируйте API Key**: Перейдите в раздел API Keys в вашей панели управления.
3. **Скопируйте ключ**: Создайте новый API key и сохраните его в безопасном месте.

## Поддерживаемые модели

### Solar Pro 2

- **Model ID**: `solar-pro2`
- **Context Window**: 32,768 tokens
- **Max Output**: 4,096 tokens
- **Description**: Продвинутая языковая модель для сложных задач

Solar Pro 2 — это последняя флагманская модель Upstage, которая отлично справляется с написанием кода, логическими рассуждениями и мультиязычными задачами.

### Solar Mini

- **Model ID**: `solar-mini`
- **Context Window**: 32,768 tokens
- **Max Output**: 4,096 tokens
- **Description**: Легкая и быстрая модель

Solar Mini оптимизирована для скорости и эффективности, подходит для задач, требующих быстрого отклика.

## Настройка в Caret

1. **Откройте Settings**: Нажмите на иконку настроек (⚙️) в панели Caret.
2. **Выберите Provider**: Выберите "Upstage" в выпадающем списке "API Provider".
3. **Введите API Key**: Вставьте ваш API key в поле "Upstage API Key".
4. **Выберите Model**: Выберите нужную модель из списка.

## Особенности Upstage

### Оптимизация для корейского языка

Модели Solar прошли интенсивное обучение на корейских данных, обеспечивая отличную производительность при работе с корейскими комментариями в коде, документацией и задачами обработки естественного языка.

### Эффективная архитектура

Серия Solar использует технологию Depth Up-Scaling (DUS) для реализации эффективной архитектуры моделей. Это позволяет достичь высокой производительности с меньшим количеством параметров.

### OpenAI-совместимый API

Upstage API использует OpenAI-совместимый формат, что упрощает интеграцию в существующие рабочие процессы на базе OpenAI.

## Советы по использованию

- **Coding Tasks**: Solar Pro 2 эффективна для генерации кода, рефакторинга и исправления ошибок.
- **Быстрые ответы**: Используйте Solar Mini для простых запросов или когда требуется быстрый отклик.
- **Корейские проекты**: Особенно полезно для проектов с комментариями или документацией на корейском языке.
- **Экономичность**: Solar Mini — это экономичный вариант для этапов прототипирования или разработки.
