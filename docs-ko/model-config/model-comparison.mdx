1. **Claude Sonnet 4.5** - 가장 안정적
2. **GPT-5** - 우수하지만 간헐적 포맷 문제
3. **Gemini 2.5 Pro** - 표준 도구에 강함
4. **DeepSeek V3** - 기본 도구에 안정적
5. **Qwen3 계열** - 복잡한 도구는 재시도 필요
