<Info>
**구성 경로: Self-Hosted**

이 문서는 Self-Hosted 배포용 Vertex AI 구성 가이드입니다. 웹 기반 설정은 [Google Vertex SaaS 구성](/ko/enterprise-solutions/configuration/remote-configuration/google-vertex/admin-configuration)을 참고하세요.
</Info>

Caret를 Google Vertex AI에 연결해 Gemini 및 Google 모델을 엔터프라이즈 방식으로 사용합니다.

## 구성 형식

원격 구성 JSON의 `providerSettings.Vertex`에 Vertex AI 설정을 추가합니다.

```json
{
  "providerSettings": {
    "Vertex": {
      "models": [
        {
          "id": "claude-3-5-sonnet-v2@20241022",
          "name": "Claude 3.5 Sonnet"
        }
      ],
      "vertexProjectId": "my-project-id",
      "vertexRegion": "us-central1"
    }
  }
}
```

## 구성 필드

| 필드 | 타입 | 설명 | 필수 |
|-------|------|-------------|----------|
| `models` | Array | 모델 구성 목록 | Yes |
| `vertexProjectId` | String | Google Cloud 프로젝트 ID | Yes |
| `vertexRegion` | String | GCP 리전(e.g., `us-central1`) | Yes |

### 모델 구성

`models` 배열의 각 항목은 다음을 포함합니다.

```json
{
  "id": "claude-3-5-sonnet-v2@20241022",
  "name": "Claude 3.5 Sonnet",
  "info": {
    "maxTokens": 8192,
    "contextWindow": 200000,
    "supportsImages": true,
    "supportsPromptCache": true
  }
}
```

## 주요 모델 ID

| Model ID | 설명 | 컨텍스트 창 |
|----------|-------------|----------------|
| `claude-3-5-sonnet-v2@20241022` | Claude 3.5 Sonnet | 200K tokens |
| `claude-3-5-haiku@20241022` | Claude 3.5 Haiku | 200K tokens |
| `claude-3-opus@20240229` | Claude 3 Opus | 200K tokens |
| `gemini-2.0-flash-exp` | Gemini Flash(실험적) | 1M tokens |
| `gemini-1.5-pro-002` | Gemini Pro | 2M tokens |
| `gemini-1.5-flash-002` | Gemini Flash | 1M tokens |

<Note>
모델 가용성은 리전별로 다릅니다. [Vertex AI 문서](https://cloud.google.com/vertex-ai/docs/generative-ai/learn/models)를 참고하세요.
</Note>

## 사전 요구 사항

Vertex AI를 구성하기 전에 다음이 필요합니다.

1. **Google Cloud 프로젝트**(Vertex AI API 활성화)
2. **서비스 계정**(Vertex AI User 역할: `roles/aiplatform.user`)
3. **서비스 계정 자격 증명** 구성
4. **모델 접근 확인**(프로젝트/리전)

<Tip>
설정 방법은 [Vertex AI 빠른 시작](https://cloud.google.com/vertex-ai/docs/generative-ai/start/quickstarts/quickstart-multimodal)을 참고하세요.
</Tip>

## 문제 해결

**"Permission Denied" 오류**

서비스 계정에 필요한 Vertex AI 권한이 있는지 확인하세요. [Google Cloud IAM 문서](https://cloud.google.com/vertex-ai/docs/general/access-control)를 참고하세요.

**"API Not Enabled" 오류**

프로젝트에서 Vertex AI API가 활성화되어 있는지 확인하세요.

**"Model Not Found" 오류**

모델이 리전에서 지원되는지, 프로젝트 접근이 허용되어 있는지 확인하세요.
