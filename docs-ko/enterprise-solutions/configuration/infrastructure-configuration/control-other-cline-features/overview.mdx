이 섹션은 엔터프라이즈 배포에서 Caret의 동작을 제어하는 구성 옵션을 다룹니다.

## 사용 가능한 설정

<Card title="YOLO 모드" icon="rocket" href="/ko/enterprise-solutions/configuration/infrastructure-configuration/control-other-cline-features/yolo-mode">
  완전 자동 승인 기반의 자율 동작 모드 사용 여부를 제어합니다.
</Card>

## 구성 방식

이 설정은 다음 방식으로 구성할 수 있습니다.

### 개별 사용자
- 사용자는 로컬 Caret UI에서 설정을 토글할 수 있습니다.
- 엔터프라이즈 정책으로 특정 설정을 제한할 수 있습니다.
- 변경 사항은 새 작업에 즉시 적용됩니다.

## 엔터프라이즈 제어

관리자는 원격 구성을 통해 정책을 강제할 수 있습니다.

```json
{
  "yoloModeAllowed": false
}
```

`yoloModeAllowed`가 `false`이면 사용자는 로컬 Caret UI에서 YOLO 모드를 켤 수 없습니다.
