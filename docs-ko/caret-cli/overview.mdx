<Warning>
**프리뷰 릴리스 - macOS 및 Linux 전용**

캐러티 CLI는 현재 프리뷰 상태이며 macOS와 Linux 사용자만 사용할 수 있습니다. Windows 지원은 곧 제공됩니다.
</Warning>

## 캐러티 CLI란?

캐러티 CLI는 터미널에서 AI 코딩 에이전트를 직접 실행합니다. CI/CD에서 자동 코드 리뷰를 위해 git diff를 파이프로 넘기거나, 여러 인스턴스를 동시에 실행해 병렬 개발을 하거나, 기존 쉘 워크플로우에 캐러티를 통합할 수 있습니다.

CLI는 시스템 전반의 인스턴스를 추적하고, 사람이 읽기 좋은 출력/스크립트 처리용 JSON/일반 텍스트/리치 터미널 출력을 제공합니다.

<Tip>
바로 시작하시겠어요? [설치 가이드](/ko/caret-cli/installation)에서 캐러티 CLI를 설치하고 첫 작업을 실행해 보세요.
</Tip>

## 지원 모델 프로바이더

캐러티 CLI는 여러 AI 모델 프로바이더를 지원합니다. 예:

- **Anthropic**
- **OpenAI**
- **OpenAI Compatible**
- **OpenRouter**
- **X AI (Grok)**
- **AWS Bedrock**
- **Google Gemini**
- **Ollama**
- **Cerebras**

설치 과정에서 `caret auth` 명령으로 인증과 설정을 진행합니다.

## 무엇을 만들 수 있나요?

**자동 코드 유지보수**
- 매일 실행해 린트 이슈를 찾아 자동 수정
- 보안 취약점을 스캔하고 자동 패치
- deprecated 의존성 업데이트 및 테스트 실행

**멀티 인스턴스 개발**
- 프론트엔드/백엔드를 동시에 다른 인스턴스로 실행
- 기능 브랜치별로 인스턴스를 분리해 병렬 작업
- 여러 PR에 대한 병렬 리뷰 프로세스 구성

**커스텀 워크플로우**
- git hooks와 결합한 셸 스크립트 구성
- 복잡한 데이터 구조를 캐러티에 파이프하여 처리
- jq/grep/awk 등 기존 도구와 결합

**CI/CD 통합**
- GitHub Actions에서 PR 자동 코드 리뷰
- GitLab 파이프라인에서 스키마 변경 분석
- Jenkins에서 테스트 실패 분석 및 수정 제안
