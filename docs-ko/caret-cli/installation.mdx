## 사전 준비

Caret CLI는 Node.js 20 이상이 필요합니다. 최상의 환경을 위해 Node.js 22를 권장합니다.

Node.js 버전을 확인하려면:

```bash
node --version
```

## 설치

```bash
npm install -g caret
```

설치 후, Caret 계정을 인증합니다:

```bash
caret auth
```

이 과정에서 인증 마법사가 실행되며, 로그인과 선호하는 AI 모델 프로바이더 설정을 안내합니다.

## 빠른 시작

몇 초 만에 Caret을 시작할 수 있습니다:

```bash
caret
```

이제 끝입니다. 어떤 디렉터리에서든 `caret`을 실행하면 대화형 세션이 시작됩니다. 작업을 입력하고 계획을 검토한 뒤 `/act`로 실행을 승인하세요.

대화 없이 더 빠르게 실행하려면:

```bash
caret "Add unit tests to utils.js"
```

단일 명령으로 Caret을 실행하므로 빠른 작업이나 자동화에 적합합니다.

<Tip>
Caret CLI가 처음이라면 대화형 모드(`caret`)부터 시작하세요. 익숙해지면 [세 가지 핵심 흐름](/caret-cli/three-core-flows)에서 고급 사용 패턴을 확인할 수 있습니다.
</Tip>
