# YOLO 모드

YOLO 모드는 이름 그대로 - Caret이 모든 것을 자동으로 승인합니다. 기능 설정에서 체크박스를 선택하면 파일 변경, 터미널 명령, 심지어 Agent에서 Act 모드로의 전환까지 묻지 않고 실행합니다.

[자동 승인](/ko/features/auto-approve) 기능의 강화 버전이라고 생각하세요 - 세부적인 권한 대신, YOLO 모드는 Caret에게 완전한 자율성을 부여합니다.

<Note>
**경고: 이것은 위험합니다.** YOLO 모드는 모든 안전 장치를 비활성화합니다. Caret은 결정한 모든 것을 허가 없이 실행합니다.
</Note>

## 자동 승인되는 항목

YOLO 모드가 활성화되면 Caret은 다음을 자동으로 승인합니다:

- **모든 파일 작업** - 시스템 어디든지 파일 읽기, 쓰기, 수정
- **모든 터미널 명령** - 잠재적으로 파괴적인 작업 포함
- **브라우저 액션** - 웹 스크래핑, 폼 제출, 네비게이션
- **MCP 서버 도구** - 외부 통합 및 API 호출
- **모드 전환** - Agent에서 Act 모드로 자동 전환

본질적으로 모든 안전 가드레일이 제거됩니다. Caret은 완전한 자율성으로 작동합니다.

## YOLO 모드 활성화 방법

Caret 설정 → 기능으로 이동하여 "YOLO 모드" 체크박스를 선택하세요. 그게 전부입니다 - 확인 대화상자도, 추가 경고도 없습니다. 활성화되면 Caret은 즉시 모든 작업을 자동 승인하기 시작합니다.

비활성화하려면 체크박스를 해제하기만 하면 됩니다. YOLO 모드가 꺼지면 대기 중인 작업은 여전히 승인이 필요합니다.

## 언제 사용할까요?

YOLO 모드는 주로 완전 자율 실행이 필요한 스크립트 가능한 CLI를 위해 만들어졌습니다. GUI에서는 다음과 같은 경우에 사용할 수 있습니다:

**빠른 프로토타이핑**: 마찰이 전혀 없고 실수를 신경 쓰지 않을 때. 일회용 실험이나 새로운 아이디어를 빠르게 탐색할 때 완벽합니다.

**검증된 반복 작업**: 이미 Caret의 접근 방식을 검증했고 승인 오버헤드를 제거하고 싶을 때. 일상적인 리팩토링이나 확립된 패턴을 생각해보세요.

**데모 목적**: 지속적인 중단 없이 Caret의 기능을 보여주고 싶을 때.

## 무엇이 잘못될 수 있을까요?

YOLO 모드는 모든 안전 장치를 제거하므로, Caret은 다음을 할 수 있습니다:

- 경고 없이 중요한 파일 삭제
- 시스템 설정을 수정하는 명령 실행
- 외부 서비스로 네트워크 요청
- 설정 파일 덮어쓰기
- 소프트웨어 패키지 설치 또는 제거
- 버전 관리에 변경사항 커밋 및 푸시

위험 수준은 전적으로 Caret에게 요청하는 작업에 달려 있습니다. 간단한 작업은 비교적 안전하지만, 복잡한 요청은 예측할 수 없는 결과를 가져올 수 있습니다.

## 모범 사례

YOLO 모드를 사용하기로 결정했다면:

**격리된 환경에서 시작하세요.** 먼저 일회용 프로젝트나 샌드박스 환경에서 사용하세요. 위험을 이해하기 전까지는 프로덕션 코드베이스에서 절대 활성화하지 마세요.

**구체적인 요청을 하세요.** 모호한 지시와 무제한 권한이 결합되면 예상치 못한 결과로 이어질 수 있습니다. 요구사항이 명확할수록 Caret의 행동을 더 예측할 수 있습니다.

**출력을 모니터링하세요.** Caret이 허가를 요청하지 않더라도 여전히 무엇을 하고 있는지 보여줍니다. 터미널 출력과 파일 변경사항을 실시간으로 관찰하세요.

**버전 관리를 준비하세요.** 무언가 잘못되었을 때 쉽게 변경사항을 되돌릴 수 있는지 확인하세요. YOLO 모드가 워크플로우일 때 Git이 안전망이 됩니다.

## 영감: 무엇이 가능해질까요?

YOLO 모드를 활성화하면:

**단일 프롬프트로 전체 애플리케이션 구축**: 원하는 것을 설명하면 Caret이 모든 것을 처리합니다 - 파일 생성, 의존성 설치, 설정 구성, 심지어 배포 스크립트까지.

**복잡한 워크플로우 자동화**: 일반적으로 수십 번의 승인 클릭이 필요한 작업을 자동화합니다. 데이터 처리 파이프라인, 빌드 시스템 설정 또는 다단계 리팩토링 작업이 원활해집니다.

**빠른 반복 주기**: 승인 마찰 없이 아이디어를 빠르게 테스트할 수 있습니다. 다양한 접근 방식을 탐색하거나 새로운 기술을 실험하는 데 완벽합니다.

**라이브 데모**: 모든 작업을 승인하기 위해 멈추지 않고 Caret의 전체 기능을 보여줄 수 있습니다. 프레젠테이션이나 교육 시나리오에 훌륭합니다.

핵심은 YOLO 모드가 Caret을 대화형 어시스턴트에서 자율 에이전트로 변환한다는 것을 이해하는 것입니다. 그 힘을 현명하게 사용하세요.
