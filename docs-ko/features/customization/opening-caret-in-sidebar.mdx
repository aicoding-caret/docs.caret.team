---
title: "오른쪽 사이드바에서 Caret 열기"
description: "VS Code와 Cursor에서 Caret을 오른쪽 사이드바로 옮기는 방법"
---

기본적으로 Caret은 VS Code의 왼쪽 사이드바(파일 탐색기 등과 함께)에서 열립니다. 하지만 오른쪽 사이드바로 옮기면 왼쪽에 프로젝트 파일을 그대로 두고, 오른쪽에서 Caret과 대화할 수 있어 훨씬 효율적입니다.

## VS Code

오른쪽 사이드바에서 Caret을 열려면:

<Steps>
  <Step title="확장 뷰 정렬 확인">
    확장 뷰가 왼쪽에 세로로 정렬되어 있는지 확인합니다.
  </Step>
  <Step title="오른쪽 패널 열기">
    VS Code의 오른쪽 패널을 여는 버튼(보통 Copilot Chat을 여는 버튼)을 클릭합니다. 필요하면 `Option + CMD/Ctrl + B` 단축키를 사용하세요.
  </Step>
  <Step title="Caret 아이콘 드래그">
    Caret 아이콘을 오른쪽 패널 상단의 네비게이션 영역으로 드래그합니다.
  </Step>
</Steps>

## Cursor

Cursor에서 오른쪽 사이드바에 Caret을 배치하려면:

<Steps>
  <Step title="확장 뷰 방향 변경">
    Cursor는 기본적으로 AI 채팅 인터페이스 공간을 확보하기 위해 활동 바를 가로로 배치합니다. 세로로 변경하려면:

    1. Command Palette 열기 (`CMD/Ctrl + Shift + P`)
    2. "Preferences: Open Settings (UI)" 검색
    3. `workbench.activityBar.orientation` 검색
    4. 값을 `vertical`로 변경
    5. Cursor 재시작
  </Step>
  <Step title="AI 패널 열기">
    Cursor의 큐브 아이콘(AI Pane)을 눌러 오른쪽 패널을 엽니다.
  </Step>
  <Step title="Caret 드래그">
    Caret 아이콘을 AI 패널 사이드바로 드래그합니다.
  </Step>
</Steps>
