## 제약 없는 오픈소스 AI 코딩

Caret는 한계 없는 프런티어 AI 접근을 제공합니다. 모든 결정은 투명하게 드러나며, 모델 생태계에 종속되지 않습니다. 어떤 모델이든 선택하고 비용을 통제할 수 있습니다.

<CardGroup cols={1}>
  <Card title="완전한 투명성" icon="eye">
    Caret가 파일을 읽고 접근법을 고민하며 변경을 제안하는 과정을 실시간으로 확인할 수 있습니다. 모든 결정은 가시적이고, 모든 수정은 사전에 검토 가능합니다.
  </Card>

  <Card title="추론 선택권" icon="sliders">
    복잡한 추론은 Claude, 대용량 컨텍스트는 Gemini, 효율성은 Qwen3 Coder처럼 상황에 맞게 선택할 수 있습니다. 새로운 모델이 등장하면 즉시 전환하고, API 키는 직접 관리합니다.
  </Card>

  <Card title="제로 트러스트 설계" icon="lock">
    코드는 서버로 전송되지 않습니다. Caret는 클라이언트에서 API 키로 동작하며, 오픈소스라 보안 팀이 전체 코드를 검토할 수 있습니다.
  </Card>
</CardGroup>

## Plan & Act 모드

Caret는 코드를 쓰기 전에 전체 맥락을 이해하도록 탐색하고, 계획을 만들어 사용자와 합의한 뒤 실행합니다.

<Tabs>
  <TabItem title="Plan 모드" value="plan-mode" icon="lightbulb">
    복잡한 작업에서 Caret가 코드베이스를 탐색하고 질문을 던지며 상세 실행 계획을 수립합니다.

    - 정보 수집과 컨텍스트 구성
    - 확인 질문 제기
    - 상세 실행 계획 생성
    - 사용자와 접근 방식 논의
  </TabItem>

  <TabItem title="Act 모드" value="act-mode" icon="play">
    계획이 승인되면 Caret가 도구를 사용해 실행에 들어갑니다.

    - 승인된 계획 실행
    - 파일 수정 및 명령 실행
    - 솔루션 구현
    - 완료 결과 보고
  </TabItem>
</Tabs>
