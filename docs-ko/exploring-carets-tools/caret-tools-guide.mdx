## Caret은 무엇을 할 수 있나요?

Caret은 다음과 같은 작업을 수행할 수 있는 AI 비서입니다:

- 프로젝트 내 파일 생성/편집
- 터미널 명령 실행
- 코드 검색 및 분석
- 디버깅 및 문제 해결 지원
- 반복 작업 자동화
- 외부 도구 통합

## 첫 단계

1. **작업 시작**

    - 채팅에 요청을 입력합니다
    - 예시: "Header라는 새 React 컴포넌트를 만들어줘"

2. **컨텍스트 제공**

    - @ 멘션으로 파일, 폴더, URL을 추가합니다
    - 예시: "@file:src/components/App.tsx"

3. **변경 사항 검토**
    - Caret은 변경 내용을 적용하기 전에 diff를 보여줍니다
    - 사용자가 수정하거나 거부할 수 있습니다

## 주요 기능

1. **파일 편집**

    - 새 파일 생성
    - 기존 코드 수정
    - 여러 파일에 대한 검색/치환

2. **터미널 명령**

    - npm 명령 실행
    - 개발 서버 실행
    - 의존성 설치

3. **코드 분석**

    - 오류 탐지 및 수정
    - 코드 리팩터링
    - 문서 추가

4. **브라우저 통합**
    - 웹 페이지 테스트
    - 스크린샷 캡처
    - 콘솔 로그 확인

## 사용 가능한 도구

최신 구현 상세는 [Careti 저장소](https://github.com/caret/caret/blob/main/src/core/prompts/system-prompt/tools)에서 전체 소스 코드를 확인하세요.

Caret은 다양한 작업을 위해 아래 도구에 접근할 수 있습니다:

1. **파일 작업**

    - `write_to_file`: 파일 생성 또는 덮어쓰기
    - `read_file`: 파일 내용 읽기
    - `replace_in_file`: 파일의 특정 부분 수정
    - `search_files`: 정규식으로 파일 검색
    - `list_files`: 디렉터리 내용 나열

2. **터미널 작업**

    - `execute_command`: CLI 명령 실행
    - `list_code_definition_names`: 코드 정의 목록 조회

3. **MCP 도구**

    - `use_mcp_tool`: MCP 서버의 도구 사용
    - `access_mcp_resource`: MCP 서버 리소스 접근
    - 사용자가 직접 MCP 도구를 만들어 Caret이 접근하도록 구성할 수 있음
    - 예시: Caret이 일기예보를 조회하는 날씨 API 도구를 사용하도록 설정

4. **상호작용 도구**
    - `ask_followup_question`: 사용자에게 추가 질문
    - `attempt_completion`: 최종 결과 제시

각 도구에는 특정 파라미터와 사용 패턴이 있습니다. 예시는 다음과 같습니다.

- 새 파일 생성 (`write_to_file`):

    ```xml
    <write_to_file>
    <path>src/components/Header.tsx</path>
    <content>
    // Header component code
    </content>
    </write_to_file>
    ```

- 패턴 검색 (`search_files`):

    ```xml
    <search_files>
    <path>src</path>
    <regex>function\s+\w+\(</regex>
    <file_pattern>*.ts</file_pattern>
    </search_files>
    ```

- 명령 실행 (`execute_command`):
    ```xml
    <execute_command>
    <command>npm install axios</command>
    <requires_approval>false</requires_approval>
    </execute_command>
    ```

## 자주 사용하는 작업

1. **새 컴포넌트 만들기**

    - "Footer라는 새 React 컴포넌트를 만들어줘"

2. **버그 수정**

    - "src/utils/format.ts의 오류를 수정해줘"

3. **코드 리팩터링**

    - "Button 컴포넌트를 TypeScript로 리팩터링해줘"

4. **명령 실행**
    - "npm install로 axios를 추가해줘"
