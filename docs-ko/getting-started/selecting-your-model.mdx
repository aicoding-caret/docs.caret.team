Caret은 요청을 이해하고 코드를 작성하기 위해 AI 모델이 필요합니다. 어떤 전문가와 함께할지 고르는 것처럼, 모델마다 강점과 비용이 다릅니다.

## 빠른 시작: 프로바이더 선택

가장 쉬운 시작은 **Careti** 프로바이더를 선택하는 것입니다:

1. **Careti 설정 열기**: Careti 채팅 우측 상단의 톱니바퀴(⚙️) 클릭
2. API Provider 드롭다운에서 **"Careti"** 선택
3. 모델 선택 - **Claude Sonnet 4.5** 또는 **DeepSeek V3**로 시작하는 것을 권장

<Frame>
  <img src="https://storage.googleapis.com/cline_public_images/step2-provider.png" alt="Select Careti Provider" />
</Frame>

**끝입니다!** API 키를 관리할 필요 없이 여러 모델을 바로 사용할 수 있습니다.

<Tip>
**무료 모델 제공**: Caret은 파트너 프로바이더를 통해 무료 추론을 제공하기도 합니다. 제공 시 모델 드롭다운에서 확인할 수 있습니다.
</Tip>

## 다른 프로바이더 사용하기

직접 API 키를 쓰고 싶다면 다음과 같은 프로바이더를 선택할 수 있습니다:

- **OpenRouter** - 가성비, 다양한 모델
- **Anthropic** - Claude 모델 직접 사용
- **OpenAI** - GPT 모델 사용
- **Google Gemini** - Google AI 모델
- **Ollama** - 로컬 모델 실행

프로바이더를 선택한 뒤에는:
1. 프로바이더 사이트에서 API 키 발급
2. Careti 설정의 API Key 필드에 붙여넣기
3. 모델 선택

<Note>
대부분의 프로바이더는 API 키 발급 전 결제 정보 등록이 필요합니다.
</Note>

## 어떤 모델을 선택해야 할까요?

처음 시작한다면 아래를 추천합니다:

| 우선순위 | 추천 모델 | 이유 |
|---------------|-------------------|-----|
| **안정성** | Claude Sonnet 4.5 | 코딩 작업에서 가장 안정적 |
| **가성비** | DeepSeek V3 | 낮은 비용 대비 우수한 성능 |
| **속도** | Qwen3 Coder | 빠른 응답 |
| **프라이버시** | Ollama 모델 | 내 컴퓨터에서 실행 |

대화 기록을 잃지 않고 언제든 모델을 변경할 수 있습니다.
