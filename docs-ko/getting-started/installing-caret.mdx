## 시작 전 확인

<CardGroup cols={1}>
  <Card title="계정 만들기" icon="user-plus" href="https://app.caret.bot/signup">
    **무료 Caret 계정**을 만들면:
    - 다양한 AI 모델(스텔스 모델 포함) 사용 가능
    - API 키를 직접 관리하지 않아도 되는 간편한 시작
    - 파트너 제공자 기반의 무료 추론 기회
  </Card>

  <Card title="호환 에디터 준비" icon="code">
    Caret은 다음 에디터에서 동작합니다:
    - **VS Code** / **Cursor**
    - **JetBrains IDEs** (IntelliJ, PyCharm, WebStorm 등)
    - **VSCodium** / **Windsurf**

    설치 전에 사용하는 에디터를 준비하세요.
  </Card>

## 설치 안내
<Tabs>
  <TabItem title="VS Code/Cursor" value="vs-code-cursor" icon="code">
    <Frame>
      <img
        src="https://storage.googleapis.com/cline_public_images/vscode.png"
        alt="VS Code logo"
        style={{ width: "100px", height: "auto", display: "block" }}
      />
    </Frame>
    <Steps>
      <Step title="확장 뷰 열기">
        VS Code에서 확장 뷰를 엽니다:
        - `Ctrl/Cmd + Shift + X` 또는
        - 활동 표시줄의 확장 아이콘 클릭
      </Step>

      <Step title="Caret 검색">
        확장 마켓플레이스 검색창에 **"Caret"**을 입력합니다.
        <Frame>
          <img src="https://storage.googleapis.com/cline_public_images/extension-installation.png" alt="VS Code marketplace showing Caret extension" />
        </Frame>
      </Step>

      <Step title="설치">
        Caret 확장의 **Install** 버튼을 클릭합니다.

        <Frame caption="VS Code marketplace with Caret extension ready to install">
          <img
            src="https://storage.googleapis.com/cline_public_images/docs/assets/image%20(20).png"
            alt="VS Code marketplace showing Caret extension"
          />
        </Frame>
      </Step>

      <Step title="Caret 열기">
        설치가 완료되면:
        - 활동 표시줄의 **Caret 아이콘** 클릭 또는
        - 명령 팔레트(`Ctrl/Cmd + Shift + P`)에서 **"Caret: Open In New Tab"** 실행

        <Frame caption="Caret opened in VSCode">
          <img
            src="https://storage.googleapis.com/cline_public_images/after-installation.png"
            alt="Caret opened in VSCode"
          />
        </Frame>

        <Warning>
        VS Code에 "Running extensions might..." 메시지가 표시되면 **Allow**를 클릭하세요. 아이콘이 보이지 않으면 VS Code를 재시작합니다.
        </Warning>
      </Step>
    </Steps>

    <Check>
    **설치 완료!** 이제 에디터에서 Caret UI를 볼 수 있습니다. 다음은 로그인 단계입니다.
    </Check>

    <Accordion title="문제 해결">

    **플러그인 설치가 안 보일 때**

    Caret 확장을 찾을 수 없다면:
    - **Marketplace** 탭에서 검색 중인지 확인
    - "Caret AI" 또는 "Caret"으로 검색
    - VS Code 버전이 호환되는지 확인

    설치가 실패한다면:
    - VS Code를 재시작하고 다시 시도
    - 인터넷 연결 확인
    - VSIX 파일 설치를 고려

    **플러그인이 보이지 않을 때**

    설치 후 Caret 창이 보이지 않으면:
    - VS Code를 완전히 종료 후 재실행
    - **View** → **Command Palette** → "Caret: Open In New Tab"
    - **Extensions** 뷰에서 확장이 활성화되어 있는지 확인
    - 활동 표시줄 왼쪽에 Caret 아이콘이 있는지 확인

    **일반 문제**

    플러그인이 설치됐는데 동작하지 않으면:
    - 설치 후 VS Code 재시작 여부 확인
    - 개발자 콘솔 오류 메시지 확인
    - 확장을 비활성화 후 재활성화

    성능/호환성 문제가 있으면:
    - 지원되는 VS Code 버전을 사용 중인지 확인
    - VS Code 업데이트 확인
    - 리소스를 많이 쓰는 확장 종료 고려

    도움이 필요하면 [Discord 커뮤니티](https://https://discord.gg/WB6yaR89YN)에 문의하세요.

    </Accordion>
  </TabItem>

    ### 설치 방법 선택

    <Tabs>
      <TabItem title="IDE에서 설치 (권장)" value="from-ide">
        <Steps>
          <Step title="설정 열기">
            JetBrains IDE에서 **Settings**로 이동합니다:
            - Windows/Linux: `Ctrl+Alt+S`
            - macOS: `Cmd+,`
          </Step>

          <Step title="플러그인 메뉴로 이동">
            **Plugins** → **Marketplace** 탭으로 이동합니다.
          </Step>

          <Step title="Caret 설치">
            **"Caret"**을 검색하고 **Install**을 클릭합니다.

            <Frame>
              <img
                src="https://storage.googleapis.com/cline_public_images/jetbrains-installation.png"
                alt="JetBrains marketplace showing Caret plugin search results"
              />
            </Frame>
          </Step>

          <Step title="IDE 재시작">
            설치를 완료하려면 IDE를 재시작합니다.
            <Frame>
              <img
                src="https://storage.googleapis.com/cline_public_images/jetbrains-post-installation.png"
                alt="JetBrains marketplace showing Caret plugin search results"
              />
            </Frame>
          </Step>
        </Steps>
      </TabItem>

      <TabItem title="브라우저에서 설치" value="browser-install">
        <Steps>
          <Step title="마켓플레이스 방문">
            [JetBrains Marketplace](https://plugins.jetbrains.com/plugin/28247-caret)로 이동합니다.
          </Step>

          <Step title="Install to IDE 클릭">
            **Install to IDE** 버튼을 클릭합니다.
          </Step>

          <Step title="IDE에서 확인">
            IDE가 열리면 설치를 확인합니다.
          </Step>

          <Step title="재시작">
            재시작 후 설치를 완료합니다.
          </Step>
        </Steps>
      </TabItem>

      <TabItem title="수동 설치" value="manual-install">
        <Steps>
          <Step title="플러그인 다운로드">
            [마켓플레이스 페이지](https://plugins.jetbrains.com/plugin/28247-caret)에서 다운로드합니다.
          </Step>

          <Step title="설정 열기">
            **Settings** → **Plugins**로 이동합니다.
          </Step>

          <Step title="디스크에서 설치">
            톱니바퀴 아이콘 → **Install Plugin from Disk**를 선택합니다.
          </Step>

          <Step title="파일 선택 및 재시작">
            다운로드한 `.zip` 파일을 선택하고 IDE를 재시작합니다.
          </Step>
        </Steps>
      </TabItem>
    </Tabs>

    <Check>
    **설치 완료!** **View** → **Tool Windows** → **Caret**에서 Caret 창을 찾을 수 있습니다.
    </Check>

    ### JetBrains에서 제공되는 기능

    <AccordionGroup>
      <Accordion title="✅ 핵심 기능" defaultOpen>
        - Diff 편집과 파일 수정
        - 다양한 API 제공자(Anthropic, OpenAI, 로컬 모델 등)
        - MCP 서버 및 커스텀 도구
        - Caret 규칙과 워크플로
        - 파일/폴더/문제에 대한 @멘션
        - 드래그 & 드롭 지원
      </Accordion>

      <Accordion title="📋 터미널 출력 차이">
        **JetBrains는 터미널 출력 UI가 VS Code와 다릅니다:**
        - VS Code: 출력이 채팅에 바로 스트리밍
        - JetBrains: 출력이 접을 수 있는 "Command Output" 섹션에 표시

        명령은 동일하게 실행되며, JetBrains에서는 섹션을 펼쳐 확인하면 됩니다.
      </Accordion>
    </AccordionGroup>

    <Accordion title="문제 해결">

    **플러그인 설치가 안 보일 때**

    Caret을 찾을 수 없다면:
    - **Marketplace** 탭에서 검색 중인지 확인
    - "Caret AI" 또는 "Caret"으로 검색
    - IDE 버전이 호환되는지 확인 (권장: 2023.1 이상)

    설치가 실패한다면:
    - IDE를 재시작하고 다시 시도
    - 인터넷 연결 확인
    - 디스크 설치 방식으로 전환

    **플러그인이 보이지 않을 때**

    Caret 창이 보이지 않으면:
    - IDE 완전히 종료 후 재실행
    - **View** → **Tool Windows** → **Caret** 확인
    - **Settings** → **Plugins** → **Installed**에서 활성화 여부 확인
    - 도구 창 영역에서 Caret 아이콘 확인

    **일반 문제**

    설치됐지만 동작하지 않으면:
    - 설치 후 IDE 재시작 여부 확인
    - IDE 이벤트 로그의 오류 확인
    - 플러그인 비활성화 후 재활성화

    성능/호환성 문제가 있으면:
    - 지원되는 JetBrains 버전인지 확인
    - IDE 업데이트 확인
    - 필요 시 메모리 할당량 증가

    도움이 필요하면 [Discord 커뮤니티](https://https://discord.gg/WB6yaR89YN)에 문의하세요.

## 다음 단계: 로그인 및 시작

<Steps>
  <Step title="Caret 열기">
    에디터에서 Caret을 엽니다:
    - **VS Code/Cursor/VSCodium/Windsurf:** 활동 표시줄의 Caret 아이콘 클릭
    - **JetBrains:** **View** → **Tool Windows** → **Caret**
  </Step>

  <Step title="로그인">
    Caret 인터페이스에서 **Sign Up** 버튼을 클릭합니다.

    <Info>
    [app.caret.bot](https://app.caret.bot)으로 이동하여 인증하며, 완료 후 에디터로 자동 복귀합니다.
    </Info>
    <Info>
    Caret 인증 페이지로 이동해 계정을 연결합니다.
    </Info>
  </Step>

  <Step title="준비 완료!">
    <Check>
    **축하합니다!** 이제 Caret을 바로 사용할 수 있습니다.

    Caret은 프로젝트 구축을 도와주는 동반자가 됩니다.
    </Check>
  </Step>
</Steps>
