## 📁 이중 디렉토리 & /init

**토큰 최적화 AI 컨텍스트 + 사용자 언어 문서를 1:1 미러링으로 관리**

```
.agents/                       # AI용 (영어, 토큰 최적화)
├── context/agents-rules.json  # 규칙 SoT (단일 진실 원천)
├── workflows/                 # 작업 워크플로우
├── skills/                    # AI 스킬
└── hooks/                     # 자동화 훅

.users/                        # 사용자용 (사용자 언어)
├── context/agents-rules.md    # 상세 설명
├── workflows/                 # (.agents/ 미러링)
├── skills/                    # (.agents/ 미러링)
└── hooks/                     # (.agents/ 미러링)
```

**핵심 기능**:
- **`/init`**: 프로젝트 분석 후 컨텍스트 파일 자동 생성
- **1:1 미러링**: `.agents/` ↔ `.users/` 변경사항 동기화 유지
- **AGENTS.md / CLAUDE.md**: 컨텍스트 연결 표준 진입점
- **레거시 폴백**: 기존 규칙 경로도 자동 감지하여 마이그레이션 안내

👉 **자세한 내용**: [AI 조직 지식 및 규칙 표준화](/ko/caret-exclusive/knowledge-parity-system)
