Caret은 Cline을 기반으로 한 향상된 AI 코딩 에이전트로, 최첨단 AI 모델을 VS Code 에디터에 직접 가져옵니다. 자동 완성 도구와 달리, Caret은 전체 코드베이스를 이해하고, 복잡한 변경사항을 계획하며, 다단계 작업을 실행할 수 있는 진정한 코딩 에이전트입니다.

**Caret = Cline + 개인화된 AI 컴패니언 + 향상된 시스템 프롬프트**

## 🌟 Caret을 선택해야 하는 이유

**🔄 Cline/Caret 토글** - 두 모드 간 원클릭 전환  
**🎭 개인화된 AI 컴패니언** - 즐거운 코딩을 위한 나만의 페르소나  
**🧠 향상된 시스템 프롬프트** - 더 스마트하고 효율적인 AI 응답  
**🌍 4개 언어 지원** - 영어, 한국어, 일본어, 중국어 + 4개 언어 매뉴얼  
**🎯 25개 제공업체, 223개 모델** - Cline v3.26.6 완전 호환 + 확장  

## 🎭 #1 특징: 개인화된 AI 코딩 파트너 생성

다른 AI 도구와 완전히 차별화되는 **페르소나 시스템**! 단순한 텍스트 채팅을 넘어서, 시각적으로 생생한 AI 컴패니언과 함께 코딩하세요.

### 기본 제공 페르소나들:

- **Caret** - 친근한 로봇 친구
- **오사랑** - K-POP 아이돌 개발자  
- **마도베 이치카** - Windows 11 OS-tan
- **시안 맥킨** - macOS 어시스턴트
- **탄도 우분투** - 오픈소스 협력자

**🎨 커스터마이징**: 나만의 AI 이름과 프로필 이미지를 등록하세요!

## Cline 기반의 검증된 아키텍처

Caret은 검증된 Cline 아키텍처를 기반으로, 한국 개발자와 글로벌 사용자를 위한 추가 기능을 제공합니다. 제한 없이, 놀라움 없이, 모델 생태계 종속성 없이 최첨단 AI에 대한 직접적이고 투명한 접근을 제공합니다.

### 완전한 투명성

Caret이 파일을 읽고, 접근 방식을 고려하며, 변경사항을 제안하는 과정을 실시간으로 관찰하세요. 모든 결정이 보이고, 모든 편집은 실행 전에 검토할 수 있습니다. 이것은 단순한 "설명 가능한 AI"가 아니라 완전한 투명성입니다.

### 당신의 모델, 당신의 통제

복잡한 추론에는 Claude를, 대규모 컨텍스트에는 Gemini를, 효율성에는 Qwen3 Coder를 사용하세요. 새로운 모델이 출시되면 즉시 전환할 수 있습니다. 당신의 API 키, 당신의 선택. 혁신을 가로막지 않습니다.

### 실제 엔지니어링을 위한 구축

Caret은 다음을 수행할 수 있습니다:
-   전체 코드베이스에서 **파일 읽기 및 쓰기**
-   **터미널 명령 실행** 및 오류 디버깅
-   코드를 작성하기 전에 **복잡한 기능 계획**
-   MCP 서버를 통한 **외부 시스템 연결**
-   지능적인 컨텍스트 관리로 **대규모 코드베이스 이해**

## Plan & Act 모드

Caret의 고급 계획 모드에서는 복잡한 작업을 실행 전에 단계별로 분해합니다. 이는 단순한 "코드 생성"이 아니라 전체 문제를 이해하고 체계적으로 접근하는 진정한 추론입니다.

## 시작하기

3. **페르소나 선택**: 기본 페르소나 중 선택하거나 나만의 것을 만드세요
4. **코딩 시작**: Ctrl/Cmd+Shift+P로 명령 팔레트를 열고 "Caret: New Task"를 선택하세요

## Cline과의 관계

**Caret은 [Cline](https://github.com/cline/cline) 오픈소스 프로젝트의 파생(fork)입니다.** 우리는 원본 Cline 프로젝트와 개발자들에게 깊은 감사를 표합니다.

Caret은 Cline v3.26.6을 기반으로 하며, 다음과 같은 추가 기능을 제공합니다:
- 🎭 개인화된 AI 페르소나 시스템
- 🌍 한국어, 일본어, 중국어 지원
- 🧠 향상된 시스템 프롬프트
- 🔄 Cline/Caret 모드 전환 기능

**모든 Cline의 기본 기능은 Caret에서 완전히 보존되고 작동합니다.**
