# Caret이란 무엇인가요?

**Caret = Cline v3.38.2 + 듀얼 모드 + 페르소나 + 풀 로컬라이제이션**

Caret은 최신 Cline(v3.38.2)을 100% 호환하면서, 개인화된 AI 경험과 확장 기능을 더한 AI 코딩 에이전트입니다. 자동 완성이 아닌, 코드베이스를 이해하고 계획·실행까지 수행하는 진짜 코딩 파트너입니다.

## ✨ Caret을 선택해야 하는 이유

1. **🔄 듀얼 모드** — Caret(확장) ↔ Cline(클래식) 원클릭 전환
2. **🎭 페르소나 시스템** — 나만의 AI 컴패니언과 함께 코딩
3. **🧠 최적화 프롬프트** — 비용·성능 균형 잡힌 응답
4. **🌍 풀 로컬라이제이션** — UI/문서 4개 언어 (en/ko/ja/zh)
5. **⌨️ 향상된 UX** — 입력 히스토리·단축키·자동 모델 조회
6. **🤖 폭넓은 모델** — 31개 프로바이더, 266개 고유 모델(Cline v3.38.2 호환)

### Cline 모드 (클래식 방식)
- Plan/Act 접근법 - 기존 Cline 사용자에게 익숙한 방식
- 모든 설정 호환 - 기존 설정을 그대로 사용
- 안정성 보장 - Cline의 검증된 핵심 엔진

### Caret 모드 (확장 기능)
- 페르소나 시스템 - 나만의 개인화된 AI 컴패니언
- 향상된 시스템 프롬프트 - 비용/성능 최적화
- 4개 언어 지원 - 완벽한 다국어 환경
- Chatbot/Agent - 직관적인 대화 스타일

👉 **자세한 내용**: [듀얼 모드 시스템](/ko/caret-exclusive/dual-prompt-modes)

## 🌍 언어 장벽 없는 코딩

전 세계 개발자들이 모국어로 편안하게 사용할 수 있도록 **완벽한 4개 언어 지원**:

- **영어** - 네이티브 지원 (UI, AI 채팅, 문서, 매뉴얼)
- **한국어** - 완벽한 한국어 지원 (UI, AI 채팅, 문서, 매뉴얼)
- **일본어** - 완벽한 일본어 지원 (UI, AI 채팅, 문서, 매뉴얼)
- **중국어** - 완벽한 중국어 지원 (UI, AI 채팅, 문서, 매뉴얼)

👉 **자세한 내용**: [완전한 다국어 UI](/ko/caret-exclusive/multilingual-ui)

## Cline 기반의 검증된 아키텍처

Caret은 검증된 Cline 아키텍처를 기반으로, 한국 개발자와 글로벌 사용자를 위한 추가 기능을 제공합니다.

### 완전한 투명성
Caret이 파일을 읽고, 접근 방식을 고려하며, 변경사항을 제안하는 과정을 실시간으로 관찰하세요. 모든 결정이 보이고, 모든 편집은 실행 전에 검토할 수 있습니다.

### 당신의 모델, 당신의 통제
복잡한 추론에는 Claude를, 대규모 컨텍스트에는 Gemini를, 효율성에는 Qwen3 Coder를 사용하세요. 새로운 모델이 출시되면 즉시 전환할 수 있습니다. 당신의 API 키, 당신의 선택.

### 실제 엔지니어링을 위한 구축
Caret은 다음을 수행할 수 있습니다:
- 전체 코드베이스에서 **파일 읽기 및 쓰기**
- **터미널 명령 실행** 및 오류 디버깅
- 코드를 작성하기 전에 **복잡한 기능 계획**
- MCP 서버를 통한 **외부 시스템 연결**
- 지능적인 컨텍스트 관리로 **대규모 코드베이스 이해**

## Cline과의 관계

**Caret은 [Cline](https://github.com/cline/cline) 오픈소스 프로젝트의 파생(fork)입니다.** 우리는 원본 Cline 프로젝트와 개발자들에게 깊은 감사를 표합니다.

Caret은 **Cline v3.38.2**를 기반으로 하며, **4가지 핵심 확장**을 제공합니다:
1. 🎭 개인화된 AI 페르소나 시스템
2. 🌍 한국어·일본어·중국어 풀 로컬라이제이션
3. 🧠 최적화된 프롬프트 시스템
4. 🔄 Caret/Cline 듀얼 모드 전환

**그 외에도 프롬프트 히스토리, 향상된 프로바이더 설정, 그리고 AI의 신뢰도를 높이는 [지식 동기화 워크플로우](/ko/caret-exclusive/knowledge-parity-system) 등 [추가 독점 기능](/ko/caret-exclusive/overview)을 제공합니다.**

**모든 Cline의 기본 기능은 Caret에서 완전히 보존되고 작동합니다.**
