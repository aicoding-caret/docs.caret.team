Caret을 직접 써볼 준비가 되셨나요? 이 튜토리얼은 1분 안에 웹사이트를 만드는 과정을 안내합니다. Caret이 요구사항을 이해하고 파일을 생성하며, 피드백을 반영해 수정하는 흐름을 직접 경험할 수 있습니다.

이 가이드를 끝내면 실제로 동작하는 웹사이트를 만들고 Caret과 함께 작업하는 기본을 익히게 됩니다.

## 사전 준비

- **에디터에 Caret 설치** ([설치 가이드](/ko/getting-started/installing-caret))
- **AI 모델 선택 완료** ([모델 설정](/ko/getting-started/selecting-your-model))
- **에디터에서 폴더 열기** (또는 새 빈 폴더 생성)

## 1단계: Caret 열기

에디터 왼쪽 사이드바에서 Caret 아이콘을 클릭하면 채팅 패널이 열립니다.

<Info>
**빠른 팁:** `Cmd+Shift+P`(Mac) 또는 `Ctrl+Shift+P`(Windows/Linux)를 눌러 "Caret: Open In New Tab"을 검색해도 열 수 있습니다.
</Info>

## 2단계: Caret에게 작업 요청하기

아래 프롬프트를 Caret 채팅에 복사해 붙여넣으세요:

```
Create a simple website in a single HTML file. It should have:
- A welcome message saying "Hello from Caret!"
- A colorful gradient background
- A button that cycles through different color themes when clicked
- Modern, clean design
- All CSS and JavaScript should be included in the same HTML file
```

<Frame>
  <img src="https://storage.googleapis.com/cline_public_images/chat-prompt.png" alt="Caret Chat Prompt"/>
</Frame>

Enter를 누르면 Caret이 바로 작업을 시작합니다.

## 3단계: 다음에 일어나는 일

Caret은 다음을 수행합니다:

1. **단일 파일 생성**
   - `index.html` - CSS/JavaScript가 포함된 완성형 웹페이지

2. **승인 요청** (자동 승인 활성화 시 제외)
   - "Approve"를 클릭하면 파일을 생성
   - 실행 전에 계획을 검토할 수 있음

3. **수 초 내 작업 완료**

## 4단계: 웹사이트 확인하기

Caret이 완료하면:

1. **`index.html` 찾기** - 에디터의 파일 탐색기에서 확인
2. **우클릭 후 열기**
   - "Reveal in Finder/Explorer"로 위치를 확인한 뒤 브라우저에서 열기
3. **버튼 클릭** - 색상 테마가 바뀌는지 확인

## 변경 요청해 보기

같은 채팅에서 아래처럼 추가 요청해 보세요:

```
Add a counter that shows how many times the button has been clicked
```

또는

```
Make the welcome message fade in when the page loads
```

Caret은 이전 대화의 컨텍스트를 이해하고 파일을 업데이트합니다.

<Tip>
**이제 할 수 있는 것:**
- 명확한 프롬프트로 Caret에게 작업 요청하기
- Caret의 변경 사항을 검토하고 승인하기
- 수 초 만에 완성형 프로젝트 생성하기
- 기존 작업을 반복 개선하기
</Tip>
