<Note>
캐럿(Caret) 기준 문서입니다. Caret v3.38.1 머지본을 따르며, 캐럿 전용 정책(지원 모델/권장 설정, 인증/라우팅)이 있을 경우 본문에서 `<Note>`로 표시합니다.
</Note>

Caret가 어떻게 동작하는지 직접 체험해 보세요. 이 튜토리얼은 1분 안에 웹사이트를 만드는 과정을 안내합니다. Caret이 요구사항을 이해하고, 파일을 만들고, 피드백에 따라 반복하는 흐름을 자연스러운 대화로 경험할 수 있습니다.

이 가이드를 끝내면 실제로 동작하는 웹사이트를 만들고, Caret 사용의 기본을 익히게 됩니다.

## 준비 사항

- **Caret 설치 완료** ([설치 가이드](/ko/getting-started/installing-caret))
- **AI 모델 선택** ([모델 설정](/ko/getting-started/selecting-your-model))
- **에디터에 폴더 열기** (새 빈 폴더도 가능)

## 1단계: Caret 열기

에디터 사이드바(왼쪽)에서 Caret 아이콘을 클릭하면 채팅 패널이 열립니다.

<Info>
**빠른 팁:** `Cmd+Shift+P`(Mac) 또는 `Ctrl+Shift+P`(Windows/Linux)에서 "Caret: Open In New Tab"을 검색해 열 수도 있습니다.
</Info>

## 2단계: 작업 요청하기

다음 프롬프트를 Caret 채팅에 복사해 붙여 넣으세요:

```
Create a simple website in a single HTML file. It should have:
- A welcome message saying "Hello from Caret!"
- A colorful gradient background
- A button that cycles through different color themes when clicked
- Modern, clean design
- All CSS and JavaScript should be included in the same HTML file
```

  <img src="https://storage.googleapis.com/cline_public_images/chat-prompt.png" alt="Caret Chat Prompt"/>
</Frame>

Enter를 누르면 Caret가 작업을 시작합니다.

## 3단계: 다음에 일어나는 일

Caret는 다음을 수행합니다:

1. **단일 파일 생성**
   - `index.html` (CSS/JS 포함)

2. **승인 요청** (자동 승인 설정이 없을 때)
   - "Approve"를 눌러 파일 생성 허용
   - 실행 계획을 먼저 검토 가능

3. **작업 완료**
   - 몇 초 내에 완료

## 4단계: 웹사이트 확인

Caret 작업이 끝나면:

1. 파일 탐색기에서 **`index.html` 찾기**
2. **우클릭** 후 다음 중 선택:
   - "Reveal in Finder/Explorer" → 브라우저로 열기
3. 버튼을 클릭해 테마가 바뀌는지 확인

## 변경 시도하기

같은 대화에서 아래처럼 요청해 보세요:

```
Add a counter that shows how many times the button has been clicked
```

또는

```
Make the welcome message fade in when the page loads
```

Caret는 이전 대화의 컨텍스트를 이해하고 파일을 업데이트합니다.

<Tip>
**이제 다음을 할 수 있습니다:**
- 명확한 프롬프트로 작업 요청
- Caret의 작업을 검토하고 승인
- 빠르게 프로젝트 생성
- 기존 결과를 기반으로 반복 개선
</Tip>
