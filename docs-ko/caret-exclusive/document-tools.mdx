# ^ 문서 읽기 도구

캐러티의 AI는 **경로만으로 다양한 문서 파일을 직접 읽을 수 있습니다**. 사용자가 파일을 첨부할 필요가 없습니다.

## Cline과의 차이점

| 기능 | Cline | 캐러티 |
|------|-------|-------|
| PDF 읽기 | 사용자 첨부 시에만 | **AI가 경로로 직접 읽기** |
| DOCX/XLSX 읽기 | 사용자 첨부 시에만 | **AI가 경로로 직접 읽기** |
| 한글 (HWPX) | 미지원 | **지원** |
| 한글 5.0 (HWP) | 미지원 | **지원** |
| PowerPoint (PPTX) | 미지원 | **지원** |
| Jupyter Notebook | 미지원 | **지원** |

## 지원 형식

| 형식 | 확장자 | 설명 |
|------|--------|------|
| PDF | `.pdf` | 텍스트 추출 |
| Word | `.docx` | 최신 Word 문서 |
| Excel | `.xlsx` | 최신 Excel 스프레드시트 |
| PowerPoint | `.pptx` | 슬라이드 텍스트 추출 |
| 한글 (신형) | `.hwpx` | 한글 2014 이후 |
| 한글 (구형) | `.hwp` | 한글 5.0 ~ 2010 |
| Jupyter | `.ipynb` | 노트북 셀 내용 |

### 미지원 형식

레거시 바이너리 형식은 지원하지 않습니다:

| 형식 | 확장자 | 대안 |
|------|--------|------|
| Word 97-2003 | `.doc` | `.docx`로 변환 |
| Excel 97-2003 | `.xls` | `.xlsx`로 변환 |
| PowerPoint 97-2003 | `.ppt` | `.pptx`로 변환 |

변환 도구: LibreOffice, Google Docs, Microsoft Office

## 사용 방법

### 예시 1: 사양 문서 분석

```
사용자: docs/spec.pdf 분석해줘
AI: [문서 읽기 도구 사용] → PDF 내용 추출 → 분석 결과 제공
```

### 예시 2: 한글 문서 읽기

```
사용자: 계약서.hwp 내용 요약해줘
AI: [문서 읽기 도구 사용] → HWP 텍스트 추출 → 요약 제공
```

### 예시 3: 엑셀 데이터 분석

```
사용자: sales-report.xlsx 데이터 분석해줘
AI: [문서 읽기 도구 사용] → 스프레드시트 데이터 추출 → 분석 결과 제공
```

## 보안

### 경로 보호
- 경로 정규화로 디렉토리 탈출 공격 방지
- 워크스페이스 내부 파일만 자동 승인
- 외부 파일은 사용자 승인 필요

### 파일 크기 제한
- 최대 50MB
- 초과 시 명확한 에러 메시지 표시

## 설정

문서 읽기 도구는 기본적으로 활성화되어 있습니다. 읽기 전용 작업이므로 별도의 설정 토글이 없습니다.

## 알려진 제한사항

1. **이미지/차트 미포함**: 문서 내 이미지, 차트는 텍스트로 변환되지 않습니다
2. **복잡한 레이아웃**: 표, 다단 레이아웃은 단순 텍스트로 변환됩니다
3. **원본 서식 손실**: 글꼴, 색상 등 서식 정보는 추출되지 않습니다
