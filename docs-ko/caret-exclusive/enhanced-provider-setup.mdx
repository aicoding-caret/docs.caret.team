# 향상된 프로바이더 설정

## 🚀 Caret의 프로바이더 경쟁력

**Caret은 Cline보다 더 많은 프로바이더를 지원하고, 더 특화된 프로바이더 기능을 제공합니다.**

### 왜 Caret의 프로바이더가 더 강력한가?

1. **자동 모델 조회**: LiteLLM API 연동으로 100개 이상 프로바이더의 모델을 자동으로 표시
2. **실시간 업데이트**: 새 모델 출시 시 Caret 업데이트 없이 즉시 사용 가능
3. **BizRouter·국내 프로바이더**: BizRouter/LiteLLM 개선, 국내 프로바이더(Naver, Upstage 등) 순차 추가 예정
4. **간편한 설정**: 수동 입력 불필요, 드롭다운에서 선택만 하면 완료

## 📋 개요

Caret은 **LiteLLM API 연동**을 통해 프로바이더 설정 과정을 완전히 자동화했습니다. 더 이상 지원 모델을 일일이 찾아보거나 수동으로 입력할 필요가 없습니다.

## 🎯 핵심 기능

### 자동 모델 조회

프로바이더를 선택하면 해당 프로바이더가 지원하는 **모든 AI 모델이 자동으로 표시**됩니다.

**작동 방식**:
1. Caret 설정에서 AI 프로바이더 선택 (예: OpenAI, Anthropic)
2. LiteLLM API가 자동으로 지원 모델 목록 조회
3. 드롭다운에서 원하는 모델 선택

### 실시간 업데이트

새로운 AI 모델이 출시되면 **Caret을 업데이트할 필요 없이** 자동으로 모델 목록에 추가됩니다.

- LiteLLM API를 통해 최신 모델 정보를 실시간으로 받아옵니다
- 항상 최신 모델 라인업을 사용할 수 있습니다

### 광범위한 프로바이더 지원

**100개 이상의 AI 프로바이더**를 지원합니다:

- **주요 글로벌 프로바이더**:
  - OpenAI (GPT-4, GPT-3.5-Turbo 등)
  - Anthropic (Claude 시리즈)
  - Google (Gemini)
  - Cohere
  - AI21 Labs
  - Hugging Face

- **추가 지원 프로바이더**:
  - Mistral AI
  - Fireworks AI
  - Groq
  - Cerebras
  - Deepseek
  - 그 외 다수

## 🔄 Cline과의 차이점

| 항목 | Cline | Caret |
|------|-------|-------|
| 모델 입력 방식 | 수동 입력 | 자동 조회 |
| 모델 목록 확인 | 문서 참고 필요 | 드롭다운 자동 표시 |
| 신규 모델 지원 | 업데이트 필요 | 자동 반영 |
| 지원 프로바이더 | 제한적 | 100개 이상 |

## 💡 사용 시나리오

### 시나리오 1: 프로바이더 탐색

여러 AI 프로바이더를 비교하고 싶을 때:

1. 각 프로바이더를 선택하면 지원 모델이 자동으로 표시됩니다
2. 모델 이름, 성능, 가격을 비교합니다
3. 작업에 가장 적합한 모델을 선택합니다

### 시나리오 2: 최신 모델 사용

새로운 AI 모델이 출시되었을 때:

1. Caret을 재시작합니다
2. 프로바이더 설정을 엽니다
3. 새 모델이 자동으로 목록에 표시됩니다

### 시나리오 3: 한국 AI 모델 사용

한국 기업의 AI 모델을 사용하고 싶을 때:

1. 한국 프로바이더 선택 (예: 웍스AI - 추가 예정)
2. 지원 모델 자동 표시
3. 한국어에 특화된 모델 선택

## 🌏 BizRouter·국내 프로바이더 협력

- **BizRouter**: LiteLLM과 함께 모델 자동 조회/건강 체크 개선
- **국내 프로바이더**: Naver, Upstage 등 순차 추가 예정(한국어 모델 우선)
- 모든 통합은 자동 모델 조회·실시간 업데이트를 그대로 활용

## 💼 프로바이더 추가 협력

### 새로운 프로바이더를 추가하고 싶으신가요?

**Caret 팀과 비즈니스 협력**을 통해 귀사의 AI 프로바이더를 Caret에 공식 통합할 수 있습니다.

### 협력의 장점

1. **자동 모델 조회**: LiteLLM API 연동으로 귀사의 모든 모델이 자동으로 표시됩니다
2. **마케팅 효과**: Caret 사용자들에게 귀사의 AI 서비스를 노출할 수 있습니다
3. **기술 지원**: Caret 팀이 통합 과정을 전담 지원합니다
4. **지속적인 업데이트**: 신규 모델 출시 시 자동으로 Caret에 반영됩니다

### 협력 프로세스

1. **문의**: support@caretive.ai로 협력 제안
2. **검토**: Caret 팀이 기술적 타당성 검토
3. **통합**: API 연동 및 테스트
4. **출시**: Caret 공식 프로바이더로 등록
