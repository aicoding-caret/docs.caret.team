# Caret 제공자

Caret 제공자는 API 키 없이도 즉시 AI 코딩 어시스턴트를 사용할 수 있게 해주는 통합 인증 시스템입니다. 현재 개발 중이며, 곧 출시될 예정입니다.

## 무엇인가요?

Caret을 처음 사용하는 많은 개발자들이 겪는 첫 번째 장벽은 바로 API 키 설정입니다. OpenAI나 Anthropic 같은 AI 제공자의 API 키를 발급받고, 결제 정보를 등록하고, 사용량을 관리하는 과정은 번거로울 수 있습니다.

Caret 제공자는 이러한 초기 진입 장벽을 제거합니다. Caret 계정 하나만 있으면, 별도의 API 키 설정 없이 즉시 AI 어시스턴트를 사용할 수 있습니다.

## Google Gemini 우선 지원

Caret 제공자의 첫 번째 지원 모델은 **Google Gemini**가 될 예정입니다. Gemini는 강력한 코딩 능력을 가지고 있으면서도, 상대적으로 저렴한 가격으로 제공되어 초보자와 개인 개발자에게 적합합니다.

## 무료 크레딧

Caret 제공자를 처음 사용하는 모든 사용자에게 **무료 크레딧**을 제공할 예정입니다. 이를 통해 실제 비용 부담 없이 Caret의 기능을 충분히 테스트해볼 수 있습니다.

## 향후 계획

Gemini 지원 이후, 다른 AI 모델들도 순차적으로 Caret 제공자를 통해 사용할 수 있도록 추가할 계획입니다.

## 기존 API 키와의 관계

Caret 제공자가 출시되어도, 기존의 직접 API 키 입력 방식은 계속 지원됩니다. 다음과 같은 선택지를 가질 수 있습니다:

**Caret 제공자 사용**: 간편한 시작, 통합 관리, 무료 크레딧 활용 (Gemini 우선, ChatGPT·Claude·국내 프로바이더(Naver/Upstage 등) 순차 지원 예정)

**직접 API 키 사용**: 기존 계정 활용, 더 많은 모델 선택, 비용 직접 관리

두 방식을 함께 사용하는 것도 가능합니다.
