# Caret만의 특별한 기능들

Caret은 단순히 Cline을 리브랜딩한 것이 아닙니다. Cline의 견고한 기반 위에 실질적인 확장 기능을 추가하여 더 나은 코딩 경험을 제공합니다.

## 한눈에 비교하기

| 기능 | Cline | Caret |
|------|-------|-------|
| 페르소나 시스템 | ❌ | ✅ |
| 듀얼 모드 (Agent/Chatbot) | ❌ | ✅ |
| 프롬프트 히스토리 (↑↓ 키) | ❌ | ✅ |
| 규칙 우선순위 시스템 | ❌ | ✅ |
| 완전한 다국어 UI | ❌ | ✅ (4개 언어) |
| 프로바이더 자동 조회 | ❌ | ✅ |

## Caret 독점 기능 둘러보기

### 🎭 페르소나 시스템

AI의 이름과 아바타를 커스터마이징하여 나만의 코딩 파트너를 만드세요.

- 5가지 기본 페르소나 제공 (Caret, 오사랑, 마도베 이치카, 시안 맥킨, 탄도 우분투)
- 커스텀 이미지 업로드 (일반/생각중 2가지)
- persona.md 파일로 성격 정의

[자세히 보기 →](/ko/caret-exclusive/persona-system)

### 🔄 듀얼 모드 시스템

작업에 따라 최적화된 두 가지 모드를 전환하세요:

- **Agent 모드**: 자율적인 AI 파트너 (대화 전환 유연)
- **Chatbot 모드**: 개발자 주도 조언자 (AI 통제)

*참고: Cline은 Plan/Act 단일 방식만 지원*

[자세히 보기 →](/ko/caret-exclusive/dual-prompt-modes)

### ⌨️ 프롬프트 히스토리 시스템

터미널과 동일한 ↑↓ 방향키로 이전 입력을 빠르게 재사용하세요.

- 영구 저장 (VS Code 재시작 후에도 유지)
- 최대 1000개 히스토리
- 중복 자동 제거

### 📋 규칙 우선순위 시스템

여러 규칙 파일을 지능적으로 통합합니다:

```
우선순위: .caretrules > .clinerules > .cursorrules
```

- 중복 규칙 자동 차단 (토큰 절약)
- 최우선 규칙만 프롬프트에 포함

[자세히 보기 →](/ko/caret-exclusive/advanced-rules)

### 🤝 AI-개발자 지식 동기화

Caret의 AI는 어떻게 개발자와 동일한 지식을 공유하여 높은 신뢰도를 확보할까요? 그 배경이 되는 오픈소스 개발 워크플로우를 소개합니다. 이 시스템은 Caret AI의 투명성과 신뢰성의 근간이 됩니다.

[자세히 보기 →](/ko/caret-exclusive/knowledge-parity-system)

### 🌍 완전한 다국어 지원

UI와 문서가 모두 번역된 진정한 다국어 지원:

- 한국어, 영어, 일본어, 중국어
- 모든 설정과 메시지 번역
- 언어별 문서 사이트

[자세히 보기 →](/ko/caret-exclusive/multilingual-ui)

### 🎨 OEM 브랜드 전환 (기업용)

Caret과 CodeCenter 브랜딩을 실시간으로 전환 가능한 시스템입니다.

- 비공개 커스터마이징 프로젝트
- 화이트라벨 솔루션

**문의**: support@caret.team

[자세히 보기 →](/ko/caret-exclusive/brand-switching)

### 🚀 프로바이더 설정 자동화 (연관 기능)

AI 모델을 선택할 때 어떤 모델이 사용 가능한지 일일이 찾아보는 것은 번거로운 일입니다. Caret은 LiteLLM API와 통합하여 이 과정을 자동화했습니다.

**자동 모델 조회**: 프로바이더를 선택하면, 해당 프로바이더가 지원하는 모든 AI 모델이 자동으로 표시됩니다. OpenAI를 선택하면 GPT-4, GPT-3.5-Turbo 등의 모델 목록이 즉시 나타나며, Anthropic을 선택하면 Claude 시리즈가 표시됩니다.

**실시간 업데이트**: LiteLLM API를 통해 최신 모델 정보를 실시간으로 받아오기 때문에, 새로운 AI 모델이 출시되면 Caret을 업데이트할 필요 없이 자동으로 목록에 추가됩니다.

**광범위한 지원**: OpenAI, Anthropic, Google Gemini를 포함하여 **100개 이상의 AI 프로바이더**를 지원합니다. Cohere, AI21 Labs, Hugging Face 등 다양한 프로바이더의 모델을 손쉽게 사용할 수 있습니다.

**한국 AI 기업 협력**: Caret은 한국 AI 기업들과 협력하여 더 많은 프로바이더를 지원하고 있습니다. 한국어에 특화된 AI 모델을 활용하고 싶다면, 앞으로 Caret에서 간편하게 설정할 수 있게 될 것입니다.

이러한 자동화 덕분에 개발자는 복잡한 설정 과정 없이, 원하는 AI 모델을 빠르게 선택하고 코딩을 시작할 수 있습니다.

[자세히 보기 →](/ko/caret-exclusive/enhanced-provider-setup)

## 왜 Caret을 선택해야 할까요?

### 즉시 시작 가능

무료 Gemini 크레딧으로 API 키 설정 없이 바로 코딩을 시작할 수 있습니다 (출시 예정).

### 터미널과 일관된 UX

프롬프트 히스토리 시스템으로 bash/zsh와 동일한 ↑↓ 키 경험을 제공합니다.

### 당신만의 AI

페르소나 시스템으로 AI의 이름과 아바타를 원하는 대로 커스터마이징하세요.

### 유연한 작업 방식

Agent 모드는 유연한 대화, Chatbot 모드는 개발자 주도 작업에 최적화되어 있습니다.

### 모국어 지원

한국어를 포함한 4개 언어의 완전한 지원으로 더 편안하게 사용하세요.
