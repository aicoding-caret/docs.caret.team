# 에이전트 표준화(AAIF SoT)

Caret는 규칙 경로를 단일 표준으로 통일했습니다: `.agents/context/`. 레거시 규칙 경로는 지원하지 않습니다.

## 표준 규칙 소스

- 워크스페이스 규칙: `.agents/context/`
- 디렉토리 범위 지시사항: `AGENTS.md` (재귀)
- 전역 규칙: Documents/Caret/Rules
- 워크플로우: `.agents/context/workflows/` (온디맨드)

## 변경 이유

- 중복 로딩 방지
- 규칙 충돌 제거
- 결정적 로딩 유지

## 초기화

- 표준 구조가 없으면 `/init`이 스캐폴드를 생성합니다.
- 기존 파일은 덮어쓰지 않습니다.

## 마이그레이션

레거시 규칙 파일/디렉토리를 `.agents/context/`로 이동하세요.
