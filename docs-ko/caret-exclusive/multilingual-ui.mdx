# 완전한 다국어 UI

모국어로 편안하게 코딩하세요. Caret은 UI와 문서를 네이티브 언어로 제공하며, 한국을 포함한 동아시아권부터 순차적으로 확장 중입니다.

## 지원 언어

Caret은 4개 언어를 완벽하게 지원합니다:

### 🇰🇷 한국어 (Korean)
- **UI**: 모든 설정, 메뉴, 메시지 한글화
- **문서**: docs.caret.team/ko
- **AI 응답**: 한국어로 자동 응답

### 🇺🇸 영어 (English)
- **UI**: 기본 언어
- **문서**: docs.caret.team/en
- **AI 응답**: 영어로 자동 응답

### 🇯🇵 일본어 (Japanese)
- **UI**: 완전한 일본어 번역
- **문서**: docs.caret.team/ja
- **AI 응답**: 일본어로 자동 응답

### 🇨🇳 중국어 (Chinese)
- **UI**: 간체 중국어 지원
- **문서**: docs.caret.team/zh
- **AI 응답**: 중국어로 자동 응답

## Cline과의 차이

| 항목 | Cline | Caret |
|------|-------|-------|
| UI 다국어 | ❌ 영어만 | ✅ 4개 언어 |
| 문서 다국어 | ❌ 영어만 | ✅ 4개 언어 |
| 에러 메시지 | ❌ 영어만 | ✅ 번역됨 |
| 설정 화면 | ❌ 영어만 | ✅ 완전 번역 |
| AI 응답 언어 | 🔄 요청 시 | ✅ 자동 감지 |

## 언어 설정 방법

### 1. 자동 감지 (권장)

Caret은 VS Code 언어 설정을 자동으로 감지합니다:

```json
// VS Code settings.json
{
  "locale": "ko"  // 한국어 자동 적용
}
```

### 2. 수동 설정

Caret 설정에서 직접 선택:

1. 설정 열기 (⚙️)
2. **일반** 탭 → **언어** 섹션
3. 원하는 언어 선택
4. VS Code 재시작

### 3. 명령 팔레트

```
Cmd+Shift+P (Mac) / Ctrl+Shift+P (Windows/Linux)
> Caret: Change Language
```

## 번역된 UI 요소

### 설정 화면

**모든 설정 항목이 번역됩니다**:

- ✅ 탭 이름 (일반, API, 기능, 터미널)
- ✅ 설정 레이블 (프롬프트 모드, 브랜드, API 키)
- ✅ 도움말 텍스트
- ✅ 버튼 ("저장", "취소", "재설정")
- ✅ 드롭다운 옵션

**예시 (한국어)**:
```
일반 설정
├─ AI 모드: Agent 모드
├─ 브랜드: Caret
└─ 자동 승인: 프로젝트 파일 읽기
```

### 채팅 인터페이스

- ✅ 입력 플레이스홀더: "Caret에게 무엇을 도와드릴까요?"
- ✅ 상태 메시지: "작업 진행 중...", "완료"
- ✅ 에러 메시지: "API 키가 설정되지 않았습니다"
- ✅ 시스템 알림

### 사이드바 및 메뉴

- ✅ 사이드바 타이틀
- ✅ 컨텍스트 메뉴
- ✅ 툴팁
- ✅ 상태 표시줄

## 문서 사이트 다국어

### 언어별 전용 도메인

각 언어마다 독립적인 문서 사이트:

- **한국어**: https://docs.caret.team/ko
- **영어**: https://docs.caret.team/en
- **일본어**: https://docs.caret.team/ja
- **중국어**: https://docs.caret.team/zh

### 자동 리다이렉션

Caret UI에서 문서 링크 클릭 시:

```typescript
// 현재 설정 언어에 따라 자동 연결
언어: 한국어 → docs.caret.team/ko/getting-started
언어: 영어   → docs.caret.team/en/getting-started
언어: 일본어 → docs.caret.team/ja/getting-started
언어: 중국어 → docs.caret.team/zh/getting-started
```

## AI 응답 언어

### 자동 언어 매칭

설정한 UI 언어로 AI가 자동 응답합니다:

**한국어 설정 시**:
```
사용자: "React 컴포넌트 만들어줘"
AI: "React 컴포넌트를 생성하겠습니다..."
```

**영어 설정 시**:
```
User: "Create a React component"
AI: "I'll create a React component..."
```

### 수동 언어 지정

특정 요청에만 다른 언어 사용:

```
"Explain this code in English"
→ 한국어 UI지만 영어로 응답

"이 코드를 한국어로 설명해줘"
→ 영어 UI지만 한국어로 응답
```

## 혼합 언어 프로젝트

다국어 팀에서도 문제없이 사용:

### 시나리오: 한국-미국 협업 팀

**개발자 A (한국)**:
- UI 언어: 한국어
- AI 응답: 한국어
- 문서: docs.caret.team/ko

**개발자 B (미국)**:
- UI 언어: 영어
- AI 응답: 영어
- 문서: docs.caret.team/en

**공유 코드**: 동일하게 작동 ✅

### 코드 주석 언어

AI에게 특정 언어로 주석 작성 요청:

```
"Add comments in Korean"
→ 한국어 주석 추가

"Add comments in English"
→ 영어 주석 추가
```
