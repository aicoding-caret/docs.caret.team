# AI 페르소나 커스터마이징

Caret의 페르소나 시스템으로 AI의 이름과 아바타를 커스터마이징하세요. 나만의 개인화된 코딩 파트너를 만들 수 있습니다.

## 페르소나란?

페르소나는 AI의 다음 요소를 정의합니다:

- **이름**: AI의 표시 이름
- **아바타 이미지**: 기본 상태 이미지
- **생각 중 이미지**: AI 작업 시 표시되는 이미지
- **성격 정의**: persona.md 파일로 AI의 말투와 행동 정의

## 기본 제공 페르소나

Caret은 5가지 템플릿 페르소나를 제공합니다:

### 🤖 Caret
- 친근한 로봇 친구
- 기본 페르소나

### 💖 오사랑 (Oh Sarang)
- K-POP 아이돌 개발자
- 밝고 긍정적인 톤

### 🪟 마도베 이치카 (Madobe Ichika)
- Windows 11 OS-tan
- 전문적이고 효율적

### 🍎 시안 맥킨 (Cyan Macin)
- macOS 어시스턴트
- 세련되고 우아한 스타일

### 🐧 탄도 우분투 (Tando Ubuntu)
- 오픈소스 협력자
- 커뮤니티 지향적

## 페르소나 사용 방법

### 페르소나 선택하기

1. 설정 열기 (⚙️)
2. **Persona Management** 섹션 찾기
3. **Select Persona Template** 버튼 클릭
4. 원하는 페르소나 선택

### 템플릿 전환

설정 화면에서 언제든지 다른 템플릿으로 전환할 수 있습니다.

## 커스텀 페르소나 만들기

### 이미지 업로드

Caret은 **2가지 이미지**를 사용합니다:

**1. 일반 이미지**
- 기본 상태에서 표시
- 설정에서 "Upload Normal Image" 클릭
- 이미지 파일 선택 (모든 이미지 형식 지원)

**2. 생각 중 이미지**
- AI가 작업할 때 표시
- 설정에서 "Upload Thinking Image" 클릭
- 이미지 파일 선택

**저장**: 이미지는 VS Code 내부 저장소에 자동 저장됩니다.

### 성격 정의 (persona.md)

persona.md 파일로 AI의 성격과 말투를 정의할 수 있습니다.

**저장 위치**:
```
~/Documents/Caret/Rules/persona.md
```

**예시**:
```markdown
당신은 친절하고 전문적인 개발 파트너입니다.
항상 한국어로 응답하고, 코드 설명 시 주석을 포함하세요.
변경사항은 단계별로 설명하고, 각 단계의 이유를 명시하세요.
```

파일을 저장하면 즉시 적용됩니다.

## 페르소나 vs 규칙

| 항목 | 페르소나 | 규칙 (.agents/context) |
|------|----------|-------------------|
| 범위 | AI 이름, 이미지, 성격 | 코딩 가이드라인 |
| 저장 위치 | 전역 (VS Code 내부 + persona.md) | 프로젝트 폴더 |
| 조합 | 규칙과 함께 사용 | 페르소나와 함께 사용 |

**최고의 결과**: 페르소나 + 프로젝트 규칙을 함께 사용하세요.

## 자주 묻는 질문

**Q: 페르소나가 AI 성능에 영향을 주나요?**
A: 페르소나의 성격 정의(persona.md)에 부정적인 성격을 부여하면 AI의 작업 품질에 영향을 줄 수 있습니다. "실수가 많다", "소극적이다" 같은 표현은 피하세요.

**Q: 프로젝트별로 다른 페르소나를 사용할 수 있나요?**
A: 아니요. 페르소나는 전역으로만 관리됩니다. 프로젝트별 맞춤이 필요하면 `.agents/context` 파일에 프로젝트별 지시사항을 작성하세요. (단, 이미지는 전역이므로 프로젝트별 변경 불가)

**Q: 페르소나는 어디에 저장되나요?**
A: 이미지는 VS Code 내부 저장소(globalStorage)에, persona.md는 `~/Documents/Caret/Rules/`에 저장됩니다.

**Q: 이미지 크기나 형식 제한이 있나요?**
A: 모든 이미지 형식을 지원하며, 크기 제한은 없습니다. 업로드 시 자동으로 base64로 변환됩니다.
