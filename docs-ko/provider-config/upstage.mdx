Upstage는 한국의 선도적인 AI 기업으로, Solar 시리즈 대규모 언어 모델을 개발합니다. Solar 모델은 뛰어난 한국어 성능과 효율적인 추론 속도로 알려져 있습니다.

**웹사이트:** [https://console.upstage.ai](https://console.upstage.ai)

## API 키 얻기

1. **계정 생성**: [Upstage Console](https://console.upstage.ai)에서 계정을 만드세요.
2. **API 키 생성**: 대시보드에서 API Keys 섹션으로 이동하세요.
3. **키 복사**: 새 API 키를 생성하고 안전하게 보관하세요.

## 지원되는 모델

### Solar Pro 2

- **모델 ID**: `solar-pro2`
- **컨텍스트 윈도우**: 32,768 토큰
- **최대 출력**: 4,096 토큰
- **설명**: 복잡한 작업을 위한 고급 언어 모델

Solar Pro 2는 Upstage의 최신 플래그십 모델로, 코딩, 추론, 다국어 작업에서 뛰어난 성능을 발휘합니다.

### Solar Mini

- **모델 ID**: `solar-mini`
- **컨텍스트 윈도우**: 32,768 토큰
- **최대 출력**: 4,096 토큰
- **설명**: 경량화된 빠른 모델

Solar Mini는 속도와 효율성에 최적화되어 빠른 응답이 필요한 작업에 적합합니다.

## Caret에서 구성

1. **설정 열기**: Careti 패널에서 설정 아이콘(⚙️)을 클릭하세요.
2. **프로바이더 선택**: "API 프로바이더" 드롭다운에서 "Upstage"를 선택하세요.
3. **API 키 입력**: "Upstage API 키" 필드에 API 키를 붙여넣으세요.
4. **모델 선택**: 드롭다운에서 원하는 모델을 선택하세요.

## Upstage의 특징

### 한국어 최적화

Solar 모델은 한국어 데이터로 광범위하게 학습되어 한국어 코드 주석, 문서화, 자연어 처리 작업에서 뛰어난 성능을 보입니다.

### 효율적인 아키텍처

Solar 시리즈는 Depth Up-Scaling(DUS) 기술을 사용하여 효율적인 모델 아키텍처를 구현합니다. 이를 통해 적은 파라미터로도 높은 성능을 달성합니다.

### OpenAI 호환 API

Upstage API는 OpenAI 호환 형식을 사용하므로 기존 OpenAI 기반 워크플로우와 쉽게 통합할 수 있습니다.

## 활용 팁

- **코딩 작업**: Solar Pro 2는 코드 생성, 리팩토링, 버그 수정에 효과적입니다.
- **빠른 응답**: 간단한 쿼리나 빠른 피드백이 필요한 경우 Solar Mini를 사용하세요.
- **한국어 프로젝트**: 한국어 주석이나 문서가 포함된 프로젝트에서 특히 유용합니다.
- **비용 효율성**: Solar Mini는 비용 효율적인 옵션으로, 프로토타이핑이나 개발 단계에서 활용하기 좋습니다.
