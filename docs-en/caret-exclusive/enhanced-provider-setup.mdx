## üéØ Core Features

### Automatic Model Lookup

When you select a provider, **all AI models supported by that provider are automatically displayed**.

**How It Works**:
1. Select an AI provider in Caret settings (e.g., OpenAI, Anthropic)
2. LiteLLM API automatically retrieves the list of supported models
3. Select your desired model from the dropdown

### Real-time Updates

When new AI models are released, they're **automatically added to the model list without needing to update Caret**.

- Receive the latest model information in real-time through the LiteLLM API
- Always have access to the most current model lineup
