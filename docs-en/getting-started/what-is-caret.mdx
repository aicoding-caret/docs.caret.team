## Proven Architecture Based on Cline

Caret provides additional features for Korean developers and global users based on the proven Cline architecture.

### Complete Transparency
Watch in real-time as Caret reads files, considers approaches, and suggests changes. Every decision is visible, and every edit can be reviewed before execution.

### Your Models, Your Control
Use Claude for complex reasoning, Gemini for large contexts, Qwen3 Coder for efficiency. Switch instantly when new models are released. Your API keys, your choice.

### Built for Real Engineering
Caret can:
- **Read and write files** across your entire codebase
- **Execute terminal commands** and debug errors
- **Plan complex features** before writing code
- **Connect to external systems** through MCP servers
- **Understand large codebases** with intelligent context management
