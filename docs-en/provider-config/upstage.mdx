Upstage is a leading Korean AI company developing the Solar series of large language models. Solar models are known for their excellent Korean language performance and efficient inference speed.

**Website:** [https://console.upstage.ai](https://console.upstage.ai)

## Getting an API Key

1. **Create an account**: Sign up at [Upstage Console](https://console.upstage.ai).
2. **Generate API Key**: Navigate to the API Keys section in your dashboard.
3. **Copy the key**: Create a new API key and store it securely.

## Supported Models

### Solar Pro 2

- **Model ID**: `solar-pro2`
- **Context Window**: 32,768 tokens
- **Max Output**: 4,096 tokens
- **Description**: Advanced language model for complex tasks

Solar Pro 2 is Upstage's latest flagship model, excelling in coding, reasoning, and multilingual tasks.

### Solar Mini

- **Model ID**: `solar-mini`
- **Context Window**: 32,768 tokens
- **Max Output**: 4,096 tokens
- **Description**: Lightweight and fast model

Solar Mini is optimized for speed and efficiency, suitable for tasks requiring quick responses.

## Configuration in <BrandName />

1. **Open Settings**: Click the settings icon (⚙️) in the <BrandName /> panel.
2. **Select Provider**: Choose "Upstage" from the "API Provider" dropdown.
3. **Enter API Key**: Paste your API key in the "Upstage API Key" field.
4. **Select Model**: Choose your desired model from the dropdown.

## Upstage Features

### Korean Language Optimization

Solar models are extensively trained on Korean data, delivering excellent performance for Korean code comments, documentation, and natural language processing tasks.

### Efficient Architecture

The Solar series uses Depth Up-Scaling (DUS) technology to implement an efficient model architecture. This achieves high performance with fewer parameters.

### OpenAI-Compatible API

The Upstage API uses an OpenAI-compatible format, making it easy to integrate with existing OpenAI-based workflows.

## Usage Tips

- **Coding Tasks**: Solar Pro 2 is effective for code generation, refactoring, and bug fixes.
- **Quick Responses**: Use Solar Mini for simple queries or when fast feedback is needed.
- **Korean Projects**: Particularly useful for projects with Korean comments or documentation.
- **Cost Efficiency**: Solar Mini is a cost-effective option for prototyping or development phases.
