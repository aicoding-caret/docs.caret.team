# 요구사항 : Cline 최신버전 머징 후 문서 업데이트
 - caret이 업데이트 됨에 따라, 메뉴얼이 모두 업데이트 되어야 함. 현재 레포지토리 이후 하위의 ./caret폴더에 있는 레파지토리는 무엇이 바뀌었는지 분석 후 메뉴얼 업데이트 필요
 - docs-en 폴더는 캐러티의 리브랜딩을 유지한채로 클라인 원본인 ./caret/docs 를 참고하여 갱신되어야함.
 - 한글, 중문, 일어도 업데이트에 맞춰서 갱신 필요
