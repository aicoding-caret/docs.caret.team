Standardmäßig wird Careti bei der ersten Installation in der linken Seitenleiste von VS Code neben dem Datei-Explorer und anderen Erweiterungen angezeigt. Für ein besseres Programmiererlebnis empfehlen wir jedoch, Careti in die rechte Seitenleiste zu verschieben. Auf diese Weise bleiben Ihre Projektdateien in der linken Seitenleiste sichtbar, während Sie rechts mit Careti chatten, sodass Sie die volle Sichtbarkeit Ihrer Codebasis haben, während Careti an Ihrem Projekt arbeitet.

## VS Code

So öffnen Sie Careti in der rechten Seitenleiste:

<Steps>
  <Step title="Erweiterungsansicht ausrichten">
    Stellen Sie sicher, dass Ihre Erweiterungsansicht vertikal linksbündig ausgerichtet ist
  </Step>
  <Step title="Rechte Seitenansicht öffnen">
    Klicken Sie auf die Schaltfläche, die das rechte Seitenfenster in VS Code öffnet (wird normalerweise verwendet, um den GitHub Copilot Chat zu öffnen). Verwenden Sie optional die Tastenkombination `Option + CMD/Ctrl + B`.
  </Step>
  <Step title="Careti-Symbol ziehen">
    Ziehen Sie das Careti-Symbol in den Navigationsbereich oben in dieser rechten Ansicht
  </Step>
</Steps>

## Cursor

So öffnen Sie Careti in der rechten Seitenleiste:

<Steps>
  <Step title="Erweiterungen ausrichten">
    Cursor verwendet standardmäßig eine horizontale Aktivitätsleiste, um den Platz für die KI-Chat-Oberfläche zu optimieren ([siehe hier für Details](https://cursor.com/docs/configuration/migrations/vscode#activity-bar-orientation)). Um auf vertikal umzuschalten:
    
    1. Öffnen Sie die Befehlspalette (`CMD/Ctrl + Shift + P`)
    2. Suchen Sie nach "Preferences: Open Settings (UI)"
    3. Suchen Sie nach `workbench.activityBar.orientation`
    4. Setzen Sie den Wert auf `vertical`
    5. Starten Sie Cursor neu, damit die Änderungen wirksam werden
  </Step>
  <Step title="KI-Bereich öffnen">
    Klicken Sie auf die Cursor-Würfel-Symbolschaltfläche (KI-Bereich), die den Agenten von Cursor öffnet (rechte Seitenansicht)
  </Step>
  <Step title="Careti in die Seitenleiste des KI-Bereichs ziehen">
    Ziehen Sie das Careti-Symbol direkt in die Seitenleiste des KI-Bereichs.
  </Step>
</Steps>

Nach der Einrichtung wird Careti auf der rechten Seite geladen und Sie können es wie gewohnt verwenden.
