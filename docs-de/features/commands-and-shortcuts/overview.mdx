Careti integriert sich direkt in die Oberfläche von VS Code, sodass Sie auf KI-Unterstützung zugreifen können, ohne Ihren Workflow zu unterbrechen. Diese Integrationen erscheinen als Befehle in Kontextmenüs, Tastenkombinationen und Quick Fixes im gesamten Editor.

### Was sind Editor-Integrationen?

Editor-Integrationen sind Befehle und Shortcuts, mit denen Sie Careti direkt dort verwenden können, wo Sie gerade arbeiten. Anstatt zuerst zum Careti-Panel zu wechseln, können Sie Code auswählen, mit der rechten Maustaste klicken und ihn sofort zur Unterstützung an Careti senden.
Diese Integrationen erscheinen an verschiedenen Stellen in VS Code:

-   Im Kontextmenü des Editors (Rechtsklickmenü) - "Add to Careti"
-   Im Kontextmenü des Terminals - "Add to Careti"
-   In der Source Control Ansicht - "Generate Commit Message"
-   Als Tastenkombinationen - Verschiedene Careti-Befehle
-   Als Quick Fix Optionen (Glühbirnenmenü) - "Fix with Careti", "Explain with Careti", "Improve with Careti"

### Wie sie funktionieren

Wenn Sie diese Befehle verwenden, führt Careti Folgendes aus:

-   Erfasst den relevanten Kontext (ausgewählter Code, Dateipfad, Terminalausgabe usw.)
-   Fokussiert die Careti-Oberfläche
-   Erstellt eine Konversation mit dem erfassten Kontext
-   In einigen Fällen wird automatisch ein Vorschlag generiert

Hinter den Kulissen verwenden diese Befehle die VS Code Extension API, um Befehle zu registrieren, auf den Editorstatus zuzugreifen und die VS Code-Oberfläche zu steuern.
