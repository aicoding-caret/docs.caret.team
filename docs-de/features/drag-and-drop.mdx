Das Drag & Drop von Dateien in Caret ist eine schnelle Methode, um Bilder, Code und andere Dateien zu Ihren Konversationen hinzuzufügen.

<Frame>
	<img
		src="https://storage.googleapis.com/cline_public_images/docs/assets/dran-n-drop.gif"
		alt="Dateien per Drag & Drop in den Caret-Chat ziehen"
	/>
</Frame>

<Note>Aufgrund von Besonderheiten in VS Code müssen Sie beim Ziehen von Dateien in das Caret-Chat-Eingabefeld die `Shift`-Taste gedrückt halten.</Note>

Das Drag & Drop von Workspace-Dateien in Caret erstellt automatisch eine [Dateierwähnung](/german/features/at-mentions/file-mentions). Dies ermöglicht es Ihnen, auf die Datei in Ihrer Konversation zu verweisen, ohne den Pfad manuell eingeben zu müssen.

### Ziehen aus dem Finder/File Explorer

Sie können Dateien direkt aus dem Dateimanager Ihres Systems in Caret ziehen:

	<img
		src="https://storage.googleapis.com/cline_public_images/docs/assets/drag-n-drop-finder.gif"

### Unterstützte Dateitypen

Caret unterstützt das Ziehen von externen Bildern, PDFs, CSV, Excel und anderen Textdateien aus Ihrem Dateisystem sowie Dateien aus Ihrem Workspace.
