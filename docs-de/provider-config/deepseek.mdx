Caret unterstützt den Zugriff auf Modelle über die DeepSeek API, einschließlich `deepseek-chat` und `deepseek-reasoner`.

**Website:** [https://platform.deepseek.com/](https://platform.deepseek.com/)

### API Key erhalten

1.  **Registrieren/Anmelden:** Gehen Sie zur [DeepSeek Platform](https://platform.deepseek.com/). Erstellen Sie ein Konto oder melden Sie sich an.
2.  **Zu API Keys navigieren:** Finden Sie Ihre API Keys im Bereich [API keys](https://platform.deepseek.com/api_keys) der Plattform.
3.  **Key erstellen:** Klicken Sie auf „Create new API key“. Geben Sie Ihrem Key einen aussagekräftigen Namen (z. B. „Caret“).
4.  **Key kopieren:** **Wichtig:** Kopieren Sie den API Key _sofort_. Er wird nicht erneut angezeigt. Bewahren Sie ihn sicher auf.

### Unterstützte Modelle

Caret unterstützt die folgenden DeepSeek-Modelle:

-   `deepseek-v3-0324` (Empfohlen für coding tasks)
-   `deepseek-r1` (Empfohlen für reasoning tasks)

### Konfiguration in Caret

1.  **Caret-Einstellungen öffnen:** Klicken Sie auf das ⚙️-Icon im Caret-Panel.
2.  **Provider auswählen:** Wählen Sie „DeepSeek“ aus dem „API Provider“-Dropdown.
3.  **API Key eingeben:** Fügen Sie Ihren DeepSeek API Key in das Feld „DeepSeek API Key“ ein.
4.  **Modell auswählen:** Wählen Sie Ihr gewünschtes Modell aus dem „Model“-Dropdown.

### Tipps und Hinweise

-   **Preise:** Weitere Informationen zu den Modellkosten finden Sie auf der Seite [DeepSeek Pricing](https://api-docs.deepseek.com/quick_start/pricing/).
