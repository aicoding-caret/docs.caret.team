Upstage ist ein führendes koreanisches KI-Unternehmen, das die Solar-Serie von Large Language Models entwickelt. Solar-Modelle sind bekannt für ihre exzellente Leistung in der koreanischen Sprache und ihre effiziente Inference-Geschwindigkeit.

**Website:** [https://console.upstage.ai](https://console.upstage.ai)

## API-Key erhalten

1. **Konto erstellen**: Registrieren Sie sich in der [Upstage Console](https://console.upstage.ai).
2. **API-Key generieren**: Navigieren Sie zum Bereich API Keys in Ihrem Dashboard.
3. **Key kopieren**: Erstellen Sie einen neuen API-Key und bewahren Sie ihn sicher auf.

## Unterstützte Modelle

### Solar Pro 2

- **Model ID**: `solar-pro2`
- **Context Window**: 32,768 tokens
- **Max Output**: 4,096 tokens
- **Beschreibung**: Fortschrittliches Sprachmodell für komplexe Aufgaben

Solar Pro 2 ist das neueste Flaggschiff-Modell von Upstage und zeichnet sich durch hervorragende Leistungen in den Bereichen Coding, Reasoning und mehrsprachige Aufgaben aus.

### Solar Mini

- **Model ID**: `solar-mini`
- **Context Window**: 32,768 tokens
- **Max Output**: 4,096 tokens
- **Beschreibung**: Leichtgewichtiges und schnelles Modell

Solar Mini ist auf Geschwindigkeit und Effizienz optimiert und eignet sich für Aufgaben, die schnelle Antworten erfordern.

## Konfiguration in Careti

1. **Einstellungen öffnen**: Klicken Sie auf das Zahnrad-Icon (⚙️) im Careti-Panel.
2. **Provider auswählen**: Wählen Sie „Upstage“ aus dem Dropdown-Menü „API Provider“.
3. **API-Key eingeben**: Fügen Sie Ihren API-Key in das Feld „Upstage API Key“ ein.
4. **Modell auswählen**: Wählen Sie Ihr gewünschtes Modell aus dem Dropdown-Menü.

## Upstage Features

### Optimierung für die koreanische Sprache

Solar-Modelle wurden intensiv mit koreanischen Daten trainiert und liefern exzellente Ergebnisse für koreanische Code-Kommentare, Dokumentationen und Aufgaben im Bereich Natural Language Processing.

### Effiziente Architektur

Die Solar-Serie nutzt die Depth Up-Scaling (DUS) Technologie zur Implementierung einer effizienten Modellarchitektur. Dies ermöglicht eine hohe Performance bei weniger Parametern.

### OpenAI-kompatible API

Die Upstage API verwendet ein OpenAI-kompatibles Format, was die Integration in bestehende OpenAI-basierte Workflows erleichtert.

## Tipps zur Verwendung

- **Coding-Aufgaben**: Solar Pro 2 ist effektiv für Code-Generierung, Refactoring und Bugfixes.
- **Schnelle Antworten**: Verwenden Sie Solar Mini für einfache Abfragen oder wenn schnelles Feedback benötigt wird.
- **Koreanische Projekte**: Besonders nützlich für Projekte mit koreanischen Kommentaren oder Dokumentationen.
- **Kosteneffizienz**: Solar Mini ist eine kosteneffiziente Option für Prototyping oder Entwicklungsphasen.
