Caret unterstützt den direkten Zugriff auf Modelle über die offizielle OpenAI API.

**Website:** [https://openai.com/](https://openai.com/)

### Getting an API Key

1.  **Registrieren/Anmelden:** Besuchen Sie die [OpenAI Platform](https://platform.openai.com/). Sie müssen ein Konto erstellen oder sich anmelden, falls Sie bereits eines haben.
2.  **Zu API Keys navigieren:** Sobald Sie angemeldet sind, gehen Sie zum [API keys Bereich](https://platform.openai.com/api-keys) Ihres Kontos.
3.  **Create a Key:** Klicken Sie auf „Create new secret key“. Es empfiehlt sich, Ihrem Key einen aussagekräftigen Namen zu geben (z. B. „Caret API Key“).
4.  **Key kopieren:** **Wichtig:** Kopieren Sie den generierten API Key sofort. Aus Sicherheitsgründen wird OpenAI ihn Ihnen nicht erneut anzeigen. Bewahren Sie diesen Key an einem sicheren Ort auf.

### Unterstützte Modelle

Caret ist mit einer Vielzahl von OpenAI-Modellen kompatibel, unter anderem:

-   'o3'
-   `o3-mini` (mittlerer Reasoning-Aufwand)
-   'o4-mini'
-   `o3-mini-high` (hoher Reasoning-Aufwand)
-   `o3-mini-low` (geringer Reasoning-Aufwand)
-   `o1`
-   `o1-preview`
-   `o1-mini`
-   `gpt-4o`
-   `gpt-4o-mini`
-   'gpt-4.1'
-   'gpt-4.1-mini'

Die aktuellste Liste der verfügbaren Modelle und ihrer Funktionen finden Sie in der offiziellen [OpenAI Models Dokumentation](https://platform.openai.com/docs/models).

### Konfiguration in Caret

1.  **Caret-Einstellungen öffnen:** Klicken Sie auf das Zahnrad-Symbol für Einstellungen (⚙️) im Caret-Panel.
2.  **Provider auswählen:** Wählen Sie „OpenAI“ aus dem Dropdown-Menü „API Provider“ aus.
3.  **API Key eingeben:** Fügen Sie Ihren OpenAI API Key in das Feld „OpenAI API Key“ ein.
4.  **Modell auswählen:** Wählen Sie das gewünschte Modell aus der Dropdown-Liste „Model“ aus.
5.  **(Optional) Base URL:** Wenn Sie einen Proxy oder eine benutzerdefinierte Base URL für die OpenAI API verwenden müssen, können Sie diese hier eingeben. Die meisten Benutzer müssen den Standardwert nicht ändern.

### Tipps und Hinweise

-   **Preise:** Lesen Sie unbedingt die [OpenAI Pricing-Seite](https://openai.com/pricing) für detaillierte Informationen zu den Kosten der verschiedenen Modelle.
-   **Azure OpenAI Service:** Wenn Sie den Azure OpenAI Service nutzen möchten, beachten Sie bitte, dass eine spezifische Dokumentation für Azure OpenAI mit Caret separat zu finden sein kann oder dass Sie ihn als OpenAI-kompatiblen Endpoint konfigurieren müssen, falls eine solche Funktionalität von Caret für benutzerdefinierte Konfigurationen unterstützt wird.
