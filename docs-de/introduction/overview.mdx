## Open Source KI-Programmierung, Unkompromittiert

Careti gibt Ihnen direkten, transparenten Zugriff auf modernste KI ohne Grenzen, ohne Überraschungen und ohne Modell-Ökosystem-Lock-in. Sehen Sie jede Entscheidung. Wählen Sie jedes Modell. Kontrollieren Sie Ihre Kosten.

  <Card title="Vollständige Transparenz" icon="eye">

  </Card>

  <Card title="Wählen Sie Ihren Inference" icon="sliders">
    Verwenden Sie Claude für komplexe Überlegungen, Gemini für massive Kontexte oder Qwen3 Coder für Effizienz. Wechseln Sie sofort, wenn neue Modelle erscheinen. Bringen Sie Ihre API-Schlüssel mit, Ihre Wahl.
  </Card>

  <Card title="Zero Trust von Anfang an" icon="lock">
    Ihr Code berührt niemals unsere Server. Careti läuft vollständig auf der Client-Seite mit Ihren API-Schlüsseln. Open Source bedeutet, dass Ihr Sicherheitsteam jede Zeile überprüfen kann.
  </Card>
</CardGroup>

## Planungs- und Aktionsmodus

Careti erforscht Ihre Codebasis und arbeitet mit Ihnen zusammen, um umfassende Pläne zu erstellen, bevor Code geschrieben wird, um sicherzustellen, dass es den vollständigen Kontext Ihres Projekts versteht.

<Tabs>
  <TabItem title="Planungsmodus" value="plan-mode" icon="lightbulb">
    Für komplexe Aufgaben erforscht Careti Ihre Codebasis, stellt Klärungsfragen und erstellt detaillierte Implementierungspläne, bevor Änderungen vorgenommen werden.

    - Informationsbeschaffung und Kontextaufbau
    - Klärungsfragen stellen
    - Detaillierte Ausführungspläne erstellen
    - Ansätze mit Ihnen besprechen
  </TabItem>

  <TabItem title="Aktionsmodus" value="act-mode" icon="play">
    Nachdem Sie den Plan genehmigt haben, implementiert Careti die Lösung mit vollständiger Transparenz und Kontrolle.

    - Geplante Aktionen ausführen
    - Tools zum Ändern von Dateien und Ausführen von Befehlen verwenden
    - Die Lösung implementieren
    - Ergebnisse und Abschlussrückmeldung bereitstellen
  </TabItem>
</Tabs>
