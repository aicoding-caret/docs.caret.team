Bereit, Caret in Aktion zu sehen? Dieses praxisorientierte Tutorial führt Sie durch die Erstellung einer Website in weniger als einer Minute. Sie werden erleben, wie Caret Ihre Anforderungen versteht, Dateien erstellt und auf Ihr Feedback eingeht – alles durch natürliche Konversation.

Am Ende dieses Leitfadens haben Sie eine funktionierende Website erstellt und die Grundlagen der Arbeit mit Caret erlernt.

## Voraussetzungen

- **Caret installiert** in Ihrem Editor ([Install Guide](/german/getting-started/installing-caret))
- **AI-Modell ausgewählt** ([Model Setup](/german/getting-started/selecting-your-model))
- **Ein beliebiger Ordner geöffnet** in Ihrem Editor (oder erstellen Sie einen neuen leeren Ordner)

## Schritt 1: Caret öffnen

Klicken Sie auf das Caret-Icon in der Seitenleiste Ihres Editors (linke Seite). Das Chat-Panel öffnet sich.

<Info>
**Schneller Tipp:** Sie können auch `Cmd+Shift+P` (Mac) oder `Ctrl+Shift+P` (Windows/Linux) verwenden und nach "Caret: Open In New Tab" suchen.
</Info>

## Schritt 2: Geben Sie Caret eine Aufgabe

Kopieren Sie diesen Prompt und fügen Sie ihn in den Caret-Chat ein:

```
Create a simple website in a single HTML file. It should have:
- A welcome message saying "Hello from Caret!"
- A colorful gradient background
- A button that cycles through different color themes when clicked
- Modern, clean design
- All CSS and JavaScript should be included in the same HTML file
```

<Frame>
  <img src="https://storage.googleapis.com/cline_public_images/chat-prompt.png" alt="Caret Chat Prompt"/>
</Frame>

Drücken Sie Enter und schauen Sie Caret bei der Arbeit zu!

## Schritt 3: Was als Nächstes passiert

Caret wird:

1. **Eine einzelne Datei erstellen:**
   - `index.html` – Eine vollständige Webseite mit eingebettetem CSS und JavaScript

2. **Um Zustimmung bitten** (außer Sie haben auto-approve aktiviert)
   - Klicken Sie auf "Approve", damit Caret die Datei erstellen kann
   - Sie können zuerst überprüfen, was Caret geplant hat

3. **Die Aufgabe abschließen** innerhalb von Sekunden

## Schritt 4: Website ansehen

Sobald Caret fertig ist:

1. **Suchen Sie `index.html`** im File Explorer Ihres Editors
2. **Rechtsklick darauf** und wählen Sie:
   - "Reveal in Finder/Explorer" und dann Doppelklick, um sie im Browser zu öffnen
3. **Klicken Sie auf den Button**, um zu sehen, wie sich die Farbthemen ändern!

## Versuchen Sie Änderungen vorzunehmen

Versuchen Sie im selben Chat zu fragen:

```
Add a counter that shows how many times the button has been clicked
```

oder

```
Make the welcome message fade in when the page loads
```

Caret versteht den Kontext aus Ihrer vorherigen Konversation und wird die Datei entsprechend aktualisieren.

<Tip>
**Sie wissen nun, wie Sie:**
- Caret eine Aufgabe mit einem klaren Prompt geben
- Die Aktionen von Caret überprüfen und genehmigen
- Ein vollständiges Projekt in Sekunden erstellen
- Bestehende Arbeiten iterieren und verbessern
</Tip>
