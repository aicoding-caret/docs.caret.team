### Wesentliche Werkzeuge

Dies sind die grundlegenden Werkzeuge, die Sie für die Entwicklung benötigen.

-   **Node.js & npm:** Erforderlich für JavaScript und Web-Entwicklung.
-   **Git:** Wird verwendet, um Codeänderungen zu verfolgen und mit anderen zusammenzuarbeiten.
-   **Paketmanager:** Ermöglicht es Ihnen, andere Entwicklungswerkzeuge einfach zu installieren.
    -   Homebrew für macOS
    -   Chocolatey für Windows
    -   apt/yum für Linux

> **Tipp:** Diese Werkzeuge bilden die Grundlage Ihres Entwickler-Toolkits. Eine ordnungsgemäße Installation sichert Ihren Erfolg.

### Lassen Sie Careti alles installieren

Kopieren Sie einen der folgenden Prompts und fügen Sie ihn in **Careti** ein, je nach Ihrem Betriebssystem.

#### Für macOS

```
Hallo Careti! Ich benötige Hilfe beim Einrichten meines Mac für die Softwareentwicklung. Können Sie mir helfen, wesentliche Entwicklungswerkzeuge und grundlegende Dienstprogramme zu installieren, die üblicherweise zum Programmieren verwendet werden, wie Homebrew, Node.js und Git? Bitte führen Sie mich Schritt für Schritt.
```

#### Für Windows

```
Hallo Careti! Ich benötige Hilfe beim Einrichten meines Windows-PCs für die Softwareentwicklung. Können Sie mir helfen, wesentliche Entwicklungswerkzeuge und grundlegende Dienstprogramme zu installieren, die üblicherweise zum Programmieren verwendet werden, wie Node.js und Git? Bitte führen Sie mich Schritt für Schritt.
```

#### Für Linux

```
Hallo Careti! Ich benötige Hilfe beim Einrichten meines Linux-Systems für die Softwareentwicklung. Können Sie mir helfen, wesentliche Entwicklungswerkzeuge und grundlegende Dienstprogramme zu installieren, die üblicherweise zum Programmieren verwendet werden, wie Node.js und Git? Bitte führen Sie mich Schritt für Schritt.
```

> **Profi-Tipp:** Careti zeigt jeden Befehl vor der Ausführung an. Sie behalten immer die Kontrolle.

### Was wird passieren?

Careti führt Sie durch diese Schritte:

1.  Installieren Sie den richtigen Paketmanager für Ihr System
2.  Verwenden Sie den Paketmanager, um Node.js und Git zu installieren
3.  Zeigen Sie die genauen Befehle vor der Ausführung an (Sie genehmigen jeden Schritt)
4.  Überprüfen Sie, ob jede Installation erfolgreich war

> **Hinweis:** Einige Schritte erfordern möglicherweise Ihr Computerpasswort. Das ist normal.

### Warum diese Werkzeuge wichtig sind

-   **Node.js & npm:**
    -   Erstellen Sie Websites mit Frameworks wie React oder Next.js
    -   Führen Sie JavaScript-Code aus
    -   Installieren Sie JavaScript-Pakete
-   **Git:**
    -   Speichern Sie verschiedene Versionen Ihres Codes
    -   Arbeiten Sie mit anderen Entwicklern zusammen
    -   Sichern Sie Ihre Arbeit
-   **Paketmanager:**
    -   Installieren und aktualisieren Sie Entwicklungswerkzeuge schnell
    -   Halten Sie Ihre Umgebung sauber und aktuell

### Notizen

> **Tipp:** Der Installationsprozess ist interaktiv. Careti führt Sie Schritt für Schritt.

-   Jeder Befehl wird vor der Ausführung zur Genehmigung angezeigt.
-   Wenn etwas schiefgeht, hilft Ihnen Careti bei der Fehlerbehebung.
-   Einige Schritte erfordern möglicherweise Ihr Computerpasswort.

### Zusätzliche Tipps für neue Programmierer

#### Terminal verstehen

Das Terminal ist eine App, in der Sie Befehle eingeben, um mit Ihrem Computer zu interagieren.

-   **macOS:** Öffnen Sie es, indem Sie "Terminal" in Spotlight suchen.
-   **Beispiel:**

```
$ open -a Terminal
```

#### VS Code-Funktionen verstehen

-   **VS Code-Terminal:** Führen Sie Befehle direkt in VS Code aus.
    -   Gehen Sie zu **View > Terminal** oder drücken Sie ``Ctrl + ` ``.
    -   **Beispiel:**

```
$ node -v
v16.14.0
```

-   **Editor-Ansicht:** Wo Sie Codedateien bearbeiten.
    -   Öffnen Sie Dateien über das Explorer-Panel auf der linken Seite.
-   **Probleme-Abschnitt:** Zeigen Sie Fehler oder Warnungen in Ihrem Code an.
    -   Klicken Sie auf das Glühbirnensymbol oder gehen Sie zu **View > Problems**.

#### Häufige Begriffe

-   **Befehlszeilenschnittstelle (CLI):** Eine leistungsstarke Möglichkeit, Befehle auszuführen.
-   **Berechtigungen:** Einige Befehle erfordern erhöhte Berechtigungen. Dies trägt dazu bei, Ihr System sicher zu halten.
