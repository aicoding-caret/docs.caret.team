# Aufgabenverwaltung

Mit der Verwendung von Careti sammeln Sie im Laufe der Zeit viele Aufgaben an. Das Aufgabenverwaltungssystem hilft Ihnen, den Aufgabenverlauf zu organisieren, zu filtern, zu durchsuchen und zu bereinigen, damit Ihr Arbeitsbereich effizient bleibt.

## Zugriff auf den Aufgabenverlauf

Sie können den Aufgabenverlauf auf zwei Arten öffnen:

1.  Klicken Sie auf die **History**-Schaltfläche in der Careti-Seitenleiste
2.  Verwenden Sie die Befehlspalette und suchen Sie nach **"Careti: Show Task History"**

## Aufgabenverlauf Funktionen

Die Aufgabenverlaufsansicht enthält mehrere leistungsstarke Tools.

### Suche und Filterung

-   **Suchleiste**: Verwenden Sie die unscharfe Suche oben, um schnell Aufgaben nach Inhalt zu finden.
-   **Sortieroptionen**: Sortieren Sie Aufgaben nach:
    -   Zuletzt hinzugefügt (Standard)
    -   Älteste
    -   Teuerste (höchste API-Kosten)
    -   Meisten Token (höchste Token-Nutzung)
    -   Am relevantesten (bei Suche)
-   **Favoriten-Filter**: Umschalten, um nur Lieblingsaufgaben anzuzeigen.

### Aufgabenaktionen

Jede Aufgabe in der Verlaufsansicht unterstützt diese Aktionen:

-   **Öffnen**: Klicken Sie auf eine Aufgabe, um sie im Careti-Chat erneut zu öffnen.
-   **Favorit**: Klicken Sie auf das Sternsymbol, um eine Aufgabe als Favorit zu markieren.
-   **Löschen**: Entfernen Sie eine einzelne Aufgabe (Favoriten sind geschützt).
-   **Exportieren**: Exportieren Sie das Gespräch als Markdown.

## Favoriten-Aufgaben

Favoriten helfen Ihnen, wichtige Aufgaben leicht zu finden und zu bewahren.

### Wie Favoriten funktionieren

-   **Favorit umschalten**: Klicken Sie auf das Sternsymbol neben einer Aufgabe.
-   **Schutz**: Lieblingsaufgaben sind vor einzelnem und Massen-Löschen geschützt (können überschrieben werden).
-   **Filterung**: Verwenden Sie den Favoriten-Filter, um schnell auf wichtige Aufgaben zuzugreifen.

## Massenaktionen

Die Aufgabenverlaufsansicht unterstützt mehrere Massenaktionen.

-   **Mehrfachauswahl**: Verwenden Sie Kontrollkästchen, um mehrere Aufgaben auszuwählen.
-   **Alle auswählen / Alle abwählen**: Schnell auswählen oder Auswahl löschen.
-   **Ausgewählte löschen**: Entfernen Sie alle ausgewählten Aufgaben.
-   **Alle löschen**: Entfernen Sie alle Aufgaben im Verlauf (Favoriten werden beibehalten, falls nicht explizit eingeschlossen).

## Best Practices

1.  **Wichtige Aufgaben als Favorit markieren**: Markieren Sie Referenzaufgaben oder häufig zugegriffene Gespräche.
2.  **Regelmäßig aufräumen**: Entfernen Sie alte oder ungenutzte Aufgaben, um die Leistung zu verbessern.
3.  **Suche verwenden**: Nutzen Sie die unscharfe Suche, um bestimmte Gespräche schnell zu finden.
4.  **Wertvolle Aufgaben exportieren**: Speichern Sie wichtige Aufgaben als Markdown zur externen Referenz.

Die Aufgabenverwaltung hilft Ihnen, einen organisierten Arbeitsablauf in Careti zu pflegen--finden Sie schnell vergangene Gespräche, bewahren Sie wichtige Aufgaben auf und halten Sie Ihren Verlauf sauber und effizient.
