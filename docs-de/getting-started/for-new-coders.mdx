> **Tipp:** Wenn Programmieren für dich völlig neu ist, gehe jeden Schritt langsam an. Es gibt keinen Grund zur Eile – Cline wird dich begleiten.

### Erste Schritte

Bevor du mit dem Programmieren beginnst, solltest du die folgenden Grundlagen vorbereiten.

#### 1. **VS Code**

Ein beliebter, kostenloser und leistungsstarker Code-Editor.

-   [<u>VS Code herunterladen</u>](https://code.visualstudio.com/)

**Empfohlenes YouTube-Tutorial:** [<u>Installation von VS Code</u>](https://www.youtube.com/watch?v=MlIzFUI1QGA)

> **Profi-Tipp:** Installiere VS Code in deinem Applications-Ordner (macOS) oder unter Program Files (Windows), damit du über das Dock oder das Startmenü leicht darauf zugreifen kannst.

#### 2. **Projektorganisation**

Erstelle einen eigenen `Careti`-Ordner in deinem Dokumente-Ordner für all deine Coding-Projekte.

-   **macOS:** `/Users/[Your Name]/Documents/Careti`
-   **Windows:** `C:\Users\[Your Name]\Documents\Careti`

Organisiere deine Projekte übersichtlich innerhalb des `Careti`-Ordners.

-   `Documents/Careti/workout-app` _(z. B. eine Fitness-Tracker-App)_
-   `Documents/Careti/portfolio-website` _(z. B. eine Portfolio-Website)_

> **Tipp:** Von Anfang an für Ordnung zu sorgen, spart später Zeit und vermeidet Verwirrung.

#### 3. **Installiere die Careti VS Code Extension**

Installiere die Careti Extension direkt in VS Code, um deinen Workflow zu optimieren.

-   Getting-Started-Tutorial für die Careti Extension

**Empfohlenes YouTube-Tutorial:** [<u>Installation von VS Code Extensions</u>](https://www.youtube.com/watch?v=E7trgwZa-mk)

> **Profi-Tipp:** Lade VS Code nach der Installation neu, um sicherzustellen, dass die Extension vollständig aktiviert ist.

Grundlegende Software, die du für effizientes Programmieren benötigst:

-   Node.js
-   Git
