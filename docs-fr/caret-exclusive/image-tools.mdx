# ^ Image Tools

L'IA de Careti peut **générer** et **analyser** des images.

## Difference from Cline

| Fonctionnalité | Cline | Careti |
|------|-------|-------|
| Image Generation | Non supporté | **Supporté via l'outil generate_image** |
| Image Analysis (modèles non-vision) | Non supporté | **Supporté via l'outil analyze_image** |
| Paramètres de Ratio/Taille | Non supporté | **Configurable dans l'UI** |
| Génération basée sur une image de référence | Non supporté | **Image-to-Image supporté** |

## Image Generation

### Exemple d'utilisation

```
User: Create a cute cat image
AI: [Uses generate_image tool] → Generates image → Saves to assets/
```

### Options de configuration

| Option | Valeurs | Description |
|------|-----|------|
| **Ratio** | `16:9`, `9:16`, `4:3`, `3:4`, `1:1` | Format d'image |
| **Size** | `1K`, `2K`, `3K`, `4K` | Résolution de l'image |

Emplacement des paramètres : Settings > Image Generation

### Emplacement de sauvegarde

Les images générées sont sauvegardées dans le dossier `assets/` du projet :

```
assets/
├── img_abc123.png        # Generated image
└── img_abc123.md         # Metadata (prompt, settings, etc.)
```

## Image Analysis

### Modèles Vision vs Modèles Texte

| Type de modèle | Méthode d'analyse d'image |
|----------|----------------|
| **Vision Models** (GPT-4o, Claude 3.5, etc.) | Analyse directe via `read_file` |
| **Text Models** (o1, GLM-4, etc.) | Utilise l'outil `analyze_image` |

### Exemple d'utilisation

```
# Non-vision model (GLM-4.7, etc.)
User: [Attaches image] What's in this image?
AI: [Uses analyze_image tool] → Analyzes with Gemini Flash → Returns results
```

### Sélection du modèle d'analyse

Lors de l'utilisation de modèles texte, vous pouvez sélectionner le modèle pour l'analyse d'image :

- `gemini-2.5-flash` (par défaut)
- `gemini-3.0-flash-preview`

Emplacement des paramètres : Settings > Model Info > Image Analysis Model

## Exigences d'authentification

| Outil | Exigences |
|------|----------|
| `generate_image` | Connexion à Careti requise |
| `analyze_image` | Connexion à Careti requise (modèles texte uniquement) |

## Configuration

### Paramètres d'Auto-approve

| Paramètre | Par défaut | Description |
|------|--------|------|
| `generateImages` | `true` | Activer l'outil de génération d'images |
| `analyzeImages` | `true` | Activer l'outil d'analyse d'images |

Emplacement des paramètres : Settings > Auto-approve

## Sécurité

### Approbation de l'analyse d'image

| Emplacement du fichier | Comportement |
|-----------|------|
| À l'intérieur du workspace | ✅ Auto-approuvé |
| À l'extérieur du workspace | ⚠️ Approbation de l'utilisateur requise |

### Formats d'image

Extensions d'image supportées :
- `.png`, `.jpg`, `.jpeg`, `.webp`
- `.gif`, `.avif`, `.bmp`, `.tiff`

## Limitations connues

1. **Authentification requise** : Les outils d'image nécessitent une connexion à Careti
2. **Exécution unique** : Les outils d'image ne peuvent s'exécuter qu'un seul à la fois
3. **Limite de taille** : La dimension maximale d'une image est de 7500px
