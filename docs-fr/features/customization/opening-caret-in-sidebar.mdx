Par défaut, lorsque vous installez Careti pour la première fois, il apparaît dans la barre latérale gauche de VS Code, à côté de votre explorateur de fichiers et d'autres extensions. Cependant, pour une meilleure expérience de codage, nous vous recommandons de déplacer Careti vers la barre latérale droite. Cela vous permet de garder vos fichiers de projet visibles dans la barre latérale gauche tout en discutant avec Careti à droite, vous offrant ainsi une visibilité complète de votre codebase pendant que Careti travaille sur votre projet.

## VS Code

Pour ouvrir Careti dans la barre latérale droite :

<Steps>
  <Step title="Aligner la vue de l'extension">
    Assurez-vous que la vue de votre extension est alignée verticalement à gauche
  </Step>
  <Step title="Ouvrir la vue latérale droite">
    Cliquez sur le bouton qui ouvre le panneau latéral droit dans VS Code (généralement utilisé pour ouvrir la conversation GitHub Copilot). Vous pouvez également utiliser le raccourci `Option + CMD/Ctrl + B`.
  </Step>
  <Step title="Faire glisser l'icône Careti">
    Faites glisser l'icône Careti vers le panneau de navigation en haut de cette vue de droite
  </Step>
</Steps>

## Cursor

Pour ouvrir Careti dans la barre latérale droite :

<Steps>
  <Step title="Aligner les extensions">
    Cursor utilise par défaut une barre d'activité horizontale pour optimiser l'espace pour l'interface de conversation AI ([voir ici pour plus de détails](https://cursor.com/french/docs/configuration/migrations/vscode#activity-bar-orientation)). Pour passer à la verticale :
    
    1. Ouvrez la Palette de commandes (`CMD/Ctrl + Shift + P`)
    2. Recherchez "Preferences: Open Settings (UI)"
    3. Recherchez `workbench.activityBar.orientation`
    4. Définissez la valeur sur `vertical`
    5. Redémarrez Cursor pour que les modifications prennent effet
  </Step>
  <Step title="Ouvrir le panneau AI">
    Cliquez sur le bouton d'icône de cube Cursor (panneau AI) qui ouvre l'agent de Cursor (panneau de vue latérale droite)
  </Step>
  <Step title="Faire glisser Careti vers la barre latérale du panneau AI">
    Faites glisser l'icône Careti directement dans la barre latérale du panneau AI.
  </Step>
</Steps>

	<img
		src="https://storage.googleapis.com/cline_public_images/Cursor-sidebar.gif"

Une fois configuré, Careti se chargera sur le côté droit et vous pourrez l'utiliser normalement.
