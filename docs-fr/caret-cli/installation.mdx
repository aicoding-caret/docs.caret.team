## Prérequis

Caret CLI nécessite Node.js version 20 ou supérieure. Nous recommandons d'utiliser Node.js 22 pour une expérience optimale.

Pour vérifier votre version de Node.js :

```bash
node --version
```

## Installation

```bash
npm install -g caret
```

Après l'installation, authentifiez-vous avec votre compte Caret :

```bash
caret auth
```

Cela lance un assistant d'authentification pour vous connecter et configurer votre fournisseur de modèle AI préféré.

<Note>
Caret CLI ne s'authentifie pas automatiquement lors de l'installation. Contrairement aux CLI classiques, vous devez exécuter `caret auth` explicitement.
`caret config set ...` sert aux paramètres du fournisseur et ne remplace pas la connexion au compte.
</Note>

## Démarrage rapide

Commencez avec Caret en quelques secondes :

```bash
caret
```

C'est tout ! L'exécution de `caret` dans n'importe quel répertoire lance une session interactive où vous pouvez discuter avec l'agent AI. Saisissez votre tâche, examinez le plan et tapez `/act` quand vous êtes prêt à l'exécuter.

Pour une exécution encore plus rapide sans interaction :

```bash
caret "Add unit tests to utils.js"
```

Cela exécute Caret avec une seule commande, ce qui est parfait pour les tâches rapides ou l'automatisation.

<Tip>
Nouveau sur Caret CLI ? Commencez par le mode interactif (`caret`) pour voir comment cela fonctionne. Une fois à l'aise, explorez [les trois flux principaux](/french/caret-cli/three-core-flows) pour des modèles d'utilisation avancés.
</Tip>
