Careti a besoin d'un modèle d'IA pour comprendre vos requêtes et écrire du code. Considérez cela comme le choix de l'expert avec lequel travailler - différents modèles ont des forces et des coûts différents.

## Démarrage Rapide : Choisir Votre Fournisseur

La façon la plus simple de démarrer est avec **Careti** comme fournisseur :

1. **Ouvrez les Paramètres Careti** : Cliquez sur l'icône d'engrenage (⚙️) dans le coin supérieur droit du chat Careti
2. **Sélectionnez "Careti"** dans le menu déroulant Fournisseur d'API
3. **Choisissez un modèle** dans le menu déroulant - nous vous recommandons de commencer avec **Claude Sonnet 4.5** ou **DeepSeek V3**

<Frame>
  <img src="https://storage.googleapis.com/cline_public_images/step2-provider.png" alt="Select Careti Provider" />
</Frame>

**C'est tout !** Pas de clés API à gérer, et vous aurez accès à plusieurs modèles.

<Tip>
**Modèles gratuits disponibles** : Careti offre occasionnellement une inférence gratuite par le biais de fournisseurs partenaires. Lorsque ces options sont disponibles, vous les verrez dans votre menu déroulant de modèles.
</Tip>

## Alternative : Utiliser un Autre Fournisseur

Si vous préférez utiliser vos propres clés API, vous pouvez choisir parmi des fournisseurs tels que :

- **OpenRouter** - Excellent rapport qualité/prix, plusieurs modèles
- **Anthropic** - Accès direct aux modèles Claude
- **OpenAI** - Accès aux modèles GPT
- **Google Gemini** - Les modèles d'IA de Google
- **Ollama** - Exécuter des modèles localement sur votre ordinateur

Après avoir sélectionné un fournisseur, vous devrez :
1. Obtenir une clé API sur son site web
2. La coller dans le champ Clé API des paramètres de Careti
3. Choisir votre modèle

<Note>
La plupart des fournisseurs exigent des informations de paiement avant de générer des clés API.
</Note>

## Quel Modèle Choisir ?

Si vous débutez, nous vous recommandons :

| Votre Priorité | Choisir Ce Modèle | Pourquoi |
|---------------|-------------------|-----|
| **Fiabilité** | Claude Sonnet 4.5 | Le plus fiable pour les tâches de codage |
| **Valeur** | DeepSeek V3 | Excellentes performances à faible coût |
| **Vitesse** | Qwen3 Coder | Réponses rapides |
| **Confidentialité** | Tout modèle Ollama | S'exécute sur votre ordinateur |

Vous pouvez changer de modèle à tout moment sans perdre votre conversation.
