> **Conseil :** Si le codage est totalement nouveau pour vous, suivez chaque étape lentement. Il n'est pas nécessaire de se précipiter -- Cline vous guidera.

### Démarrage

Avant de commencer à coder, préparez les éléments essentiels suivants.

#### 1. **VS Code**

Un éditeur de code populaire, gratuit et puissant.

-   [<u>Télécharger VS Code</u>](https://code.visualstudio.com/)

**Tutoriel YouTube recommandé :** [<u>Comment installer VS Code</u>](https://www.youtube.com/watch?v=MlIzFUI1QGA)

> **Conseil de pro :** Installez VS Code dans votre dossier Applications (macOS) ou Program Files (Windows) afin qu'il soit facile d'y accéder depuis le Dock ou le menu Démarrer.

#### 2. **Organisation du Projet**

Créez un dossier `Caret` dédié dans votre dossier Documents pour tous les projets de codage.

-   **macOS :** `/Users/[Your Name]/Documents/Caret`
-   **Windows :** `C:\Users\[Your Name]\Documents\Caret`

Organisez clairement les projets à l'intérieur du dossier `Caret`.

-   `Documents/Caret/workout-app` _(par exemple, une application de suivi de la condition physique)_
-   `Documents/Caret/portfolio-website` _(par exemple, un site de portfolio)_

> **Conseil :** Une bonne organisation dès le départ permet d'économiser du temps et d'éviter la confusion par la suite.

#### 3. **Installer l'Extension Caret VS Code**

Installez l'extension Caret directement dans VS Code pour améliorer votre flux de travail.

-   Tutoriel de démarrage de l'extension Caret

**Tutoriel YouTube recommandé :** [<u>Comment installer les extensions VS Code</u>](https://www.youtube.com/watch?v=E7trgwZa-mk)

> **Conseil de pro :** Après l'installation, rechargez VS Code pour vous assurer que l'extension est entièrement activée.

#### 4. **Outils de Développement Essentiels**

Logiciels de base dont vous aurez besoin pour un codage efficace :

-   Homebrew (macOS)
-   Node.js
-   Git

[<u>Suivez le guide étape par étape de Cline pour installer les outils de développement essentiels.</u>](https://docs.caret.bot/french/getting-started/installing-dev-essentials#installing-dev-essentials)

**Tutoriels YouTube recommandés pour les installations manuelles :**

-   **Pour macOS :**
    -   [<u>Installer Homebrew sur Mac</u>](https://www.youtube.com/watch?v=hwGNgVbqasc)
    -   [<u>Installer Git sur macOS 2024</u>](https://www.youtube.com/watch?v=B4qsvQ5IqWk)
    -   [<u>Installer Node.js sur Mac (M1 | M2 | M3)</u>](https://www.youtube.com/watch?v=I8H4wolRFBk)
-   **Pour Windows :**
    -   [<u>Installer Git sur Windows 10/11 (2024)</u>](https://www.youtube.com/watch?v=yjxv1HuRQy0)
    -   [<u>Installer Node.js sur Windows 10/11</u>](https://www.youtube.com/watch?v=uCgAuOYpJd0)

> **Remarque :** Si vous rencontrez des problèmes d'autorisation lors de l'installation, essayez d'exécuter Terminal ou Command Prompt en tant qu'administrateur.

Vous êtes prêt ! Commencez à coder plus intelligemment et plus rapidement avec **Caret**.
