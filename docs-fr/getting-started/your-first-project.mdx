Prêt à voir Careti en action ? Ce tutoriel pratique vous guidera à travers la création d'un site web en moins d'une minute. Vous découvrirez comment Careti comprend vos exigences, crée des fichiers et itère sur vos commentaires, le tout grâce à une conversation naturelle.

À la fin de ce guide, vous aurez créé un site web fonctionnel et appris les bases de l'utilisation de Careti.

## Prérequis

- **Careti installé** dans votre éditeur ([Guide d'installation](/french/getting-started/installing-caret))
- **Modèle d'IA sélectionné** ([Configuration du modèle](/french/getting-started/selecting-your-model))
- **N'importe quel dossier ouvert** dans votre éditeur (ou créez un nouveau dossier vide)

## Étape 1 : Ouvrir Careti

Cliquez sur l'icône Careti dans la barre latérale de votre éditeur (à gauche). Le panneau de chat s'ouvrira.

<Info>
**Astuce rapide :** Vous pouvez également utiliser `Cmd+Shift+P` (Mac) ou `Ctrl+Shift+P` (Windows/Linux) et rechercher "Careti: Open In New Tab"
</Info>

## Étape 2 : Donner une tâche à Careti

Copiez et collez cette invite dans le chat de Careti :

```
Create a simple website in a single HTML file. It should have:
- A welcome message saying "Hello from Careti!"
- A colorful gradient background
- A button that cycles through different color themes when clicked
- Modern, clean design
- All CSS and JavaScript should be included in the same HTML file
```

<Frame>
  <img src="https://storage.googleapis.com/cline_public_images/chat-prompt.png" alt="Careti Chat Prompt"/>

Appuyez sur Entrée et regardez Careti travailler !

## Étape 3 : Ce qui se passe ensuite

Careti va :

1. **Créer un seul fichier :**
   - `index.html` - Une page web complète avec CSS et JavaScript intégrés

2. **Demander une approbation** (sauf si vous avez activé l'approbation automatique)
   - Cliquez sur "Approve" pour permettre à Careti de créer le fichier
   - Vous pouvez d'abord vérifier ce qu'il prévoit de faire

3. **Terminer la tâche** en quelques secondes

## Étape 4 : Visualiser votre site web

Une fois que Careti a terminé :

1. **Trouvez `index.html`** dans l'explorateur de fichiers de votre éditeur
2. **Faites un clic droit dessus** et sélectionnez :
   - "Reveal in Finder/Explorer" puis double-cliquez pour l'ouvrir dans votre navigateur
3. **Cliquez sur le bouton** pour voir les thèmes de couleurs changer !

## Essayer d'apporter des modifications

Dans le même chat, essayez de demander :

```
Add a counter that shows how many times the button has been clicked
```

ou

```
Make the welcome message fade in when the page loads
```

Careti comprend le contexte de votre conversation précédente et mettra à jour le fichier en conséquence.

<Tip>
**Vous savez maintenant comment :**
- Donner une tâche à Careti avec une invite claire
- Examiner et approuver les actions de Careti
- Construire un projet complet en quelques secondes
- Itérer et améliorer le travail existant
</Tip>
