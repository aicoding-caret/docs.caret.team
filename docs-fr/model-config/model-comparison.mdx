## Comparaison des performances

### Compromis entre Vitesse et Qualité

| Priorité | Modèle recommandé | Pourquoi |
|----------|------------------|-----|
| **Vitesse** | Qwen3 Coder sur Cerebras | Inférence la plus rapide disponible |
| **Qualité** | Claude Sonnet 4.5 | Le plus fiable pour les tâches complexes |
| **Équilibre** | DeepSeek V3 | Bonne qualité à faible coût |

### Fiabilité des outils

Modèles classés par fiabilité d'utilisation des outils :
1. **Claude Sonnet 4.5** - Exécution d'outils la plus fiable
2. **GPT-5** - Excellent mais problèmes de formatage occasionnels
3. **Gemini 2.5 Pro** - Bon pour les outils standard
4. **DeepSeek V3** - Fiable pour les outils de base
5. **Qwen3 variants** - Peut nécessiter une tentative supplémentaire pour les outils complexes

### Guide de sélection du fournisseur

Choisissez votre fournisseur en fonction de :
- **Simplicité** : Careti (pas de gestion de clé API)
- **Variété** : OpenRouter (accès à tous les modèles)
- **Accès direct** : Fournisseurs individuels (Anthropic, OpenAI)
- **Confidentialité** : Ollama ou LM Studio (modèles locaux)
