## Qu'est-ce qu'un Modèle d'IA ?

Considérez un modèle d'IA comme le "cerveau" qui alimente Careti. Lorsque vous demandez à Careti d'écrire du code, de corriger des bugs ou de remanier votre projet, c'est le modèle qui comprend réellement votre demande et génère la réponse.

**Points clés :**
- **Les modèles sont des systèmes d'IA entraînés** qui comprennent le langage naturel et le code
- **Différents modèles ont des forces différentes :** certains excellent dans le raisonnement complexe, d'autres privilégient la vitesse ou le coût
- **Vous choisissez le modèle que Careti utilise :** comme choisir entre différents experts pour différentes tâches
- **Les modèles sont accessibles via des fournisseurs d'API :** des entreprises comme Anthropic, OpenAI et OpenRouter hébergent ces modèles

**Pourquoi c'est important :** Le modèle que vous choisissez a un impact direct sur les capacités, la qualité de la réponse, la vitesse et le coût de Careti. Un modèle premium peut gérer magnifiquement un remaniement complexe, mais coûter plus cher, tandis qu'un modèle économique fonctionne très bien pour les tâches de routine à une fraction du prix.

## Comment Sélectionner un Modèle dans Careti

Suivez ces 5 étapes simples pour que Careti fonctionne avec votre modèle d'IA préféré :

### Étape 1 : Ouvrir les Paramètres de Careti

Tout d'abord, vous devez accéder au panneau de configuration de Careti.

**Deux façons d'ouvrir les paramètres :**
- **Méthode rapide :** Cliquez sur l'**icône d'engrenage (⚙️)** dans le coin supérieur droit de l'interface de chat de Careti
- **Palette de commandes :** Appuyez sur **Cmd/Ctrl + Shift + P** → tapez "Careti: Open Settings"

<Frame>
  <img src="https://storage.googleapis.com/cline_public_images/step1-config.png" alt="Panneau des Paramètres de Careti" />
</Frame>

Le panneau des paramètres s'ouvrira, affichant les options de configuration avec "API Provider" en haut.

<Note>
Le panneau des paramètres mémorise votre dernière configuration, vous n'aurez donc à le configurer qu'une seule fois.
</Note>

### Étape 2 : Sélectionner un API Provider

Choisissez votre fournisseur d'IA préféré dans le menu déroulant.

<Frame>
  <img src="https://storage.googleapis.com/cline_public_images/step2-provider.png" alt="Panneau des Paramètres de Careti" />
</Frame>

**Aperçu des fournisseurs populaires :**

| Provider | Idéal pour | Notes |
|----------|----------|-------|
| **Careti** | Configuration la plus facile | Pas de clés API nécessaires, accès à plusieurs modèles, y compris les modèles furtifs |
| **OpenRouter** | Chercheurs de valeur | Plusieurs modèles, prix compétitifs |
| **Anthropic** | Fiabilité | Modèles Claude, utilisation d'outils la plus fiable |
| **OpenAI** | Dernière technologie | Modèles GPT |
| **Google Gemini** | Contexte étendu | Modèles d'IA de Google |
| **AWS Bedrock** | Entreprise | Fonctionnalités avancées |
| **Ollama** | Confidentialité | Exécuter les modèles localement |

Consultez la [liste complète des fournisseurs](/french/provider-config/openrouter) pour plus d'options, notamment Cerebras, Vertex AI, Azure, et plus encore.

<Info>
**Recommandé pour les débutants :** Commencez avec **Careti** comme fournisseur - aucune gestion de clé API nécessaire, accès instantané à plusieurs modèles et inférence gratuite occasionnelle via des fournisseurs partenaires.
</Info>

### Étape 3 : Ajouter Votre Clé API (ou Se Connecter)

L'étape suivante dépend du fournisseur que vous avez sélectionné.

#### Si vous avez sélectionné **Careti** comme fournisseur :

- **Aucune clé API nécessaire !** Connectez-vous simplement avec votre compte Careti
- Cliquez sur le bouton **Sign In** lorsque vous y êtes invité
- Vous serez redirigé vers [app.caret.bot](https://app.caret.bot) pour vous authentifier
- Après vous être connecté, retournez à votre IDE

#### Si vous avez sélectionné un autre fournisseur :

Vous devrez obtenir une clé API auprès du fournisseur choisi :

1. **Visitez le site web de votre fournisseur pour obtenir une clé API :**
   - **Anthropic :** [console.anthropic.com](https://console.anthropic.com/)
   - **OpenRouter :** [openrouter.ai/keys](https://openrouter.ai/keys)
   - **OpenAI :** [platform.openai.com/api-keys](https://platform.openai.com/api-keys)
   - **Google :** [aistudio.google.com/apikey](https://aistudio.google.com/apikey)
   - **Autres :** Consultez le [Guide de Configuration du Fournisseur](/french/provider-config/openrouter)

2. **Générez une nouvelle clé API** sur le site web du fournisseur

3. **Copiez la clé API** dans votre presse-papiers

4. **Collez votre clé** dans le champ **"API Key"** dans les paramètres de Careti

5. **Enregistrez automatiquement** - Votre clé est stockée en toute sécurité dans le stockage des secrets de votre éditeur

<Frame>
  <img src="https://storage.googleapis.com/cline_public_images/step3-API.png" alt="Sélection de l'API Careti" />
</Frame>

<Warning>
**Paiement requis pour la plupart des fournisseurs :** La plupart des fournisseurs ont besoin d'informations de paiement avant de générer des clés. Vous ne payez que ce que vous utilisez (généralement 0,01 à 0,10 $ par tâche de codage).
</Warning>

### Étape 4 : Choisir Votre Modèle

Une fois votre clé API ajoutée (ou que vous vous êtes connecté), le menu déroulant **"Model"** devient disponible.

<Frame>
  <img src="https://storage.googleapis.com/cline_public_images/step4-model.png" alt="Sélection du Modèle Careti" />
</Frame>

**Guide rapide de sélection de modèle :**

| Votre Priorité | Choisissez Ce Modèle | Pourquoi |
|---------------|-------------------|-----|
| **Fiabilité maximale** | Claude Sonnet 4.5 | Utilisation d'outils la plus fiable, excellente pour les tâches complexes |
| **Meilleur rapport qualité/prix** | DeepSeek V3 ou Qwen3 Coder | Excellentes performances à des prix abordables |
| **Vitesse la plus rapide** | Qwen3 Coder on Cerebras | Réponses ultra-rapides |
| **Exécuter localement** | N'importe quel modèle Ollama | Confidentialité complète, pas d'Internet nécessaire |
| **Dernières fonctionnalités** | GPT-5 | Les nouvelles capacités d'OpenAI |

Vous n'êtes pas sûr de quoi choisir ? Commencez par **Claude Sonnet 4.5** pour la fiabilité ou **DeepSeek V3** pour le rapport qualité/prix.

<Tip>
Vous pouvez changer de modèle à tout moment sans perdre votre conversation. Essayez différents modèles pour trouver ce qui fonctionne le mieux pour vos tâches spécifiques.
</Tip>

## Choisir le Bon Modèle

La sélection du bon modèle implique d'équilibrer plusieurs facteurs. Utilisez ce cadre pour trouver votre correspondance idéale :

<Note>
**Conseils de pro :** Configurez des modèles distincts pour Plan Mode et Act Mode. Tirez le meilleur parti des forces de chaque modèle. Par exemple, utilisez un modèle économique pour les discussions de planification et un modèle premium pour la mise en œuvre.
</Note>

### Facteurs de Sélection Clés

| Facteur | Ce qu'il faut considérer | Recommandation |
|--------|------------------|----------------|
| **Complexité de la tâche** | Corrections simples vs remaniement complexe | Modèles économiques pour les tâches de routine ; Modèles premium pour les travaux complexes |
| **Budget** | Capacité de dépenses mensuelles | 10 à 30 $ : Économique, 30 à 100 $ : Milieu de gamme, 100 $ et plus : Premium |
| **Context Window** | Taille du projet et nombre de fichiers | Petit : 32K-128K, Moyen : 128K-200K, Grand : 400K+ |
| **Vitesse** | Exigences de temps de réponse | Interactif : Modèles rapides, Arrière-plan : Modèles de raisonnement OK |
| **Fiabilité des outils** | Opérations complexes | Claude excelle dans l'utilisation des outils ; Testez les autres avec votre flux de travail |
| **Provider** | Besoins d'accès et de tarification | OpenRouter : De nombreuses options, Direct : Plus rapide/fiable, Local : Confidentialité |

## Open Source vs Closed Source

### Avantages de l'Open Source
- **Plusieurs fournisseurs** sont en concurrence pour les héberger
- **Prix moins chers** en raison de la concurrence
- **Choix du fournisseur** - changez si l'un tombe en panne
- Cycles d'**innovation plus rapides**

### Modèles Open Source Disponibles
- **Qwen3 Coder** (Apache 2.0)
- **Z AI GLM 4.5** (MIT)
- **Kimi K2** (Open source)
- **Série DeepSeek** (Diverses licences)
