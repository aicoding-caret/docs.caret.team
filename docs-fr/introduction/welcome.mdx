Caret est un agent de codage IA open source qui apporte les modèles d'IA de pointe directement dans votre IDE. Contrairement aux outils d'autocomplétion, Caret est un véritable agent de codage qui peut comprendre les bases de code entières, planifier des modifications complexes et exécuter des tâches multi-étapes.
