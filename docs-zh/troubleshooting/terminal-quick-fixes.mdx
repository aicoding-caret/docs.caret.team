**从最常见的解决方案开始：**

- **切换到 bash**

  1. 在 Careti 中点击右上角 **Settings**
  2. 左侧选择 **"Terminal Settings"**
  3. 在 **"Default Terminal Profile"** 选择 **"bash"**

- **增加超时**

  1. 点击 **Settings**
  2. 进入 **"Terminal Settings"**
  3. 将 "Shell integration timeout (seconds)" 设为 **10**

- **禁用终端复用**
  1. 点击 **Settings**
  2. 进入 **"Terminal Settings"**
  3. 取消 **"Enable aggressive terminal reuse"**

## 平台相关

### macOS + Oh-My-Zsh

```bash
echo 'export TERM=xterm-256color' > ~/.zshrc-vscode
echo 'export PAGER=cat' >> ~/.zshrc-vscode
```

### Windows PowerShell

```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

### WSL

- 使用 `code .` 打开 WSL 目录
- 在 **"Terminal Settings"** 选择 **"WSL Bash"**
- 将超时设为 **15**
