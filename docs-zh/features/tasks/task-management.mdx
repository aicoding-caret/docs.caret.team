Careti 提供任务历史管理功能，方便整理与检索。

## 访问任务历史

你可以通过以下方式打开任务历史：

1. 侧边栏 **History** 按钮
2. 命令面板搜索 "Careti: Show Task History"
3. 快捷键(取决于 VS Code 设置)

## 任务历史界面

### 搜索与过滤

#### 搜索栏
- 模糊搜索
- 覆盖提示/回复/代码
- 即输即筛选

#### 排序
- 最新
- 最早
- 最贵
- Token 最多
- 最相关

#### 收藏过滤
- 仅显示收藏

## 任务操作

### 主要操作

- **打开**
- **继续**
- **导出**

### 管理操作

- **收藏** ⭐
- **删除** 🗑️
- **复制**

## ⭐ 收藏

用于保护重要任务并快速访问。

## 任务指标

- **Token 使用量**
- **API 成本**
- **检查点数量**
