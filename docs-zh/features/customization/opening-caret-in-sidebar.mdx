---
title: "在右侧边栏打开 Caret"
description: "在 VS Code 和 Cursor 中将 Caret 移到右侧边栏"
---

<Note>
本文以 Caret 为基准，遵循 Caret v3.38.1 合并版本。如有支持编辑器/商店或 UI 标签差异，会以 `<Note>` 标注。
</Note>

<Note>
品牌/UI: 请以 `caret-docs/features/f03-branding-ui.md` 为准，优先使用 Caret 品牌(图标/标签/横幅)。
</Note>

默认情况下，Caret 会出现在 VS Code 左侧边栏。为了更好的体验，建议将 Caret 放到右侧边栏，这样左侧保留文件树，右侧与 Caret 对话。

## VS Code

<Steps>
  <Step title="确认活动栏位置">
    确保活动栏在左侧垂直显示。
  </Step>
  <Step title="打开右侧面板">
    点击打开右侧面板的按钮(通常是 GitHub Copilot Chat 按钮)，也可使用 `Option + CMD/Ctrl + B`。
  </Step>
  <Step title="拖拽 Caret 图标">
    将 Caret 图标拖拽到右侧面板顶部的导航区域。
  </Step>
</Steps>

## Cursor

<Steps>
  <Step title="调整活动栏方向">
    Cursor 默认使用横向活动栏([详见](https://cursor.com/docs/configuration/migrations/vscode#activity-bar-orientation))。切换为纵向：
    
    1. 打开命令面板 (`CMD/Ctrl + Shift + P`)
    2. 搜索 "Preferences: Open Settings (UI)"
    3. 搜索 `workbench.activityBar.orientation`
    4. 设置为 `vertical`
    5. 重启 Cursor
  </Step>
  <Step title="打开 AI 面板">
    点击 Cursor 的 AI 面板(右侧面板)。
  </Step>
  <Step title="拖拽 Caret">
    将 Caret 图标拖到 AI 面板侧边栏。
  </Step>
</Steps>

<Frame>
	<img
		src="https://storage.googleapis.com/cline_public_images/Cursor-sidebar.gif"

完成后，Caret 将在右侧加载并正常使用。
