将文件拖放到 <BrandName /> 中是向对话中添加图像、代码和其他文件的快捷方式。

<figure>
	<img
		src="https://storage.googleapis.com/cline_public_images/docs/assets/dran-n-drop.gif"
		alt="将文件拖放到 Careti 聊天中"
	/>
</figure>

:::note由于 VS Code 的特性,要将文件拖放到 <BrandName /> 聊天输入框中,您需要在拖动时按住 `Shift` 键。:::

将工作区文件拖放到 <BrandName /> 中会自动创建一个[文件引用](/zh/features/at-mentions/file-mentions)。这允许您在对话中引用文件,而无需输入路径。

### 从 Finder/文件资源管理器拖动

您可以直接从系统的文件管理器将文件拖到 <BrandName /> 中:

<figure>
	<img
		src="https://storage.googleapis.com/cline_public_images/docs/assets/drag-n-drop-finder.gif"
		alt="从 Finder 将文件拖到 Careti"
	/>
</figure>

### 支持的文件类型

<BrandName /> 支持从文件系统拖动外部图像、PDF、CSV、Excel 和其他文本文件,以及工作区中的文件。
