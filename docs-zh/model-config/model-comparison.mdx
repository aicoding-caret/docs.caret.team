## 性能对比

### 速度 vs 质量取舍

| 优先级 | 推荐模型 | 原因 |
|----------|------------------|-----|
| **速度** | Cerebras 的 Qwen3 Coder | 推理速度最快 |
| **质量** | Claude Sonnet 4.5 | 复杂任务最稳定 |
| **平衡** | DeepSeek V3 | 低成本且质量不错 |

### 工具可靠性

按工具使用稳定性排序：
1. **Claude Sonnet 4.5** - 最稳定
2. **GPT-5** - 很好但偶有格式问题
3. **Gemini 2.5 Pro** - 标准工具表现强
4. **DeepSeek V3** - 基本工具稳定
5. **Qwen3 系列** - 复杂工具需重试
