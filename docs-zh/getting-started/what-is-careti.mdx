## 📁 双目录 & /init

**Token 优化 AI 上下文 + 用户语言文档 1:1 镜像管理**

```
.agents/                       # AI 用（英语，Token 优化）
├── context/agents-rules.json  # 规则 SoT（单一事实来源）
├── workflows/                 # 任务工作流
├── skills/                    # AI 技能
└── hooks/                     # 自动化钩子

.users/                        # 用户用（用户语言）
├── context/agents-rules.md    # 详细说明
├── workflows/                 # (.agents/ 镜像)
├── skills/                    # (.agents/ 镜像)
└── hooks/                     # (.agents/ 镜像)
```

**核心功能**：
- **`/init`**：项目分析后自动生成上下文文件
- **1:1 镜像**：`.agents/` ↔ `.users/` 变更同步保持
- **AGENTS.md / CLAUDE.md**：上下文连接的标准入口点
- **遗留回退**：自动检测现有规则路径并引导迁移

👉 **了解更多**：[AI 组织知识与规则标准化](/zh/caret-exclusive/knowledge-parity-system)
