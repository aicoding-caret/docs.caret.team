# 什么是 Caret？

**Caret = Cline v3.38.2 + 双模式 + 角色系统 + 全面本地化**

Caret 与最新 Cline(v3.38.2) 100% 兼容，并加入个性化 AI 体验和扩展功能。它不是自动补全，而是真正理解代码库、能规划并执行任务的编程伙伴。

## ✨ 选择 Caret 的理由

1. **🔄 双模式** — Caret(扩展) ↔ Cline(经典) 一键切换
2. **🎭 角色系统** — 专属 AI 伴侣陪你编程
3. **🧠 优化提示** — 兼顾成本与性能的响应
4. **🌍 全面本地化** — UI/文档 4 种语言 (en/ko/ja/zh)
5. **⌨️ 强化体验** — 输入历史、快捷键、自动模型查询
6. **🤖 丰富模型** — 31 家提供商、266 个独立模型（兼容 Cline v3.38.2）

👉 **查看所有独家功能详情**：[Caret 独家功能概述](/zh/caret-exclusive/overview)

- 角色系统 - 您的个性化 AI 伴侣

- 支持 4 种语言 - 完美的多语言环境

## 基于 Cline 的验证架构

Caret 基于经过验证的 Cline 架构，为韩国开发者和全球用户提供额外功能。

### 完全透明

实时观看 Caret 读取文件、考虑方法并提出变更。每个决策都是可见的，每个编辑都可以在执行前审查。

### 您的模型，您的控制

使用 Claude 进行复杂推理，使用 Gemini 处理大量上下文，或使用 Qwen3 Coder 提高效率。新模型发布后即可立即切换。您的 API 密钥，您的选择。

### 为真实工程而构建

Caret 可以：
- 在整个代码库中**读写文件**
- **执行终端命令**并调试错误
- 在编写代码前**规划复杂功能**
- 通过 MCP 服务器**连接外部系统**
- 通过智能上下文管理**理解大型代码库**

## 与 Cline 的关系

**Caret 是 [Cline](https://github.com/cline/cline) 开源项目的分支（fork）。** 感谢原始 Cline 项目与开发者。

Caret 基于 **Cline v3.38.2**，提供**核心 4 大扩展**：
1. 🎭 个性化 AI 角色系统
2. 🌍 英/韩/日/中文全面本地化
3. 🧠 优化的提示系统
4. 🔄 Caret/Cline 双模式切换

**此外，还提供提示历史、增强的提供商设置，以及提高AI可靠性的[知识同步工作流](/zh/caret-exclusive/knowledge-parity-system)等[其他独家功能](/zh/caret-exclusive/overview)。**

**所有 Cline 的基本功能在 Caret 中都得到完全保留和运行。**
