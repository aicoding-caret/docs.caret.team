# 什么是 Caret？

**Caret = Cline + 个性化 AI 伴侣 + 增强系统提示**

Caret 是基于 Cline 的增强版 AI 编程代理，将前沿 AI 模型直接集成到您的 VS Code 编辑器中。与自动补全工具不同，Caret 是一个真正的编程代理，能够理解整个代码库，规划复杂变更，并执行多步骤任务。

## 🌟 选择 Caret 的 6 大理由

1. **🔄 双模式一键切换** - 完全兼容 Cline，随时自由切换
2. **🎭 个性化 AI 伴侣** - 为愉快编程打造的专属角色
3. **🧠 增强系统提示** - 更智能、更高效的 AI 响应
4. **🌍 支持 4 种语言** - 英语、韩语、日语、中文 + 4 种语言手册
5. **⌨️ 增强用户体验** - 提示历史、自动模型查找等
6. **🎯 25 家提供商，300 个模型** - 完全兼容 Cline v3.32.7 + 扩展

👉 **查看所有独家功能详情**：[Caret 独家功能概述](/zh/caret-exclusive/overview)

## 基于 Cline 的验证架构

Caret 基于经过验证的 Cline 架构，为韩国开发者和全球用户提供额外功能。

### 完全透明

实时观看 Caret 读取文件、考虑方法并提出变更。每个决策都是可见的，每个编辑都可以在执行前审查。

### 您的模型，您的控制

使用 Claude 进行复杂推理，使用 Gemini 处理大量上下文，或使用 Qwen3 Coder 提高效率。新模型发布后即可立即切换。您的 API 密钥，您的选择。

### 为真实工程而构建

Caret 可以：
- 在整个代码库中**读写文件**
- **执行终端命令**并调试错误
- 在编写代码前**规划复杂功能**
- 通过 MCP 服务器**连接外部系统**
- 通过智能上下文管理**理解大型代码库**

## 与 Cline 的关系

**Caret 是 [Cline](https://github.com/cline/cline) 开源项目的分支（fork）。** 我们对原始 Cline 项目和开发者表示深深的感谢。

Caret 基于 Cline v3.32.7，提供**核心 4 大附加功能**：
1. 🎭 个性化 AI 角色系统
2. 🌍 支持韩语、日语、中文
3. 🧠 增强系统提示
4. 🔄 Cline/Caret 模式切换功能

**此外，还提供提示历史、增强的提供商设置，以及提高AI可靠性的[知识同步工作流](/zh/caret-exclusive/knowledge-parity-system)等[其他独家功能](/zh/caret-exclusive/overview)。**

**所有 Cline 的基本功能在 Caret 中都得到完全保留和运行。**
