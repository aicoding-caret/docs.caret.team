Baseten 提供面向生产环境的前沿模型 API，基于 Baseten Inference Stack，为 OpenAI、DeepSeek、Meta、Moonshot、阿里等模型提供优化推理。

**官网:** [https://www.baseten.co/products/model-apis/](https://www.baseten.co/products/model-apis/)

### 获取 API Key

1. 在 [Baseten](https://www.baseten.co/) 注册/登录
2. 进入 API Keys 页面
3. 创建新 Key (命名如 "<BrandName />")
4. 复制并安全保存

### 在 <BrandName /> 中配置

1. 打开 <BrandName /> 设置(⚙️)
2. 选择 Baseten 作为 API Provider
3. 填入 API Key
4. 选择模型
