## 什么是 Careti CLI？

Careti CLI 让你在终端直接运行 AI 编码代理。你可以在 CI/CD 中将 git diff 作为输入进行自动代码审查，也可以并行运行多个实例，或集成到现有的 Shell 工作流中。

CLI 会跟踪系统内的实例，并提供适合人读的输出、适合脚本的 JSON/纯文本输出和富终端输出。

<Tip>
想快速上手？请先查看 [安装指南](/zh/caret-cli/installation)。
</Tip>

## 支持的模型提供商

Careti CLI 支持多种模型提供商，包括：

- **Anthropic**
- **OpenAI**
- **OpenAI Compatible**
- **OpenRouter**
- **X AI (Grok)**
- **AWS Bedrock**
- **Google Gemini**
- **Ollama**
- **Cerebras**

安装过程中可通过 `caret auth` 完成认证与配置。

## 你可以做什么

**自动代码维护**
- 定时执行查找并修复 lint 问题
- 扫描安全漏洞并自动修复
- 更新过时依赖并运行测试

**多实例并行开发**
- 同时运行前端/后端实例
- 按分支拆分上下文
- 并行审查多个 PR

**自定义工作流**
- 与 git hooks 结合
- 通过管道向 Careti 传入复杂结构
- 与 jq/grep/awk 等工具组合

**CI/CD 集成**
- GitHub Actions 自动代码审查
- GitLab 管道生成迁移脚本
- Jenkins 分析测试失败并给出建议
