# ^ 文档读取工具

Caret的AI可以**只通过路径直接读取各种文档文件**。用户无需手动附加文件。

## 与Cline的区别

| 功能 | Cline | Careti |
|------|-------|-------|
| PDF读取 | 仅用户附加时 | **AI通过路径直接读取** |
| DOCX/XLSX读取 | 仅用户附加时 | **AI通过路径直接读取** |
| 韩文 (HWPX) | 不支持 | **支持** |
| 韩文 5.0 (HWP) | 不支持 | **支持** |
| PowerPoint (PPTX) | 不支持 | **支持** |
| Jupyter Notebook | 不支持 | **支持** |

## 支持的格式

| 格式 | 扩展名 | 说明 |
|------|--------|------|
| PDF | `.pdf` | 文本提取 |
| Word | `.docx` | 最新Word文档 |
| Excel | `.xlsx` | 最新Excel电子表格 |
| PowerPoint | `.pptx` | 幻灯片文本提取 |
| 韩文 (新版) | `.hwpx` | 韩文 2014及以后 |
| 韩文 (旧版) | `.hwp` | 韩文 5.0 ~ 2010 |
| Jupyter | `.ipynb` | 笔记本单元格内容 |

### 不支持的格式

不支持旧版二进制格式:

| 格式 | 扩展名 | 替代方案 |
|------|--------|------|
| Word 97-2003 | `.doc` | 转换为`.docx` |
| Excel 97-2003 | `.xls` | 转换为`.xlsx` |
| PowerPoint 97-2003 | `.ppt` | 转换为`.pptx` |

转换工具: LibreOffice、Google Docs、Microsoft Office

## 使用方法

### 示例1: 分析规格文档

```
用户: 分析 docs/spec.pdf
AI: [使用文档读取工具] → 提取PDF内容 → 提供分析结果
```

### 示例2: 读取韩文文档

```
用户: 总结一下合同.hwp的内容
AI: [使用文档读取工具] → 提取HWP文本 → 提供摘要
```

### 示例3: 分析Excel数据

```
用户: 分析 sales-report.xlsx 的数据
AI: [使用文档读取工具] → 提取电子表格数据 → 提供分析结果
```

## 安全性

### 路径保护
- 通过路径规范化防止目录遍历攻击
- 工作区内的文件自动批准
- 外部文件需要用户批准

### 文件大小限制
- 最大50MB
- 超过时显示明确的错误消息

## 配置

文档读取工具默认启用。由于是只读操作，没有单独的设置开关。

## 已知限制

1. **不包含图像/图表**: 文档中的图像、图表不会转换为文本
2. **复杂布局**: 表格、多列布局会转换为简单文本
3. **原始格式丢失**: 字体、颜色等格式信息不会被提取
