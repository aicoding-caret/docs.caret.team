# 增强的提供商设置

## 🚀 Caret 的提供商竞争力

**Caret 比 Cline 支持更多的提供商,并提供更专业的提供商功能。**

### 为什么 Caret 的提供商更强大?

1. **自动模型查询**: 通过 LiteLLM API 集成,自动显示 100 多个提供商的模型
2. **实时更新**: 新模型发布时无需更新 Caret 即可立即使用
3. **韩国企业合作**: 额外支持 Worksai 等韩国 AI 提供商
4. **简便设置**: 无需手动输入,只需从下拉列表中选择即可完成

## 📋 概述

Caret 通过 **LiteLLM API 集成**完全自动化了提供商设置过程。您不再需要逐一查找支持的模型或手动输入。

## 🎯 核心功能

### 自动模型查询

选择提供商后,该提供商支持的**所有 AI 模型都会自动显示**。

**工作原理**:
1. 在 Caret 设置中选择 AI 提供商(例如: OpenAI、Anthropic)
2. LiteLLM API 自动查询支持的模型列表
3. 从下拉列表中选择所需的模型

### 实时更新

当新的 AI 模型发布时,**无需更新 Caret** 即可自动添加到模型列表。

- 通过 LiteLLM API 实时获取最新模型信息
- 始终可以使用最新的模型阵容

### 广泛的提供商支持

支持 **100 多个 AI 提供商**:

- **主要全球提供商**:
  - OpenAI (GPT-4、GPT-3.5-Turbo 等)
  - Anthropic (Claude 系列)
  - Google (Gemini)
  - Cohere
  - AI21 Labs
  - Hugging Face

- **额外支持的提供商**:
  - Mistral AI
  - Fireworks AI
  - Groq
  - Cerebras
  - Deepseek
  - 以及更多

## 🔄 与 Cline 的差异

| 项目 | Cline | Caret |
|------|-------|-------|
| 模型输入方式 | 手动输入 | 自动查询 |
| 模型列表确认 | 需要参考文档 | 下拉列表自动显示 |
| 新模型支持 | 需要更新 | 自动反映 |
| 支持的提供商 | 有限 | 100 多个 |

## 💡 使用场景

### 场景 1: 探索提供商

当您想比较多个 AI 提供商时:

1. 选择每个提供商后,支持的模型会自动显示
2. 比较模型名称、性能和价格
3. 选择最适合您任务的模型

### 场景 2: 使用最新模型

当新的 AI 模型发布时:

1. 重启 Caret
2. 打开提供商设置
3. 新模型会自动显示在列表中

### 场景 3: 使用韩国 AI 模型

当您想使用韩国企业的 AI 模型时:

1. 选择韩国提供商(例如: Worksai - 即将添加)
2. 自动显示支持的模型
3. 选择专门针对韩语优化的模型

## 🔧 技术实现

### LiteLLM API 集成

Caret 利用 [LiteLLM](https://docs.litellm.ai/) API:

```
GET https://api.litellm.ai/v1/models
→ 返回所有支持的 AI 模型列表
```

### 后端架构

- 通过 gRPC 服务进行模型查询
- 高效的 API 调用和缓存
- 错误处理和回退机制
