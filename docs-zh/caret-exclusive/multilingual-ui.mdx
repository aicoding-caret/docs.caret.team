# 完整的多语言 UI

用您的母语舒适地编码。Caret 提供真正的多语言支持,从 UI 到文档都经过完整翻译。

## 支持的语言

Caret 完美支持 4 种语言:

### 🇰🇷 韩语 (Korean)
- **UI**: 所有设置、菜单、消息都已韩语化
- **文档**: docs.caret.team/ko
- **AI 响应**: 自动用韩语回复

### 🇺🇸 英语 (English)
- **UI**: 默认语言
- **文档**: docs.caret.team/en
- **AI 响应**: 自动用英语回复

### 🇯🇵 日语 (Japanese)
- **UI**: 完整的日语翻译
- **文档**: docs.caret.team/ja
- **AI 响应**: 自动用日语回复

### 🇨🇳 中文 (Chinese)
- **UI**: 简体中文支持
- **文档**: docs.caret.team/zh
- **AI 响应**: 自动用中文回复

## 语言设置方法

### 1. 自动检测(推荐)

Caret 自动检测 VS Code 语言设置:

```json
// VS Code settings.json
{
  "locale": "zh-cn"  // 自动应用中文
}
```

### 2. 手动设置

在 Caret 设置中直接选择:

1. 打开设置 (⚙️)
2. **常规** 选项卡 → **语言** 部分
3. 选择所需语言
4. 重启 VS Code

### 3. 命令面板

```
Cmd+Shift+P (Mac) / Ctrl+Shift+P (Windows/Linux)
> Caret: Change Language
```

## 文档站点多语言

### 各语言专用域名

每种语言都有独立的文档站点:

- **韩语**: https://docs.caret.team/ko
- **英语**: https://docs.caret.team/en
- **日语**: https://docs.caret.team/ja
- **中文**: https://docs.caret.team/zh

### 自动重定向

从 Caret UI 点击文档链接时:

```typescript
// 根据当前设置的语言自动链接
语言: 韩语 → docs.caret.team/ko/getting-started
语言: 英语 → docs.caret.team/en/getting-started
语言: 日语 → docs.caret.team/ja/getting-started
语言: 中文 → docs.caret.team/zh/getting-started
```

## AI 响应语言

### 自动语言匹配

AI 自动用您设置的 UI 语言回复:

**中文设置时**:
```
用户: "创建一个 React 组件"
AI: "我将创建一个 React 组件..."
```

**英语设置时**:
```
User: "Create a React component"
AI: "I'll create a React component..."
```

### 手动指定语言

仅在特定请求中使用其他语言:

```
"Explain this code in English"
→ 中文 UI 但用英语回复

"用中文解释这段代码"
→ 英语 UI 但用中文回复
```

## 常见问题

**问: 更改语言后现有聊天历史会怎样?**
答: 保持不变。从新消息开始将以选定的语言显示。

**问: AI 模型支持所有语言吗?**
答: 是的,Claude、GPT、Gemini 等主要模型都很好地支持多语言。

**问: 翻译有奇怪的地方。**
答: 请在 GitHub 上提交 issue。我们会快速修复。

**问: 计划添加其他语言吗?**
答: 我们正在根据社区请求考虑西班牙语、德语等。
