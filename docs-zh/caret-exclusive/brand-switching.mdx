# OEM品牌定制化

Caret为企业客户提供完整的白标（White Label）解决方案。如果您的企业希望以自己的品牌提供AI编码助手，可以利用Caret的OEM品牌定制化系统来实现。

## 什么是OEM品牌定制化？

OEM（Original Equipment Manufacturer，原始设备制造商）品牌定制化是一个专为企业打造的解决方案，它保留了Caret的核心功能，同时允许您完全改变外观和品牌标识。通过这种方式，企业可以以自己的品牌向客户或内部开发团队提供AI编码助手。

举个例子，假设有一家名为A的软件公司，想要在自己的开发者工具产品线中添加AI助手功能。通过Caret的OEM解决方案，他们可以将产品命名为"A Company AI Assistant"，应用自己公司的Logo和品牌色彩。对于最终用户来说，他们不需要知道这是基于Caret开发的，而会将其视为A公司的独立产品。

## 可定制化的元素

通过OEM品牌定制化系统，您可以修改以下元素：

### 品牌标识

**产品名称**：您可以自由设置VS Code扩展程序显示的名称。例如，显示为"YourCompany AI"而不是"Caret"。

**Logo和图标**：在侧边栏图标、聊天标题栏、状态栏等所有UI元素中，都会显示您公司的Logo而不是Caret的Logo。我们支持SVG和PNG格式，并且可以提供各种尺寸的图标。

**品牌色彩**：您可以根据公司的品牌指南调整扩展程序的主题颜色。这些颜色会应用到按钮、高亮显示、进度条等所有UI组件上。

### 用户体验

**欢迎消息**：用户首次启动扩展程序时显示的欢迎消息可以完全定制。您可以按照公司的语气和风格来编写这些消息。

**文档链接**：帮助和文档链接不会指向Caret的官方文档，而是可以链接到您公司提供的定制文档站点。

**支持渠道**：咨询链接和错误报告链接可以连接到您公司的支持系统。

## 技术实现

Caret的OEM品牌系统基于配置文件运行。提供给企业客户的定制构建版本会包含如下配置：

```json
{
  "brand": {
    "name": "YourCompany AI Assistant",
    "displayName": "YourCompany AI",
    "publisher": "YourCompany",
    "iconPath": {
      "light": "./assets/logo-light.svg",
      "dark": "./assets/logo-dark.svg"
    },
    "colors": {
      "primary": "#1E40AF",
      "secondary": "#3B82F6",
      "accent": "#60A5FA"
    },
    "urls": {
      "docs": "https://docs.yourcompany.com/ai-assistant",
      "support": "https://support.yourcompany.com",
      "marketplace": "https://marketplace.yourcompany.com"
    }
  }
}
```

这些设置会在扩展程序构建时应用，运行时无法更改。这是为了保证品牌一致性而特意设计的。

## 应用场景

### 企业内部工具

对于大型企业来说，可以将Caret作为内部开发者的专用工具。通过提供以公司品牌定制的AI助手，开发者会觉得这是公司批准的官方工具，从而产生信任感。

例如，您可以将其命名为"Samsung Dev AI"或"Hyundai Code Assistant"，并部署给内部开发团队。在这种情况下，它不会被视为外部服务，而是公司内部系统，从而消除政策合规和安全方面的顾虑。

### SaaS产品集成

如果您的公司提供开发者工具类SaaS产品，可以将Caret集成为产品的一部分。例如，一家提供云端IDE的公司可以将Caret作为平台的AI功能使用，同时品牌完全显示为自己公司的。

### 合作伙伴网络

软件供应商或系统集成商在向客户提供AI编码解决方案时，可以用自己或客户公司的品牌进行定制。这不仅能增加附加值，还能增强客户忠诚度。

## 许可和价格

OEM品牌定制化通过企业许可证提供。许可证包含以下内容：

- 完整的白标定制权限
- 专属构建和部署支持
- 定制域名和文档托管支持
- 优先技术支持
- 定期更新和安全补丁

具体价格会根据用户数量、部署方式（内部/外部）和技术支持级别而有所不同。详细报价需要单独协商。

## 开始使用

如果您的公司希望以自己的品牌提供Caret，请按照以下步骤操作：

1. **初步咨询**：发送邮件至support@caret.team。请简要说明您公司的商业模式和需求。

2. **需求分析**：与Caret团队一起讨论具体的品牌需求、技术栈、部署计划等。

3. **合同和许可**：根据使用范围和条件签订企业许可协议。

4. **定制化工作**：提供您公司的品牌资产（Logo、颜色、文案），Caret团队将创建定制构建版本。

5. **测试和验证**：在内部测试定制构建版本，并可以提出必要的调整需求。

6. **部署**：验证完成后，即可向内部或外部用户部署。

## 技术支持

OEM客户将获得专属的技术支持团队。与品牌定制相关的技术问题、更新请求、安全问题等都会优先处理。

在定期更新时，您的定制品牌设置也会自动保留。如有需要，我们可以在更新前进行预先测试。
